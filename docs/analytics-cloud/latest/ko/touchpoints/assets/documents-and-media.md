# 문서 및 미디어

문서와 미디어 파일은 사이트 사용자가 보거나 다운로드하기를 원하는 자산입니다. Analytics Cloud는 문서 및 미디어 자산에 대한 사용자 참여에 대한 통찰력을 제공합니다.

![Analytics data for documents can be viewed in a table format.](documents-and-media/images/01.png)

사이트의 문서 및 미디어 목록이 다양한 측정항목과 함께 표시됩니다. 이 데이터를 CSV 파일로 다운로드하려면 _보고서 다운로드_를 클릭하세요. 자세한 내용은 [보고서 다운로드](../../reference/downloading-reports.md) 참조하세요.

## 방문자 행동

자산 세부정보의 개요 탭에는 여러 데이터 프레젠테이션이 표시됩니다. 이 대시보드 보기를 PDF 파일로 다운로드하려면 _보고서 다운로드_를 클릭하세요. 자세한 내용은 [보고서 다운로드](../../reference/downloading-reports.md) 참조하세요.

방문자 행동이라는 첫 번째 차트에는 네 가지 중요한 측정항목이 포함되어 있습니다.

1. 총 다운로드
1. 총 미리보기
1. 총 댓글 수
1. 평균 평점

![User engagement with blog entries can be viewed in a chart over time.](documents-and-media/images/02.png)

### 다운로드

다운로드 추세선은 선택한 기간 동안 하루(또는 어제 또는 지난 24시간을 선택한 경우 시간당) 총 다운로드 수를 표시합니다.

### 미리보기

문서 및 미디어 파일을 미리 볼 수 있습니다. 최종적으로 다운로드되지 않더라도 미리보기는 파일과의 상호작용을 보여주므로 중요한 지표입니다. 미리 보기 추세선은 선택한 기간 동안 하루(또는 어제나 지난 24시간을 선택한 경우 시간당) 총 미리 보기 수를 표시합니다.

### 댓글

문서 및 미디어 파일에 댓글을 활성화할 수 있습니다. 활성화된 경우 댓글 추세선은 선택한 기간 동안 하루(또는 어제 또는 지난 24시간을 선택한 경우 시간당) 파일에 대한 총 댓글 수를 표시합니다.

### 평가

문서 및 미디어에 대한 등급을 활성화할 수 있습니다. 등급이 단순한 좋아요/비추천 또는 별 등급(별 0\~5개)인지 여부에 관계없이 등급을 사용하여 사용자가 사이트의 문서 및 미디어 파일을 평가하는 방법을 비교할 수 있습니다.

## 청중

청중 보고서는 차트를 사용하여 자산과 청중의 상호 작용에 대한 정보를 제공합니다. 다음 질문에 답합니다.

* 내 콘텐츠와 상호작용하는 사용자는 몇 명입니까?
* 이들 사용자 중 알려지거나 익명인 사용자는 몇 명입니까?
* 내 콘텐츠와 상호작용한 알려진 사용자 중 Analytics Cloud 세그먼트에 속하는 사용자는 몇 명입니까?
* 세그먼트의 사용자 중 상위 15개 세그먼트는 무엇입니까?

잠재고객 보고서의 차트는 다음과 같습니다.

**다운로드:** 알려지거나 익명인 총 고유 방문자의 비율을 나타내는 도넛 차트입니다. 센터에는 총 순 방문자 수가 표시됩니다. 도구 설명에는 해당 섹션의 사용자 수(예: 알려진 사용자 또는 익명 사용자 수)가 표시됩니다.

!!! 참고
로그인 시 이메일 주소가 Analytics Cloud와 동기화된 사용자 데이터와 일치하면 개인이 알려진 것으로 간주됩니다. 신규 사용자가 사이트에 등록하면 해당 데이터가 Analytics Cloud와 동기화되고 알려진 개인으로 표시되는 데 최대 2시간이 걸릴 수 있습니다.

**세분화된 다운로드:** 하나 이상의 Analytics Cloud 세그먼트에 속하거나 속하지 않는 알려진 개인의 비율을 나타내는 도넛 차트입니다. 중앙에는 알려진 개인의 총 수가 표시됩니다. 도구 설명에는 해당 섹션에 대해 알려진 개인의 수가 표시됩니다(예: 하나 이상의 세그먼트에 속하는 알려진 개인의 수).

**시청자 세그먼트:** 각 세그먼트를 구성하는 알려진 개인의 비율을 보여주는 막대 차트입니다. 차트에는 상위 5개 세그먼트 각각에 대한 막대가 표시되고 나머지 세그먼트는 마지막 막대에 집계됩니다. 마지막 막대의 도구 설명에는 나머지 각 세그먼트의 값이 표시됩니다.

![Charts help to identify unique visitors, logged in users, and users who fall into particular segments.](documents-and-media/images/03.png)

잠재고객에 대해 표시되는 세분화 데이터는 방문 당시 세그먼트에 속한 개인을 기반으로 합니다.

## 기술별 견해

기본 _Devices_ 탭에서 운영 체제별(장치 유형별로 그룹화) 페이지 보기의 누적 막대 그래프를 봅니다.

![Users can also be viewed by device.](documents-and-media/images/05.png)

선택한 기간 동안 상위 8개 웹 브라우저를 표시하는 도넛 차트를 보려면 _브라우저_를 클릭하세요. 해당하는 경우 나머지 웹 브라우저는 9번째 도넛 세그먼트에 집계됩니다.

## 자산 표시 사이트


페이지 이름과 URL이 포함된 자산이 테이블에 나타납니다. Analytics Cloud의 모든 페이지 데이터와 마찬가지로 사이트 방문자가 어떤 방식으로든 상호작용한 페이지만 추적되고 보고됩니다. 링크를 클릭해도 자산이 있는 실제 페이지로 연결되지 않는다는 점에 유의하는 것이 중요합니다. 대신 페이지의 페이지 분석 보기로 연결됩니다. 여기에서 실제 페이지의 URL을 클릭할 수 있습니다.
