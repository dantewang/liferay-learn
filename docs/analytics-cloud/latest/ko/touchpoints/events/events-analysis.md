# 이벤트 분석

개인이 귀하의 사이트를 방문하고 상호 작용할 때 Analytics Cloud는 이러한 이벤트를 추적합니다. Liferay DXP 인스턴스에서 발생하는 이러한 이벤트의 대부분은 기본적으로 추적되도록 설정됩니다. 사이트에 코드 스니펫을 추가하여 [맞춤 이벤트를 정의하고 추적](./tracking-events.md)할 수도 있습니다. 이벤트 분석 도구를 사용하여 다양한 속성을 기반으로 이벤트를 분석하고 필터링합니다. 이벤트를 분석하려면

1. **Touchpoints** &rarr; **Events** 로 이동합니다. 

1. **Create Analysis** 버튼을 클릭하여 새로운 이벤트 분석을 시작합니다. 또는 해당 이름을 클릭하여 기존 분석을 봅니다. 특정 분석을 찾으려면 검색 표시줄을 사용하십시오.

1. 분석 행에서 **추가** 아이콘(![Add](../../images/icon-add.png))을 클릭하고 분석할 이벤트를 선택합니다. 또는 검색창을 사용하여 특정 이벤트를 찾으십시오.
   
   ![분석할 이벤트를 선택합니다.](./events-analysis/images/01.png)
   
   기본 이벤트는 Analytics Cloud에서 이미 추적됩니다. 사용자 지정 이벤트는 사용자가 정의하고 설정한 이벤트입니다. 자세한 내용은 [이벤트 정의](../../workspace-data/definitions/definitions-for-events.md)를 참조하세요.
   
   드롭다운 목록에서 특정 이벤트 위로 마우스를 가져가면 이벤트 설명을 볼 수 있습니다. 설명을 편집하려면 **연필** 아이콘(![연필](../../images/icon-edit.png))을 클릭하세요. 
   
   1 분석 행에서 **_** 추가 아이콘(![Add](../../images/icon-plus.png))을 클릭하고 분석 속성을 선택하십시오. 
  
  ![이벤트에 대한 분류를 선택합니다.](./events-analysis/images/02.png)
  
  분석 행에서 최대 3개의 속성을 선택할 수 있습니다. 각 유형의 이벤트는 서로 다른 속성으로 구성됩니다. 글로벌 속성은 모든 유형의 이벤트에 대해 수집되는 속성입니다. 자세한 내용은 [이벤트 속성](../../workspace-data/definitions/definitions-for-event-attributes.md) 을 참조하십시오.
  
  드롭다운 목록에서 특정 속성 위로 마우스를 가져가면 속성의 설명과 데이터 유형 변환을 볼 수 있습니다. **연필** 아이콘(![Pencil](../../images/icon-edit.png))을 클릭하여 설명 또는 데이터 유형 변환을 편집합니다.
  
  Analytics Cloud는 해당 속성에 대해 수집된 첫 번째 데이터를 기반으로 데이터 유형을 자동으로 감지하고 캐스팅하려고 시도합니다. 데이터는 부울, 날짜, 기간, 숫자 또는 문자열로 캐스팅됩니다. 데이터 유형 변환은 분석 필터링에 사용할 수 있는 매개변수를 결정합니다. 예를 들어 날짜 유형 변환이 있는 속성을 선택하면 이전 또는 이후 기간과 같은 필터링 옵션이 있습니다. 날짜, 기간 및 숫자 유형 변환에는 설정된 숫자로 데이터를 그룹화하는 그룹화 기능도 있습니다. 예를 들어 날짜 유형 변환을 사용하면 일, 월 또는 연도별로 데이터를 그룹화할 수 있습니다. 
  
  ![날짜, 기간 및 숫자에 대해 데이터를 그룹화하는 방법을 선택합니다.](./events-analysis/images/03.png)

1 필터 행에서 **추가** 아이콘(![Add](../../images/icon-plus.png))을 클릭하고 속성을 선택하여 이벤트를 추가로 필터링합니다. 드롭다운 목록에는 필터링에 사용할 수 있는 모든 속성이 표시됩니다. 이벤트 분석 도구는 선택한 이벤트 및 분류와 관련되거나 관련되지 않은 속성별로 필터링할 수 있는 유연성을 제공합니다. 

1 기본 기간은 지난 30일로 설정됩니다. 다른 기간으로 변경하려면 드롭다운 화살표를 클릭합니다. 이벤트 데이터를 이전 기간과 비교하려면 **이전과 비교** 을 선택하십시오.
  
  ![다른 기간을 선택하거나 이전 데이터와 비교하십시오.](./events-analysis/images/04.png)
  
  이벤트 분석을 볼 수 있는 3개의 탭(합계, 고유 및 평균)이 있습니다. 총계 탭에는 모든 사용자가 트리거한 총 이벤트 수가 표시됩니다. 고유 탭에는 이벤트를 트리거한 총 고유 사용자 수가 표시됩니다. 평균 탭에는 사용자당 평균 이벤트 수가 표시됩니다.

1. 필터, 분류 또는 분석을 제거하려면 변경하려는 속성 옆에 있는 닫기 아이콘(![Close](../../images/icon-close.png))을 클릭합니다. 그런 다음 새로 선택합니다.

1. 나중에 볼 수 있도록 이벤트 분석을 저장하려면 분석에 이름을 지정하고 **분석 저장** 버튼을 클릭합니다. 분석은 나중에 볼 수 있도록 이벤트 분석 도구에서 사용할 수 있습니다.
