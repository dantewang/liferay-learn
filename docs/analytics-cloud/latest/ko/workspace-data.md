액세스 토큰을 생성하거나 취소하여 [API](./workspace-data/managing-api-access.md) 액세스를 관리합니다.

작업 공간에 연결된 다양한 데이터 소스를 보고 [관리](./workspace-data/managing-data-sources.md).
