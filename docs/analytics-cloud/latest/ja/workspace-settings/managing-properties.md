# プロパティの管理

[プロパティ](../workspace-settings/scoping-sites-and-individuals-using-properties.md#creating-a-property) を作成した後は、Analytics Cloud の Workspace Settings セクションから管理および変更することができます。 **設定** &rarr; **ワークスペース設定** &rarr; **プロパティ** に移動します。

![Navigate to the Properties section in Settings.](./managing-properties/images/01.png)

プロパティに変更を加える、

1. スクロールまたは検索バーを使用して物件を検索します。

1. プロパティ名をクリックします。

## プロパティの名前を変更する

プロパティの名前を変更するには、 **編集**（ [編集アイコン](../images/icon-edit.png) ）をクリックし、変更内容を入力します。 テキストボックスの外側をクリックすると、変更内容がコミットされます。

![Click the edit icon to change the Property name.](./managing-properties/images/02.png)

## プロパティデータの消去

物件データをクリアするには、 **Clear Data** をクリックします。 操作を確認するための新しいフォームが表示されます。 確認文をコピーしてテキストボックスに貼り付けます。 これにより、 **Clear Data** ボタンが有効になります。 ボタンをクリックすると、データが削除されます。

![Copy the sentence to enable the Clear Data button](./managing-properties/images/03.png)

```{warning}
これにより、このプロパティの履歴イベントが完全に削除されます。 この操作を元に戻すことはできません。
```
## プロパティの削除

プロパティを削除するには、 **Delete** ボタンをクリックします。 操作を確認するための新しいフォームが表示されます。 確認文をコピーしてテキストボックスに貼り付けます。 これで **Delete** ボタンが有効になる。 ボタンをクリックすると、プロパティが削除されます。 すべてのサイトとコマースチャンネルがLiferayインスタンス設定から切断されるまで、プロパティは削除できません。

```{warning}
その結果、この土地とその歴史的出来事が完全に取り除かれることになる。 この操作を元に戻すことはできません。
```
## プロパティへのユーザーの追加と削除

デフォルトでは、すべてのワークスペース ユーザーがワークスペースのさまざまなプロパティにアクセスできます。 選択したユーザのみにプロパティへのアクセス権を与えるには、 **All Users** の選択を **Select Users** に変更します。 操作を確認するための新しいフォームが表示されます。 OKボタンをクリックする。

### ユーザーを追加する

プロパティにユーザーを追加するには、 **Add User** をクリックします。 ワークスペースのすべてのユーザーのリストを表示する新しいフォームが表示されます。 追加したいユーザーの横にチェックマークを付け、 **Submit** ボタンをクリックします。 ユーザーが選択ユーザーリストに追加されました。

![Place a checkmark for a user and click the Submit button.](./managing-properties/images/04.png)

### ユーザーを削除する

ユーザーを削除するには、ユーザーのメールアドレスの右側にある［X］アイコンをクリックします。 操作を確認するための新しいフォームが表示されます。 **Remove** ボタンをクリックすると、そのユーザーは選択ユーザーリストから削除されます。

![Click the X button next to a name to remove.](./managing-properties/images/05.png)
