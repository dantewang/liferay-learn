# 興味のあるトピックの理解

個人が Liferay サイトのページを操作すると、Analytics Cloud はそのページに含まれるトピックを記録します。 個人があるトピックを含むページを訪問すればするほど、そのトピックのランキングはその個人の興味に対して高くなる。

```{note}
インタレストは、サイト上での複数のインタラクションと複数のセッションに基づいている。 インタレストデータは、個人が最低5回サイトとインタラクションした後に提示される（ページビュー、ページクリックなど）。 ページ・インタラクションとは、個人が最低5秒間ページとインタラクションすることを指す。
```

関心のあるトピックは、サイトページのタイトルと説明文に基づいて生成されます。 そのため、ページタイトルと説明文が明確に定義されているものを使用することで、より具体的な関心トピックにつながります。 [ページ単位での設定](https://learn.liferay.com/w/dxp/site-building/creating-pages/page-settings/configuring-individual-pages#seo) を追加することで、関心のあるトピックを増やすことができる。

```{note}
生成された関心トピックは英語でのみサポートされています。 英語以外のコンテンツはフィルタリングされています。 しかし、英語以外のキーワードは、回避策としてページのコンフィギュレーションに追加することができる。 例えば、スペイン語でスポーツを意味する単語（`deportes`）をキーワードとして追加することができる。 個人がページとインタラクトすると、Analytics Cloud は関心のあるトピックとして `deportes` を表示します。
```

人々が興味を持っているトピックを見つけることで、製品やサービスを彼らに関連付けることができます。 Analytics Cloudのインタレスト・インサイトは、このような情報を提供する豊富で理解しやすいビジュアルを提供します。

例えば、あなたのサイトがこれらのトピックに言及しているページを持っているとします。

* 衣料品
* スポーツ

マイケルという人物がこのページに頻繁にアクセスする場合、衣料品とスポーツのトピックに対する関心度は高い。 この情報を利用する方法をいくつかご紹介します。

スポーツアパレルを販売するなら、マイケルにスポーツアパレルを販売するキャンペーンを用意しておくといいでしょう。

もしあなたがスポーツアパレルを販売していないのであれば、スポーツアパレル業者にあなたのサイトで広告を出すよう説得するために関心データを使うことができる。

いずれにせよ、利害関係の洞察は新規ビジネスの創出に役立つ。 インタレスト・インサイトは、顧客が最もよく訪れるページやアセットで最もよく言及されるトピックを教えてくれます。
