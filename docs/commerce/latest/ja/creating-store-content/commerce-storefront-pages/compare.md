# 比較

［比較］ ページは、購入者が比較のために2つ以上の商品を並べて表示できるページです。 比較ページは、購入者が［カタログ］ページで2つ以上の商品を選択した場合にのみ読み込まれる ［**Product Comparison Table**］ ウィジェットを使用します。
