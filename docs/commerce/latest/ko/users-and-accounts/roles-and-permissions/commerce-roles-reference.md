# 상거래 역할 참조

Liferay에는 계정 관리자, 계정 구성원, 구매자 및 주문 관리자와 같은 [가지 기본 역할](#out-of-the-box-roles) 이 포함되어 있습니다. 이러한 역할은 몇 가지 일반적인 역할을 모델링하도록 구성되어 있지만 추가 계정 역할을 만들거나 사용 사례에 더 잘 맞도록 기본 역할을 수정할 수 있습니다. 일부 [일반적인 사용자 지정 역할](#custom-roles) 에는 Catalog Manager, Inventory Manager, Shipment Manager, Instance Order Manager 및 Discount Manager가 포함됩니다.

```{note}
Liferay 7.3.x 및 이전 Commerce 버전의 경우 [판매 에이전트](#sales-agent) 도 기본 역할로 포함됩니다.
```

## 권한 보기

상거래 역할과 관련된 권한을 보고 관리하려면 다음 단계를 따르십시오.

1. *글로벌 메뉴* (![Global Menu](../../images/icon-applications-menu.png))을 열고 *제어판* 탭을 클릭한 다음 *사용자* &rarr; *역할*로 이동합니다.

   ![제어판에서 역할을 클릭합니다.](./commerce-roles-reference/images/01.png)

1. 원하는 역할을 클릭합니다.

   * *계정 관리자* (계정 역할)
   * *계정 구성원* (계정 역할)
   * 구매자(계정 역할)
   * 주문 관리자(계정 역할)

   ```{note}
   Liferay 7.3 및 이전 버전에서 계정 관리자는 사이트 역할이며 계정 구성원 역할은 기본 역할이 아닙니다. 
   ```

1. *권한 정의* 탭을 클릭합니다.

   여기에서 역할에 할당된 모든 애플리케이션 및 리소스 권한에 대한 요약을 볼 수 있습니다.

   권한을 제거하거나 추가할 수도 있습니다.

   ![권한 정의 탭을 클릭하여 역할 권한을 보고 관리합니다.](./commerce-roles-reference/images/02.png)

## 즉시 사용 가능한 역할

다음 역할은 상거래 활성화 중에 DXP 인스턴스에 자동으로 추가됩니다. 기본 DXP 역할과 함께 포함되며 원하는 경우 수정할 수 있습니다. 자세한 내용은 [역할 권한 정의](https://learn.liferay.com/dxp/latest/ko/users-and-permissions/roles-and-permissions/defining-role-permissions.html) 을 참조하십시오.

### 계정 관리자

> 계정 역할

계정 관리자는 계정의 슈퍼 사용자입니다. > 계정 역할

## 사용자 지정 역할

다음 목록에는 자체 상거래 인스턴스에 대해 만들 수 있는 일반적인 사용자 지정 역할이 포함되어 있습니다. 이러한 역할은 다른 DXP 역할과 동일한 방식으로 생성할 수 있습니다. 자세한 내용은 [역할 생성 및 관리](https://learn.liferay.com/dxp/latest/ko/users-and-permissions/roles-and-permissions/creating-and-managing-roles.html) 및 [역할 권한 정의](https://learn.liferay.com/dxp/latest/ko/users-and-permissions/roles-and-permissions/defining-role-permissions.html) 을 참조하십시오.

### 카탈로그 관리자

> 일반 역할

카탈로그 관리자 역할은 특정 카탈로그 내에서 제품에 액세스, 생성, 편집 및 관리할 수 있는 권한을 부여합니다. 이 역할은 여러 공급업체 또는 회사 팀이 별도의 카탈로그에서 서로 다른 제품을 관리하는 여러 카탈로그 시나리오에서 유용합니다. 카탈로그 관리자는 재고를 관리할 수 없습니다. 이 책임에 대해서는 [재고 관리자](#inventory-manager) 을 참조하십시오.

| 권한                 | 묘사                            |
|:------------------ |:----------------------------- |
| 카탈로그: 제어판에서 액세스    | 전역 메뉴에서 카탈로그 애플리케이션에 액세스하는 기능 |
| 카탈로그: 보기           | 카탈로그 애플리케이션을 볼 수 있는 기능        |
| 통화 > 상거래 통화: 통화 관리 | 통화 관리 기능                      |
| 포털: 제어판 메뉴 보기      | 전역 메뉴에 액세스하는 기능               |
| 제품: 제어판에서 액세스      | 전역 메뉴에서 제품 애플리케이션에 액세스하는 기능   |
| 제품: 보기             | 제품 애플리케이션을 볼 수 있는 기능          |

위의 권한 외에도 역할이 관리할 개별 카탈로그에 대해 *보기* 및 *업데이트* 권한을 할당해야 합니다.

이렇게 하려면 *글로벌 메뉴* (![Global Menu](../../images/icon-applications-menu.png))을 열고 *커머스* 탭을 클릭한 다음 *상품 관리* &rarr; *카탈로그*로 이동합니다. 그런 다음 원하는 카탈로그에 대해 *작업* 버튼(![Actions Button](../../images/icon-actions.png))을 클릭하고 *권한*을 선택합니다. 확인란을 사용하여 역할 *보기* 및 *업데이트* 권한을 할당합니다.

```{note}
카탈로그 자체를 수정하는 기능 없이 카탈로그 관리자가 카탈로그 내의 제품을 관리하도록 하려면 '카탈로그: 제어판에서 액세스' 및 '카탈로그: 보기' 권한을 제거하십시오.
```

### 할인 관리자

> 일반 역할

할인 관리자 역할은 Liferay Commerce에서 할인을 생성, 업데이트 및 삭제할 수 있는 권한을 부여합니다. 상거래 할인은 주문의 다른 부분을 대상으로 하고 다른 자격 규칙을 사용할 수 있으므로 할인 관리자에 대해 가능한 많은 권한 구성이 있습니다. 다음 구성은 모든 할인 옵션에 대한 액세스 권한을 부여합니다.

| 권한                                     | 묘사                               |
|:-------------------------------------- |:-------------------------------- |
| 계정 그룹: 보기                              | 계정 그룹 애플리케이션을 볼 수 있는 기능          |
| 계정 그룹 > 상거래 계정: 모든 계정 관리               | 계정 그룹 애플리케이션에 액세스하는 데 필요합니다.     |
| 계정 그룹 > Commerce 계정: Commerce 계정 그룹 보기 | 기존 계정 그룹을 보는 기능                  |
| 카탈로그: 보기                               | 카탈로그 애플리케이션을 볼 수 있는 기능           |
| 카탈로그 > 상업 카탈로그: 상업 카탈로그 보기             | 카탈로그 애플리케이션에서 관련 리소스를 볼 수 있는 기능  |
| 채널: 보기                                 | 채널 애플리케이션을 볼 수 있는 기능             |
| 채널 > 상거래 채널: 상거래 채널 보기                 | 채널 애플리케이션에서 관련 리소스를 볼 수 있는 기능    |
| 할인: 제어판에서 액세스                          | 글로벌 메뉴에서 할인 애플리케이션에 액세스하는 기능     |
| 할인: 권한                                 | 할인 애플리케이션에서 리소스 권한을 보고 수정하는 기능   |
| 할인: 보기                                 | 할인 응용 프로그램을 볼 수 있는 기능            |
| 할인 > 상거래 할인: 삭제                        | 할인 엔터티 삭제 기능                     |
| 할인 > 상거래 할인: 권한                        | 할인 엔터티에 대한 권한을 보고 수정하는 기능        |
| 할인 > 상거래 할인: 업데이트                      | 할인 엔터티 수정 기능                     |
| 할인 > 커머스 할인: 보기                        | 할인 엔터티를 볼 수 있는 기능                |
| 할인 > 상거래 할인: 할인 추가                     | 할인 애플리케이션에서 할인 엔터티를 생성하는 기능.     |
| 할인 > 상거래 할인: 할인 보기                     | 할인 애플리케이션에서 기존 할인 엔터티를 볼 수 있는 기능 |
| 통화 > 상거래 통화: 통화 관리                     | 통화 엔터티 액세스 및 수정 기능               |
| 포털: 제어판 메뉴 보기                          | 전역 메뉴에 액세스하는 기능                  |
| 가격표: 제어판에서 액세스                         | 글로벌 메뉴에서 가격 목록 애플리케이션에 액세스하는 기능  |
| 제품 그룹: 보기                              | 제품 그룹 엔터티를 볼 수 있는 기능             |

### 재고 관리자

> 일반 역할

재고 관리자 역할을 통해 사용자는 모든 창고의 재고를 보고 관리할 수 있습니다. 역할이 있는 사용자는 수신 배송을 추가하고, 재고 수준을 업데이트하고, 창고 간에 이동하고, 변경 로그를 볼 수 있습니다.

| 권한                | 묘사                              |
|:----------------- |:------------------------------- |
| 인벤토리: 제어판에서 액세스   | 제어판에서 인벤토리 애플리케이션에 액세스하는 기능     |
| 인벤토리: 보기          | 인벤토리 애플리케이션을 볼 수 있는 기능          |
| 창고 > 상업 재고: 재고 관리 | 창고에 연결된 인벤토리 엔터티에 액세스하고 수정하는 기능 |
| 포털: 제어판 메뉴 보기     | 전역 메뉴에 액세스하는 기능                 |

### 인스턴스 주문 관리자

> 일반 역할

주문 관리자 역할은 사용자에게 여러 채널에서 주문을 관리하는 데 필요한 권한을 부여하며 단일 사이트로 제한되지 않습니다.

| 권한                            | 묘사                                               |
|:----------------------------- |:------------------------------------------------ |
| 장바구니 열기 > 상거래 주문: 주문 추가       | 구매자의 계정에 주문을 추가하는 기능                             |
| 오픈 카트 > 상거래 주문: 오픈 주문 승인      | 워크플로가 활성화된 경우 주문을 승인하는 기능                        |
| Open Carts > 상업 주문: 열린 주문 확인  | 주문에 대한 체크아웃 프로세스를 완료하는 기능                        |
| 장바구니 열기 > 상거래 주문: 주문 삭제       | 주문 삭제 기능                                         |
| Open Carts > 상거래 주문: 주문 메모 관리 | 일반 주문 메모 변경 기능                                   |
| 장바구니 열기 > 상거래 주문: 주문 제한 사항 관리 | 제한된 주문 메모를 변경하는 기능                               |
| Open Carts > 상거래 주문: 주문 관리    | 우편 및 청구 주소와 같은 주문 세부 정보를 변경하는 기능 주문에 메모를 추가하는 기능 |
| 오픈 카트 > 상거래 주문: 오픈 주문 보기      | 열려 있는 모든 주문을 볼 수 있는 기능                           |
| 장바구니 열기 > 상거래 주문: 주문 보기       | 상태에 관계없이 주문을 볼 수 있는 기능                           |
| 주문: 제어판에서 액세스                 | 전역 메뉴에서 주문 애플리케이션에 액세스하는 기능                      |
| 주문: 권한                        | 주문 애플리케이션에 대한 권한을 보고 수정하는 기능                     |
| 주문: 보기                        | 주문 애플리케이션을 볼 수 있는 기능                             |
| 계정 그룹 > 상거래 계정: 모든 계정 관리      | 계정 그룹 애플리케이션에 액세스하는 데 필요합니다.                     |
| 포털: 제어판 메뉴 보기                 | 전역 메뉴에 액세스하는 기능                                  |

### 배송 관리자

> 일반 역할

배송 관리자 역할을 통해 사용자는 배송을 처리할 수 있습니다. 여기에는 배송 세부 정보, 배송 날짜를 추가하고 배송에 항목을 추가하고 배송 상태를 업데이트하는 기능이 포함됩니다. Commerce 3.0+ 및 Liferay 7.3+에서 사용자는 주문 애플리케이션을 통해 배송을 생성합니다. 배송 관리자만 배송을 채우고 처리하도록 하려면 다음 권한으로 충분합니다. 그렇지 않으면 추가 [주문 관리자](#order-manager-regular-role) 권한을 추가할 수 있습니다.

| 권한                       | 묘사                              |
|:------------------------ |:------------------------------- |
| 장바구니 열기 > 상거래 주문: 주문 보기  | 상태에 관계없이 주문을 볼 수 있는 기능          |
| 계정 그룹 > 상거래 계정: 모든 계정 관리 | 계정 그룹 애플리케이션에 액세스하는 데 필요합니다.    |
| 창고 > 상업 재고: 재고 관리        | 창고에 연결된 인벤토리 엔터티에 액세스하고 수정하는 기능 |
| 배송 관리                    | 선적 엔터티 액세스 및 수정 기능              |
| 포털: 제어판 메뉴 보기            | 전역 메뉴에 액세스하는 기능                 |
| 선적: 제어판에서 액세스            | 전역 메뉴에서 배송 애플리케이션에 액세스하는 기능     |
| 선적: 보기                   | 발송물 애플리케이션을 볼 수 있는 기능           |
| 배송 > 상업 창고: 보기           | 창고 엔터티를 볼 수 있는 기능               |

## 라이프레이 7.3 이하

### 판매 대리점

> 일반 역할

판매 대리인은 Liferay Commerce의 일반 역할이며 양수인은 관리 권한을 부여하지 않고도 역할에 할당된 모든 계정을 관리할 수 있습니다. 일반 역할이므로 *일반 역할* 탭에 있습니다.

![권한 정의 탭에서 기본 영업 에이전트 권한을 봅니다.](./commerce-roles-reference/images/06.png)

| 권한       | 묘사                     |
|:-------- |:---------------------- |
| 조직 관리    | 조직을 추가하거나 제거하는 기능      |
| 가용 계정 관리 | 판매 대리인이 속한 계정을 관리하는 기능 |
