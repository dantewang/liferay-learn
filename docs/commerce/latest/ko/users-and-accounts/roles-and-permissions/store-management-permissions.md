# 스토어 관리 권한

Liferay Commerce를 사용하면 권한을 사용하여 전역 메뉴(![Global Menu](../../images/icon-applications-menu.png))의 상거래 탭에서 매장 관리 애플리케이션 및 리소스에 대한 액세스를 제어할 수 있습니다.

![Store Management 애플리케이션 및 리소스에 대한 액세스를 제어합니다.](./store-management-permissions/images/01.png)

권한 정의 탭의 *Commerce* &rarr; *Store Management* 에서 사용자 역할에 대한 Store Management 권한을 관리할 수 있습니다.  권한을 개별적으로 할당하거나 *작업* 을 선택하여 애플리케이션 또는 리소스에 대한 모든 권한을 할당합니다.

![권한 정의 탭에서 사용자 역할에 대한 스토어 관리 권한을 관리합니다.](./store-management-permissions/images/02.png)

## 애플리케이션 권한

애플리케이션 권한은 애플리케이션 자체에 대한 기능을 부여하지만 [관련 리소스 권한](#resource-permissions) 은 포함하지 않습니다.

매장 관리에는 다음 애플리케이션이 포함됩니다.

**채널**: 이 응용 프로그램은 제품을 표시하고 주문을 받는 데 사용되는 채널을 생성하고 관리하는 데 사용됩니다.

**통화**: 이 응용 프로그램은 Liferay 인스턴스 내에서 사용되는 통화를 추가하고 관리하는 데 사용됩니다.

모든 매장 관리 애플리케이션에는 다음 권한이 있습니다.

| 허가        | 묘사                        |
|:--------- |:------------------------- |
| 컨트롤 패널 접근 | 전역 메뉴에서 애플리케이션에 액세스하는 기능  |
| 설정        | 애플리케이션의 구성 옵션을 보고 설정하는 기능 |
| 권한        | 애플리케이션의 권한을 보고 수정하는 기능    |
| 환경 설정     | 응용 프로그램 기본 설정을 보고 지정하는 기능 |
| 보기        | 응용 프로그램을 볼 수 있는 기능        |

## 리소스 권한

리소스 권한은 애플리케이션 리소스와 관련된 특정 기능을 부여합니다. 이러한 권한 중 일부는 데이터베이스 엔터티(즉, 모델 리소스)에 대한 작업을 수행할 수 있는 기능을 부여합니다. 다른 것들은 응용 프로그램 컨텍스트에서 리소스 관련 작업을 수행하는 기능을 부여합니다(예: 새 리소스 엔터티를 생성하는 기능).

매장 관리 애플리케이션은 다음 리소스에 연결됩니다.

**상거래 채널** (채널 아래에 나열됨): 채널은 연결된 사이트를 통해 제품을 표시하고 고객 주문을 받는 데 사용되는 엔터티입니다. 각 채널에는 주문, 결제 설정, 배송 설정, 세금 계산 및 알림에 대한 자체 워크플로가 있습니다.

| 허가 | 묘사                        |
| -- | ------------------------- |
| 삭제 | 채널 엔티티를 삭제하는 기능           |
| 권한 | 채널 엔터티에 대한 권한을 보고 수정하는 기능 |
| 수정 | 채널 엔터티 수정 기능              |
| 보기 | 채널 엔터티를 볼 수 있는 기능         |

**상거래 채널** (채널 아래에 나열됨, 이전에는 제어판 > 일반 권한 아래에 나열됨): 이러한 권한은 채널 애플리케이션에서 리소스 관련 작업을 수행할 수 있는 기능을 부여합니다.

| 허가        | 묘사                             |
| --------- | ------------------------------ |
| 커머스 채널 추가 | 채널 엔터티 생성 기능                   |
| 권한        | 채널 애플리케이션에서 리소스 권한을 보고 수정하는 기능 |
| 커머스 채널 보기 | 채널 보기 기능                       |

**상거래 알림 템플릿** (채널 아래에 나열됨): 이 권한은 채널 이벤트에 대한 알림을 생성하는 데 사용되는 사용자 지정 템플릿인 알림 템플릿 엔터티에서 작업을 수행할 수 있는 기능을 부여합니다.

| 허가 | 묘사                            |
| -- | ----------------------------- |
| 삭제 | 알림 템플릿 엔티티를 삭제하는 기능           |
| 권한 | 알림 템플릿 엔터티에 대한 권한을 보고 수정하는 기능 |
| 수정 | 알림 템플릿 엔티티를 수정하는 기능           |
| 보기 | 알림 템플릿 엔티티를 보는 기능             |

**상거래 알림** (채널 아래에 나열됨): 알림은 이벤트 트리거에 의해 생성된 알림 템플릿의 인스턴스입니다. 채널 애플리케이션 내에 표시됩니다.

| 허가           | 묘사                        |
| ------------ | ------------------------- |
| 알림 템플릿 추가    | 알림 템플릿 엔터티 생성 기능          |
| 알림 대기 항목 삭제  | 대기열에서 알림 항목을 제거하는 기능      |
| 권한           | 알림 엔터티에 대한 권한을 보고 수정하는 기능 |
| 얼림 대기 항목 재발송 | 알림 대기열 항목을 다시 보내는 기능      |
| 알림 대기 항목 보기  | 대기열에서 알림을 보는 기능           |
| 알림 템플릿 보기    | 알림 템플릿 엔티티를 보는 기능         |

**상거래 통화** (통화 아래에 나열됨, 이전에는 제어판 > 일반 권한 아래에 나열됨): 이러한 권한은 통화 애플리케이션에서 리소스 관련 작업을 수행할 수 있는 기능을 부여합니다.

| 허가    | 묘사                             |
| ----- | ------------------------------ |
| 통화 관리 | 통화 엔터티 액세스 및 수정 기능             |
| 권한    | 통화 애플리케이션에서 리소스 권한을 보고 수정하는 기능 |
