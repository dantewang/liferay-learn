# 주문 관리 권한

Liferay Commerce를 사용하면 권한을 사용하여 전역 메뉴(![Global Menu](../../images/icon-applications-menu.png))의 상거래 탭에서 주문 관리 애플리케이션 및 리소스에 대한 액세스를 제어할 수 있습니다.

![Order Management 애플리케이션 및 리소스에 대한 액세스를 제어합니다.](./order-management-permissions/images/01.png)

권한 정의 탭의 **Commerce** &rarr; **Order Management** 에서 사용자 역할에 대한 Order Management 권한을 관리할 수 있습니다. 권한을 개별적으로 할당하거나 **작업** 을 선택하여 애플리케이션 또는 리소스에 대한 모든 권한을 할당합니다.

![권한 정의 탭에서 사용자 역할에 대한 주문 관리 권한을 관리합니다.](./order-management-permissions/images/02.png)

## 애플리케이션 권한

애플리케이션 권한은 애플리케이션 자체에 대한 기능을 부여하지만 [관련 리소스 권한](#resource-permissions) 은 포함하지 않습니다.

주문 관리에는 다음 애플리케이션이 포함됩니다.

**주문** : 이 애플리케이션은 Liferay 인스턴스에서 주문을 보고 관리하는 데 사용됩니다.

**배송** : 이 애플리케이션은 Liferay 인스턴스에서 주문에 대해 생성된 배송을 보고 관리하는 데 사용됩니다.

**구독** : 이 애플리케이션은 Liferay 인스턴스에서 활성 및 비활성 고객 구독을 모두 보고 관리하는 데 사용됩니다.

모든 주문 관리 애플리케이션에는 다음 권한이 있습니다.

| 허가        | 묘사                        |
|:--------- |:------------------------- |
| 컨트롤 패널 접근 | 전역 메뉴에서 애플리케이션에 액세스하는 기능  |
| 설정        | 애플리케이션의 구성 옵션을 보고 설정하는 기능 |
| 권한        | 애플리케이션의 권한을 보고 수정하는 기능    |
| 환경 설정     | 응용 프로그램 기본 설정을 보고 지정하는 기능 |
| 보기        | 응용 프로그램을 볼 수 있는 기능        |

## 리소스 권한

리소스 권한은 애플리케이션 리소스와 관련된 특정 기능을 부여합니다. 이러한 권한 중 일부는 데이터베이스 엔터티(즉, 모델 리소스)에 대한 작업을 수행할 수 있는 기능을 부여합니다. 다른 것들은 응용 프로그램 컨텍스트에서 리소스 관련 작업을 수행하는 기능을 부여합니다(예: 새 리소스 엔터티를 생성하는 기능).

주문 관리 애플리케이션은 다음 리소스를 참조합니다.

**상거래 주문**(주문 아래에 나열됨): 이 권한은 주문 엔터티에 대한 작업을 수행할 수 있는 권한을 부여합니다. 주문은 고객이 채널에서 제품을 구매할 때 생성되며 주문 정보를 저장하고 처리하는 데 사용됩니다. 각 주문 엔터티에는 고객 정보(예: 계정, 청구, 배송)뿐만 아니라 구매한 항목, 주문 합계, 날짜, 메모, 수락 워크플로 상태 등이 포함됩니다.

| 허가          | 묘사                              |
| ----------- | ------------------------------- |
| 주문 추가       | 주문 추가 기능                        |
| 오픈 오더 승인    | 워크플로가 활성화된 경우 주문을 승인하는 기능       |
| 오픈 오더 결제    | 열린 주문에 대한 체크아웃 프로세스를 완료하는 기능    |
| 주문 삭제       | 주문 삭제 기능                        |
| 모든 계정 관리    | 계정 주문을 관리하는 포틀릿에 액세스하는 데 필요합니다. |
| 주문 비고 관리    | 제한되지 않은 주문 메모에 액세스하고 수정하는 기능    |
| 주문 가격 관리    | 주문 요약에서 주문 엔터티의 가격을 편집하는 기능     |
| 주문 제한 비고 관리 | 주문에 대한 제한된 메모에 액세스하고 수정하는 기능    |
| 주문 관리       | 주문 세부 정보 변경 기능(예: 우편 및 청구 주소)   |
| 권한          | 주문 엔터티에 대한 권한을 보고 관리하는 기능       |
| 오픈 오더 보기    | 열려 있는 모든 주문을 볼 수 있는 기능          |
| 주문 보기       | 상태에 관계없이 모든 주문을 볼 수 있는 기능       |

```{note}
Liferay DXP 7.3 및 이전 버전에서는 모든 계정 관리 및 주문 가격 관리가 제어판 > 일반 권한 아래에 나열되었습니다.
```

**상업 배송**(배송 아래에 나열됨, 이전에는 제어판 > 일반 권한 아래에 나열됨): 이러한 권한은 제품 배송을 추적하는 데 사용되는 배송 엔터티에 대한 작업을 수행할 수 있는 기능을 부여합니다.

| 허가       | 묘사                        |
|:-------- |:------------------------- |
| 모든 계정 관리 | 배송을 관리하는 포틀릿에 접근하기 위해 필요  |
| 배송 관리    | 선적 엔터티 액세스 및 수정 기능        |
| 권한       | 선적 엔터티에 대한 권한을 보고 관리하는 기능 |

**상업 창고**(창고, [배송](./inventory-management-permissions.md)및 [제품](./product-management-permissions.md)아래에 나열됨): 이 권한은 창고 엔터티에서 작업을 수행할 수 있는 기능을 부여합니다. 각 웨어하우스는 제품 재고를 저장하고 채널에 연결되며 주소 및 지리적 사이트와 함께 물리적 사이트를 나타냅니다.

| 허가 | 묘사                           |
| -- | ---------------------------- |
| 삭제 | 창고 엔터티 삭제 기능                 |
| 권한 | 웨어하우스 엔터티에 대한 권한을 보고 수정하는 기능 |
| 수정 | 창고 엔터티 편집 및 수정 기능            |
| 보기 | 창고 엔터티를 볼 수 있는 기능            |

**상거래 구독**(구독 아래에 나열됨, 이전에는 제어판 > 일반 권한 아래에 나열됨): 이러한 권한은 구독 엔터티에 대한 작업을 수행할 수 있는 기능을 부여합니다.

| 허가    | 묘사                        |
|:----- |:------------------------- |
| 구독 관리 | 구독 엔터티 액세스 및 수정 기능        |
| 권한    | 구독 엔터티에 대한 권한을 보고 수정하는 기능 |
