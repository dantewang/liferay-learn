# 주문을 보류

대기 중인 주문 페이지는 주문 관리자가 계정과 연결된 모든 불완전한 주문을 볼 수 있는 페이지입니다. 완료되지 않은 주문은 카트에 항목이 있지만 체크아웃 프로세스를 거치지 않은 것입니다. 또한 보류 중인 주문 페이지에는 주문 관리자가 동시에 여러 주문을 생성할 수 있는 **주문 추가** 버튼이 있습니다.

대기 중인 주문 페이지는 **Open Carts** 위젯을 사용합니다. [Minium Accelerator](../../starting-a-store/using-the-minium-accelerator-to-jump-start-your-b2b-store.md) 을 사용하여 샘플 데이터를 생성하는 경우 **Pending Orders** 페이지가 이미 생성되어 있으며 사이트 탐색 메뉴에서 찾을 수 있습니다.

![대기 중인 주문 페이지](./pending-orders/images/01.png)

**Open Carts** 위젯은 계정과 연결된 주문 목록을 표시합니다.

![카트 위젯 열기](./pending-orders/images/02.png)

품목이 카트에 추가되면 주문 관리자는 결제 전에 대기 중인 주문을 검토할 수 있습니다.

![대기 중인 주문 검토](./pending-orders/images/04.png)

[주문 워크플로](../../order-management/order-workflows/introduction-to-order-workflows.md) 이 활성화된 경우 위에서 언급한 단계가 약간 다를 수 있습니다.
