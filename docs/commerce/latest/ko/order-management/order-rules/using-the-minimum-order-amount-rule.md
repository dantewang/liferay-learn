# 최소 주문 금액 규칙 사용

{bdg-secondary}`사용 가능한 Liferay DXP 7.4 U1+/GA5+`

주문 처리 비용 때문에 특정 금액 이상의 주문만 수락할 수 있습니다. 고객 체크아웃이 방지되는 최소 주문 금액을 설정하여 주문 규칙을 통해 이를 수행할 수 있습니다.

## 주문 규칙 개요

주문 규칙이 적용되면 구매자는 규칙을 이행해야 결제를 계속할 수 있습니다. 최소 주문 금액 규칙의 경우 구매자는 규칙에 설정된 전체 또는 소계 장바구니 금액이 있어야 체크아웃할 수 있습니다. 그렇지 않은 경우 진행하는 데 필요한 최소 주문 금액이 경고 메시지에 표시됩니다.

![최소 주문량 미충족에 대한 경고 메시지가 표시됩니다.](./using-the-minimum-order-amount-rule/images/01.png)

## 주문 규칙 생성

최소 주문 금액 규칙은 즉시 사용할 수 있습니다.

1. **글로벌 메뉴**(![Applications Menu icon](../../images/icon-applications-menu.png))를 열고 **상거래** &rarr; **주문 규칙** 을 클릭합니다.
1. **추가** 버튼(![Add icon](../../images/icon-add.png))을 클릭하고 다음 정보를 입력합니다.

   **이름:** 최소 주문 $30

   **설명:** 최소 주문 금액 주문 규칙 테스트

   **유형:** 최소 주문 수량

1. **제출** 을 클릭합니다.

   ![주문 규칙 화면에는 주문 규칙을 구성하고 활성화하는 데 필요한 모든 것이 포함되어 있습니다.](./using-the-minimum-order-amount-rule/images/02.png)

주문 규칙에 대한 구성 보기가 열립니다. **우선 순위** 은 규칙이 둘 이상일 때 어떤 순서 규칙이 우선할지 결정합니다. 낮은 번호가 높은 번호보다 우선합니다. **활성** 토글을 사용하여 활성화하고 최소 주문 금액을 합계 또는 소계에 적용하도록 구성할 수 있습니다. 여기에서 통화를 설정할 수도 있습니다.

## 주문 규칙 적격성

![계정, 계정 그룹, 채널 또는 주문 유형에 대한 주문 규칙 적격성을 설정할 수 있습니다.](./using-the-minimum-order-amount-rule/images/03.png)

자격 탭에서 계정, 계정 그룹, 채널 또는 주문 유형에 대한 주문 규칙 자격을 설정할 수 있습니다. 기본적으로 이 모든 항목에 적용됩니다. 특정 엔터티와 연결하려면 라디오 버튼을 클릭하고 검색 표시줄을 사용하여 선택합니다.
