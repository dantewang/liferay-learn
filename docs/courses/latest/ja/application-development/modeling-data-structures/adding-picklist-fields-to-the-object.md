# オブジェクトにピックリスト・フィールドを追加する

ディストリビューター・アプリケーション・オブジェクトにピックリスト・フィールドを追加するには、他のフィールド・タイプと同じ手順を踏みますが、いくつかの追加設定オプションがあります。

1. ［**グローバルメニュー**］（![Global Menu](../../images/icon-applications-menu.png)）を開き、 ［**コントロールパネル**］ タブに移動して、 ［**オブジェクト**］ をクリックします。

1. Distributor Application オブジェクトの編集を開始し、 **Fields** タブを開きます。

1. **Add**(![Add Button](../../images/icon-add.png)) をクリックし、これらの詳細を入力し、 **Save** をクリックします：

   | 項目    | 値              |
   |:----- |:-------------- |
   | ラベル   | ビジネスタイプ        |
   | 項目名   | `businessType` |
   | 種類    | 候補リスト          |
   | 候補リスト | ビジネスタイプ        |
   | 必須    | False          |
   | 状態にする | False          |

   ![ビジネス・タイプ・ピックリスト・フィールドを追加します。](./adding-picklist-fields-to-the-object/images/01.png)

1. Distribution Regionsフィールドを追加する：

   | 項目    | 値                     |
   |:----- |:--------------------- |
   | ラベル   | 販売地域                  |
   | 項目名   | `distributionRegions` |
   | 種類    | 複数選択の候補リスト            |
   | 候補リスト | 販売地域                  |
   | 必須    | False                 |

1. Business Distribution Channelsフィールドを追加する：

   | 項目    | 値                              |
   |:----- |:------------------------------ |
   | ラベル   | ビジネス流通チャネル                     |
   | 項目名   | `businessDistributionChannels` |
   | 種類    | 複数選択の候補リスト                     |
   | 候補リスト | 流通チャネル                         |
   | 必須    | False                          |

1. 関心のある注文タイプ」フィールドを追加する：

   | 項目    | 値                      |
   |:----- |:---------------------- |
   | ラベル   | 興味のあるオーダータイプ           |
   | 項目名   | `orderTypesOfInterest` |
   | 種類    | 複数選択の候補リスト             |
   | 候補リスト | 注文タイプ                  |
   | 必須    | False                  |

1. 関心のある製品」フィールドを追加する：

   | 項目    | 値                    |
   |:----- |:-------------------- |
   | ラベル   | 注目製品                 |
   | 項目名   | `productsOfInterest` |
   | 種類    | 複数選択の候補リスト           |
   | 候補リスト | 商品種類                 |
   | 必須    | False                |

1. 推定年間購入量フィールドを追加する：

   | 項目    | 値                               |
   |:----- |:------------------------------- |
   | ラベル   | 年間推定購入量                         |
   | 項目名   | `estimatedAnnualPurchaseVolume` |
   | 種類    | 候補リスト                           |
   | 候補リスト | 年間購入量                           |
   | 必須    | False                           |
   | 状態にする | False                           |

1. Product Labelingフィールドを追加する：

   | 項目    | 値                 |
   |:----- |:----------------- |
   | ラベル   | 製品ラベリング           |
   | 項目名   | `productLabeling` |
   | 種類    | 候補リスト             |
   | 候補リスト | 製品ラベル             |
   | 必須    | False             |
   | 状態にする | False             |

   ![ディストリビューター・アプリケーションに7つのピックリストフィールドを追加。](./adding-picklist-fields-to-the-object/images/02.png)

ユーザーがディストリビューターの申請書に記入する際、各項目についてあらかじめ定義されたオプションから選択することができます。 そして、Delectable Bonsaiは、これらのデータポイントを使って、それぞれのビジネスチャンスの価値を評価することができる。

![ユーザーは事前に定義されたオプションから選択する。](./adding-picklist-fields-to-the-object/images/03.png)

　 ディストリビューター・アプリケーション・オブジェクトに必要なすべてのフィールドが追加されました。 これで、オブジェクトのフォームレイアウトとテーブルビューのデザインを始めることができます。
