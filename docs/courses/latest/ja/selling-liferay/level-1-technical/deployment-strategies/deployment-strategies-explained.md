# 配備戦略の説明

**一目瞭然**

* Liferay DXPセルフホスティングは、お客様がLiferay DXPの実装をどこでどのようにホスティングするかを選択することができます。
* Liferay Experience Cloud Self-Managedは、Liferayプラットフォーム向けにカスタマイズされたPlatform-as-a-Serviceを提供します。
* Liferay Experience Cloud SaaSは、インフラやアップグレード関連の問題を心配することなく、Liferay DXPの完全な利点を提供します。 顧客はビジネス上の問題を解決するソリューションの構築に集中できる。

## 展開戦略の比較

さて、アプリケーションがセットされ（Liferay DXP）、アプリケーションをホストするコンテキストが知らされたので、利用可能なDeployment Strategiesについて説明しましょう。

### Liferay DXP Self-Hosted

Liferay DXP Self-Hostedでは、LiferayはDXPソフトウェア本体のみを提供し、それ以外はすべてお客様の責任となります。 DXP Self-Hostedは、パブリッククラウド、プライベートクラウド、自社のデータセンターなど、お客様のご希望の場所に導入することができます。

**価値の低いアクティビティ**

| 責任                        | Liferay  | 顧客       |
| ------------------------- | -------- | -------- |
| プロダクト                     | &#10004 |          |
| サブスクリプション（サポート、保証、免責など）   | &#10004 |          |
| インフラ（SLA付き稼働時間）           |          | &#10004 |
| アプリケーション稼働時間SLA           |          | &#10004 |
| プラットフォームサービス（データベース、検索など） |          | &#10004 |
| アーキテクチャと高可用性              |          | &#10004 |

| アップデート                    |          | &#10004 |
| パッチ、セキュリティホットフィックス        |          | &#10004 |
| ソフトウェアのトラブルシューティング        |          | &#10004 |
| セキュリティスキャン／SAST／DAST      |          | &#10004 |

**価値の高いアクティビティ**

| 責任          | Liferay | 顧客       |
| ----------- | ------- | -------- |
| アプリケーションの配信 |         | &#10004 |

このアプローチでは、ライフバンクはデータベース・サーバー、検索エンジン・サーバー、ストレージ・オプションなどのプラットフォーム・インフラとソフトウェアを提供する必要がある。

これはミッションクリティカルなアプリケーションになるため、高可用性、データベースのレプリケーション、ロードバランサー（おそらくウェブサーバーのクラスタ）を実装し、これをサポートするアーキテクチャを設計・維持する必要がある。

このモデルでは、Liferayはソフトウェアとサブスクリプション（サポート、保証、法的補償など）のみに責任を負います。

一方、ライフバンクはそれ以外のすべてに責任を負っている。 そして、これらの活動のほとんどは、実際にビジネス上の問題を解決していないという意味で、価値が低く、しかし、通常、時間がかかり、実装と維持に費用がかかる可能性がある。

アプリケーションをホスティングするためのすべての初期設定と決定に加え、アップグレード、アップデート、バックアップルーチンなどの保守作業も考慮しなければならない。

これらすべてが考慮されて初めて、ビジネス・ニーズを満たすソリューションを生み出す部分が実際に始まるのである！

しかし、このアプローチには、顧客が望む、あるいは必要とするプラットフォーム・ソフトウェアとホスティング環境を正確に選択できる完全な柔軟性が伴うことを忘れてはならない。 そのため、顧客がやるべきことは増えるが、柔軟性も増す。

しかし、ライフバンクにとっては、このような柔軟性は必要ないため、他の選択肢を検討している。 ライフバンクは、彼らにとって本当に重要なこと、つまりエンドユーザー体験を開発することにもっと集中したいと考えている。 業界の絶え間ない変化に伴い、競争力を維持するためには、新しい資産や機能を導入し、より速く進化させる必要がある。

そして、進化したときに問題にぶつからないと確信できることを望んでいる。

このモデルでは、LiferayはLiferay DXPをサポートするために最適なプラットフォームソフトウェアを決定し、バックアップ、DevOps / CICD、基幹システムとの統合のためのVPN接続などの追加機能を提供します。

さらに、オートスケーリングと高可用性もすぐに利用できるため、利用率のピークにも自動的に対応できる。

Liferay Experience Cloud Self-Managedでは、Liferayが価値の低い活動の数々を担当します。 インフラ、ソフトウェアプラットフォームなどに加え、Liferayは障害検知、自己修復機能、24時間365日の監視チームを提供します。

Liferayはまた、ビルド管理などを備えた完全なDevSecOps CI/CDパイプラインとともに、インフラストラクチャに対して99.95%のアップタイムSLAを提供している。

しかし、アップデート、パッチ、アップグレードなど、時間がかかり、比較的価値の低い活動については、お客様の責任となります。 そして最後に、価値の高いアプリケーション・デリバリーの仕事がある。

- コンソールはGitHub、GitLab、Bitbucketのようなコードリポジトリと統合されているので、開発者はプラットフォームに直接ビルドを生成することができます。

* 開発生産性の向上

ITオペレーションを近代化することができる：

### Liferayエクスペリエンスクラウド（SaaS）

Liferay Experience CloudはLiferayが提供するSaaSです。 Liferay Experience Cloudを使えば、お客様は本当に重要なこと、つまりビジネスアプリケーションの作成と立ち上げに集中することができます。

| 責任                        | Liferay  | 顧客 |
| ------------------------- | -------- | -- |
| プロダクト                     | &#10004 |    |
| サブスクリプション（サポート、保証、免責など）   | &#10004 |    |
| インフラ（SLA付き稼働時間）           | &#10004 |    |
| アプリケーション稼働時間SLA           | &#10004 |    |
| プラットフォームサービス（データベース、検索など） | &#10004 |    |
| アーキテクチャと高可用性              | &#10004 |    |
| アップグレード                   | &#10004 |    |
| 四半期ごとのアップデート              | &#10004 |    |
| パッチ、セキュリティホットフィックス        | &#10004 |    |
| ソフトウェアのトラブルシューティング        | &#10004 |    |
| セキュリティスキャン／SAST／DAST      | &#10004 |    |

Liferay Experience Cloud SaaSでは、価値の低い活動をすべてLiferayが引き受け、常に行っている：

* DXPのソフトウェアは、四半期ごとに更新され、更新されたバックアップコピーに対してテストされ、パッチやソフトウェアアップデートが適用されたときに何も壊れないことを確認するためにカスタマイズされたテストスイートでテストされます。
* Liferayは、ソフトウェアやインフラのトラブルシューティング、セキュリティスキャンなどを行う。

これらのことはすべて、ライフバンクのような顧客がビジネス上の問題解決に完全に集中できることを意味する。

Liferayエクスペリエンス・クラウドはSaaSサービスですが、覚えておくことが重要です。 まさにDXP-as-a-serviceである。 これは、Liferay Digital Experience Platformの完全版であり、カスタマイズや拡張が可能ですが、SaaS展開モデルで提供されます。

ターンキーアプリケーションではなく、顧客が必要とする特定のアプリケーションを構築して立ち上げる必要がある。

この提供により、顧客はアナリティクスやコマースなど、Liferayのすぐに使える機能をすべて利用することができ、Client Extensionsを介してカスタム機能をホストするクラウド環境と、それらのカスタマイズにも拡張されたすべてのセキュリティレイヤーのメリットを享受することができます。

Liferayはセキュリティに真剣に取り組んでおり、ISO 27001、ISO 27017、ISO 27018、SOC 2、HIPAAなどの主要なセキュリティ規格の認定を受けています。

Liferay Experience Cloud SaaSには、顧客のデータの安全を保証するバックアップ・ルーチンとディザスタ・リカバリ・プランもあり、さまざまなレイヤーでサイバー脅威や攻撃を打ち負かすAIベースのツールもある。

**ライフェエクスペリエンス・クラウド SaaS - ビジネス・バリュー**

要約すると、Liferay Experience Cloud SaaSは次のようなビジネス価値を提供できる：

* **柔軟性と拡張性** - プラットフォームを拡張する方法とともに、多くの統合オプションが利用可能です。 オートスケーリングはLiferayに含まれており、管理されています。
* **コアでの弾力性** - カスタマイズはLiferayコアから切り離されるため、アップグレードがよりシンプルで迅速になります。
* **常に最新** - Liferayはインフラを最新に保ち、Liferay DXPに定期的にアップデートを適用します。
* **ソリューション重視** - お客様は、すぐに使える機能を使ってビジネス上の問題を解決することができます。
