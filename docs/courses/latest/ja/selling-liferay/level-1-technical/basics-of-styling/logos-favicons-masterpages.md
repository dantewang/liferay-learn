# スタイリングの基本

**概要**

* デモを成功させる鍵は、観客を惹きつけることだ。
* 既存のデモのルック＆フィールを更新することは、エンゲージメントを生み出す効果的な方法だ。
* サイトロゴとファビコンを、見込み客が知っているグラフィックに置き換えることで、このプロセスを大幅に助けることができる。
* マスターページは、ヘッダー、フッター、ナビゲーションメニューなど、すべてのページに共通する要素を定義できるページテンプレートです。

* 本モジュールのライブワークショップの [録画](https://learn.liferay.com/web/guest/d/se1-5-basics-of-styling-in-liferay) をご覧ください。

## サイトロゴとファビコン

![サイトページ設定メニューでは、サイトロゴとファビコンを変更することができます。](./logos-favicons-masterpages/images/01.png)

スタイルブックを使って色、フォント、間隔などを変更するだけでなく、サイトロゴとファビコンの変更も迅速かつ簡単に行うことができます。

これらの変更を行うには、Pages Configurationにアクセスし、Look and Feelを選択してください。 変更は、新しい画像をアップロードすることによって、または以前にサイトにアップロードされた既存の画像を使用することによって行うことができます。

スタイルブックと同様に、これらの変更はサイト内のすべてのページに影響し、リアルタイムで確認することができます。

```{note}
多くのウェブサイトはファビコンに特定のファイル形式（ICOファイルなど）を要求しますが、LiferayはPNGなどほとんどの画像ファイル形式を受け入れるので、目的の画像を適用するのがさらに簡単になります。
```

## マスターページ

![マスターページのテンプレートは、テンプレートに基づいてすべてのページに適用される共通の要素を定義します。](./logos-favicons-masterpages/images/02.png)

マスターページは、ヘッダー、フッター、ナビゲーションメニューなど、すべてのページに共通する要素を定義できるページテンプレートです。

各マスターページには、1つ以上のドロップゾーンがあります。 ドロップゾーンは、各ページが独自のコンテンツを持つためのコンテナとして機能する要素です。

マスターページの主な利点は以下の通りである：

* 同じマスターから作られた各ページに一貫したルック＆フィールを提供する。
* 共通の要素が自動的に適用されるため、各ページの開発にかかる時間と労力を節約できる。

## なぜデモのルック＆フィールを変えることが重要なのか？

一見すると、このモジュールで提案されている単純な変更は些細なことに思えるかもしれないが、それらはすべて、見込み客にとってより適切で個人的なものにすることで、デモのインパクトを変えるということを理解することが重要である。

この種の変更で対処される重要なポイントは以下の通りだ：

* どのようなデモの目標も、観客を惹きつけることだ。
* ほとんどの観客にとって、DXPの実戦を見るのは今回が初めてだろう。
* 彼らが共感できるものを見ることは、エンゲージメントに大いに役立つ。
* もし、見込み客が自社のロゴとコーポレートカラーを見ることができれば、Liferayでアプリケーションを構築することをより容易に想像することができます。

**観客動員数**

デモのゴールは、その見込み客と関わること、つまり、Liferayでビジネス上の問題を解決できるという信頼を築くことです。

**初めてDXPを見る**

このデモは、聴衆の誰もがデジタル・エクスペリエンス・プラットフォームの動作を見る初めての機会となる可能性が高い。

**デモ関連**

デモで見込み客に見せる最も重要なことは、彼らが共感できるものである。 もし、彼らが抱えている問題と同じ問題を解決している既存のデモがあれば、それは完璧な状況だ。 同じようなものがない場合は、誰もが共感できるようなユースケース、つまり生活の中で使えるようなデモを探す。

**信頼を築く**

いずれにせよ、エンゲージメントを大幅に向上させ、見込み客がデモに共感できるようにする素晴らしい方法は、彼らの世界で使っているようなデモを見せることだ。 例えば）会社のウェブサイトのロゴや色を借りることで、彼らのためにLiferayで作られたアプリケーションがどのように見えるかを想像するのにとても役立ちます。

## 高度なスタイリング・オプション

このモジュールで説明したことは、スタイリングの基本であることに注意することが重要です。 Liferayには、Liferayアプリケーションのルック＆フィールをより洗練されたものに変更するための機能が他にもたくさんあります。

このモジュールでは、既存のデモに加えることができる、シンプルで効果的な変更に焦点を当て、デモを観客にとってより魅力的なものにします。

Liferayアプリケーションのスタイリングに影響を与えるためにできることについては、この後のモジュール、トレーニングコース、Liferay製品のドキュメントを参照してください。
