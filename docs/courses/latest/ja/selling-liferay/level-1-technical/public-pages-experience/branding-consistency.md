# ブランディングと一貫性

**概要**

* 多くの顧客が一貫したルック＆フィールを提供するのに苦労している

## スタイルとテーマ - ブランディングと一貫性

Liferayのスタイリングとテーマについて話すとき、見込み客にとって本当に意味するのはブランディングと一貫性です。 デモでは、全体的な雰囲気と、その雰囲気がページを通してどのように伝わっていくかをすでに紹介した。 さて、ここで重要なのは、これまで示されてきたルック＆フィールが非常に柔軟であり、また一貫したものであるという事実を確立することである。

アプリケーションは複数のシステムから作られており、それぞれがルック＆フィールに対して独自のアプローチを持っていたり、ルック＆フィールが個々のコンテンツ作成者に任されていたりするため、多くの顧客がルック＆フィールの一貫性のなさに苦労している。 これは、画像のズレ、色の不統一、フォントサイズの不統一、あるいはフォントそのものに関連する可能性がある。

![Liferayのブランディングと一貫性は、テーマとスタイルブックを使って提供されます。](./branding-consistency/images/01.png)

Liferayのブランディングと一貫性に関して、これは主にテーマとスタイルブックの使用を通じて提供されます：

* 最新の設計原則
* マルチサイト、マルチレベルサイト
* 色、テキスト、画像、スタイルブック
* サイトとページテンプレート

### 最新の設計原則

最新のデザイン原則 "というフレーズは、やや専門的で曖昧に思えるかもしれませんが、このような言葉でLiferayのスタイリングについて語ることの価値は、Liferayアプリケーションが例えばレスポンシブデザインのような原則をサポートしているという確信を提供し、その結果、見込み客はLiferayで最高のルックアンドフィールを実装できるという安心感を得ることができるということです。

### マルチサイト、マルチレベルサイト

もう一つのトピックは、Liferayのマルチサイト、マルチレベルサイトをサポートする能力で、各ページ、そして各サイトは全体的に同じルック＆フィールを共有することができます。 デザイン・チームは、ブランド・ガイドラインを実施するテーマを実装し、異なるスタイル・ブックを定義することで、異なるサイトにこのテーマを適用することができます。 おそらく、一般向けウェブサイトでは1つの配色を使用し、顧客セルフサービス・ポータルでは異なる配色を使用しているのだろう。 これらはすべて、一貫性の必要性を支持すると同時に、バリエーションを許容している。

### 色、テキスト、画像、スタイルブック

上記のように、Liferayではデザイナーがスタイルブックを作成し、各テーマを拡張することで、コードを書くことなくルック＆フィールのあらゆる面を調整することができます。 サイトごとに異なるバリエーションを持つ複数のスタイルブックをサポートすることに加え、ブランディングやブランディング・ガイドラインが変更された場合でも、これらを簡単に適応させることができます。 さらに、画像に関して言えば、Liferayにはアダプティブ・メディア機能があり、異なる画面解像度に適切なサイズの画像を自動的に供給することで、パフォーマンスを最適化し、ユーザー体験を向上させます。

### サイトとページのテンプレート

サイトテンプレートとページテンプレートにより、新しいサイトやページを素早く簡単に作成することができ、基本的なルック＆フィールを超えて、各ページの一貫したレイアウトを定義することで、一貫性をさらにサポートします。

## スケーラビリティと市場投入までの時間の短縮

どのようなデモの場面でも、観客にはさまざまな役割を持つ人々が含まれる。 エンタープライズアーキテクトや開発者のように、技術的な観点からLiferayを評価している人もいます。 CTO、CFO、CEOなどの経営幹部がLiferayを評価する際に考慮する点は、全体的なコスト、市場投入までの時間、拡張性です。

これらのトピックは様々な方法で扱われるが、その中でも特に重要なのは、セールス・サイクルのかなり後の段階で説明される導入である。 ここでの関連は、ページ、サイト、コンテンツの作成速度、ルック＆フィールを通じて提供される一貫性など、このモジュールで説明したすべての機能が、Liferayアプリケーションが迅速な市場投入とスケーラビリティをサポートしているという信頼を築くということです。

### 拡張性と柔軟性

公開ページのデモを終え、見込み客の特定のユースケースに深く入り込む前に、以下のトピックを紹介することで、Liferayアプリケーションがスケーラブルで柔軟であるという確信をさらに深めることができます：

![Liferayは拡張性と柔軟性に優れています。](./branding-consistency/images/02.png)

**移動中の変更**

Liferayは、開発サイクルを経ることなく、変更をライブで公開する機能を通じて、オンゴーイングで変更する能力をサポートします。

**マーケティング・ユーザー向けにより多くのコントロール**

これまでのデモの多くは、マーケティング・ユーザーのような非開発者がいかに簡単にコンテンツやページを作成・更新できるかを紹介してきた。 これらのアクティビティに対する開発者の依存度が低いため、Liferayアプリケーションは見込み客のニーズに応じて迅速に進化し、変更することができます。

**プレビューと出版物**

変更がライブで行われることについて話すとき、これは単に誰でも何かを変更できるということではなく、これらの出版物やプレビューがLiferayのロールベースのアクセスコントロールによって保護され、許可されたユーザーだけが変更できることを保証するという事実をカバーすることが重要です。

**セグメンテーションとターゲティング**

Liferayのセグメンテーションとターゲティング機能によって、コンテンツやアプリケーションをユーザーが属するセグメントに基づいて特定のユーザーにターゲティングすることができます。

**アナリティクスとABテスト**

セグメンテーションとターゲティングに密接に関連するのは、分析機能とABテストのような機能である。
