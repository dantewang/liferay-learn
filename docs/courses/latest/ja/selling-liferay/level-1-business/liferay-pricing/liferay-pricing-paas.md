# Liferayエクスペリエンスクラウドセルフマネージド（PaaS）価格

**概要**

* Liferayは、Liferay DXPを含む3つのデプロイアプローチを提供します：
  * Liferayエクスペリエンスクラウド（SaaS）
  * Liferayエクスペリエンスクラウドセルフマネージド（PaaS）
  * DXP Self-Hosted
* Liferay Experience Cloud SM はインスタンスベースのモデルを採用しています。
* プレパッケージド・バンドルが最もお得
* ライフェエクスペリエンスクラウドSMの新価格モデルを検討中

## 背景

正当な理由により、Liferay SaaSの提供は見込み客にとって適切ではないが、クラウドが許容できる場合には、Liferay PaaSの提供、Liferay Experience Cloud Self-Managedが提供する展開オプションとなり得る。

LiferayのPaaSでは、DXPのすべての機能が含まれており、DXP 7.4ではコマースとアナリティクスの機能も含まれています。 Liferayはまた、インフラストラクチャーとプラットフォームサービスを提供し、管理しますが、アップグレード、アップデート、パッチ、セキュリティ・ホットフィックス、ソフトウェアのトラブルシューティング、アプリケーションの配信など、その他のほとんどの活動はお客様が責任を負うことになります。

* 本モジュールのライブワークショップの [録画](https://learn.liferay.com/web/guest/d/sa1-2b-liferay-pricing-paas) をご覧ください。
* 上記ワークショップで使用されたプレゼンテーション資料はこちら [PDF](https://learn.liferay.com/documents/d/guest/sa1-2b-liferay-pricing-paas-pdf) からダウンロードしてください。

## Liferay Experience Cloud SM 価格の概要

現在のところ、LiferayのPaaSの価格モデルはインスタンスベースで、2つの主要なアプローチがある：

* アラカルト
* パッケージ・バンドル

この価格モデルは現在、簡素化し、先ほど説明した新しいSaaS価格モデルとの整合性を高めることを目的に、見直しが行われているところだ。

このレビューが進行中であるため、このモジュールではアラカルト・オプションの詳細については説明しないが、利用可能な各パッケージ・バンドルについて簡単に説明する。

### 価格見直しの理由

私たちは、Liferayとパートナーセラーの皆様が、一般的な価格という観点から、様々な製品間をナビゲートしやすくしたいと考えています。 また、特にPaaSの価格設定については、買い手にとって適切なサイズの見積もりと「パッケージ」を簡単にまとめられるようにしてほしい。

最も重要なことは、買い手は自分のニーズがどのように価格設定されているのか、そして短期的にも長期的にもそれに影響を与える要素は何なのかを理解することである。 評価され、検討されていることのいくつかは以下の通りである：

* 技術的に変更する必要があるのは、PaaSの価格設定とSaaSの価格設定をできるだけ一致させることだ。 特に、インスタンス・ベースの価格設定からの脱却の可能性に関連している。
* また、新しいSaaSの価格設定が効果的に機能していることを確信することも必要であり、この確信のためには十分なデータが必要である。

### バンドル

現在の価格モデルにはかなりの数の項目がある。 最も複雑なアプローチは、買い手が必要とするコンフィギュレーションを構築するために個々の項目を選択する必要があるフルカスタマイズ可能なモデルを使用することである。

別のアプローチとしては、あらかじめパッケージ化された3つのバンドルから1つを選択し、アラカルト・リストからさらに項目を追加することでこれらをカスタマイズし、より強固なものにする。

バンドルを検討する際には、以下の点に注意すること：

* バンドルは当分の間利用可能だが、予定されている価格モデルの改定では継続されないかもしれない。
* バンドルは個別のラインアイテムから割引されるため、3年間の複数年契約でのみ提供され、顧客はリファレンスカスタマーとなることに同意しなければならない。

もし顧客が3年間のコミットメントとリファレンスになることを約束できないなら、アラカルト・アプローチを使うという選択肢もある。

3つのバンドルが用意されている：

* Liferay Experience Cloud SM - プロソリューションバンドル
* Liferay Experience Cloud SM - ビジネスソリューションバンドル
* Liferay Experience Cloud SM - エンタープライズソリューションバンドル

![Liferay Experience Cloud SMには、Pro、Business、Enterpriseの3つのバンドルがあります。](./liferay-pricing-paas/images/01.png)

## Liferay Experience Cloud SM Pro バンドル

最も小さいバンドルはProバンドルで、プロジェクトの重要度が低く、ミッションクリティカルではない中小規模のお客様に最適です。

このバンドルには以下の環境が含まれる：

* 1x 標準在庫生産
* 1x 標準可用性 非生産（例． 開発、テスト、UAT)

どのバンドルでも、さらなる容量、追加環境、コネクター、エンタープライズ・サーチの追加、マネージド・サービスの購入が可能です。 Pro Bundleをご購入のお客様は、Application Performance Monitoringを追加することもできますが、これはBusiness BundleとEnterprise Bundleに追加料金なしで含まれていることは注目に値します。

Liferayのクラウドセルフマネージドプロジェクトの経験がないパートナーは特に、実装の成功を保証するためにLiferayのSMEを含めることが非常に重要です。

## セルフマネジメント・ビジネス・バンドル

2つ目のバンドルは、より大規模でミッションクリティカルなアプリケーション向けにより高性能なものだ。 Business BundleはPro Bundleをベースに、Production向けの高可用性を追加し、高可用性のUAT環境を追加します。 含まれる環境のリストは以下の通り：

* 1x 高可用性プロダクション
* 1x 高可用性UAT
* 1x 標準稼働率 非生産

さらに、ビジネス・バンドルにはアプリケーション・パフォーマンス・モニタリングが含まれます。

基本的なサブジェクト・マター・エキスパート・パッケージが含まれ、顧客はGo-Live SMEパッケージか、10週間にわたり毎週定期的に行われるSMEガイダンスのいずれかを選択できる。

プロ・バンドルと同様に、容量、環境、コネクター、エンタープライズ・サーチの追加、マネージド・サービスの購入が可能です。

## セルフマネージド・エンタープライズ・バンドル

3つ目のバンドルはエンタープライズ・バンドルで、最も要求の厳しいエンタープライズ顧客とその最も重要なプロジェクトのために設計されています。 高可用性の本番環境とUAT環境、開発用の標準的な非本番環境が含まれる。  また、地域横断的なディザスタリカバリが可能な高可用性ディザスタリカバリ環境も追加される：

* 1x 高可用性プロダクション
* 1x 高可用性UAT
* 1x 標準稼働率 非生産
* 1x 高可用性ディザスタリカバリ

ビジネス・バンドルと同様に、アプリケーション・パフォーマンス・モニタリングといくつかのサブジェクト・マター・エキスパートが含まれますが、エンタープライズ・バンドルには、より幅広いSMEパッケージのリストから2つのSMEパッケージが追加料金なしで含まれます。

すべてのバンドルと同様に、さらに容量を追加したり、エンバイロメント、コネクター、エンタープライズ・サーチを追加したり、マネージド・サービスを購入したりすることも可能です。

## 概要

* お客様がLiferay Experience Cloud Self-Managedを購入する最も費用対効果の高い方法は、あらかじめパッケージ化されたバンドルのいずれかを購入することです。
* ベースレベルのProバンドルは最小のバンドルで、Liferay Experience Cloud Self-Managedの最小構成に対するLiferayの推奨を表しています。
* すべてのバンドルは、利用可能な幅広い追加機能のいずれかで拡張することができます。
* 顧客は、3年間のコミットメントとリファレンスになることができない場合は、アラカルトの製品リストから購入することができますが、バンドルが提供する割引の恩恵を受けることはできません。
