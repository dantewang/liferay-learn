# Liferayが解決する課題とは - 水平ソリューション

**概要**

* Liferayは、次のような水平ソリューションに最適です。
  * カスタマーポータル
  * デジタル・コマース・サイト
  * 外部ウェブサイト
  * サプライヤー・ポータル
  * パートナー・ポータル
  * 最新のイントラネット
  * その他
* 顧客はしばしば1つのプロジェクトから始め、後に他のプロジェクトを追加する。
* これらのソリューションはいずれも、Liferayのコア機能の多くを利用しています。

## 背景

ここでは、Liferayがうまく解決するビジネス課題の種類について詳しく説明します。 Liferayの主な強みの1つはその柔軟性で、あらゆる業界のあらゆる問題を解決するためにLiferayを使うことができます。

しかし、私たちが特に相性の良いソリューションや業界もあり、そうしたソリューションや業界に合致する新規事業機会に集中することで、成功の可能性を最も高くしたいと考えています。

```{note}
* このモジュールのライブワークショップの [recording](https://learn.liferay.com/web/guest/d/l0-5a-what-challenges-do-we-solve-overview) をご覧ください。
* ライブ・ワークショップで使用されたプレゼンテーションの [PDF](https://learn.liferay.com/documents/d/guest/l0-5a-what-challenges-do-we-solve-overview-pdf) をダウンロードしてください。
```

## ソリューションの定義

ソリューションとは、一般的なビジネスニーズや一連の課題であり、Liferay Experience Cloudはその解決に適しています。

## 土地と拡大

ほとんどの顧客は特定のソリューションを求めてLiferayを訪れます。 時間が経つにつれて、Liferayが解決に役立つ新たなビジネス上の課題が見つかることもよくあります。

例えば、よくあるパターンとして、顧客が最新のイントラネット・ソリューションを探しているときにLiferayを見つけたとします。 顧客はイントラネットを導入した後、同じ機能と性能（そして同じサブスクリプション）が、公開ウェブサイトや認証された顧客ポータルでの課題解決にも役立つことに気づく。

![基本的なソリューションから始め、社内で徐々にLiferayの利用を広げていくことは、拡張と成長への素晴らしいアプローチです。](./what-challenges-does-liferay-solve-horizontal-solutions/images/01.png)

## 水平ソリューション

各タイルをクリックすると、Liferayで最もよく解決される6つのソリューションについて読むことができます。 それぞれのケースで

* ソリューションの定義
* ソリューションの購買層
* 主なメリット
* Liferay DXPの機能を採用

何から始めたらいいかわからない？ カスタマーポータルから始めよう。
