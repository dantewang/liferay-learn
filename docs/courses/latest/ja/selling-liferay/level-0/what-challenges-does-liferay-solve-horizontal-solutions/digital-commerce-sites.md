# デジタル・コマース・サイト

デジタル・コマース・サイトは、人々がインターネット上で物理的な商品、サービス、デジタル製品を購入することを可能にする。

## ターゲット・オーディエンス

デジタル・コマースの購買層は通常、マーケティングとセールスの両組織に属する。 多くの統合要件では、ほとんどの場合、IT部門が大きく関与する：

* CMO
* CRO
* 部長、取締役、副社長、マネージャー
  * マーケティング
  * デジタル・エクスペリエンス
  * デジタル・コマース
  * 事業開発、
  * デジタル・チャンネル
  * eコマース
* CIO／IT部門責任者

## 主なメリット

デジタルコマースサイトの基盤としてLiferay DXPを使用することで、見込み客と話す際に強調したいいくつかの重要な利点が得られます：

* 効果的なデジタル・コマース・サイトは、**オンライン・ショッピングを簡単にする**べきである。顧客が適切な製品を発見し、適合性と互換性を確認できるようにします。Liferay DXPは、強力な検索機能とともに、コンテンツ管理とコマース機能のネイティブな統合によって価値を提供します。
* 優れたデジタルコマースサイトは、単なる製品カタログではありません。また、複雑な注文ルールや複雑な商品であっても、**注文を簡素化**します。これは特にB2Bで一般的だが、B2Cの要因にもなり得る。注文が完了すると、顧客は注文や出荷状況を確認できるため、**フォローアップが簡素化**される。
* 販売チームの中には、デジタル・コマース・サイトを競争相手と見なす者もいる。しかし、デジタル・コマースは、セルフサービスによって簡単に達成できる平凡な作業から解放し、より価値の高い作業や対話に集中できるようにすることで、**販売チームやサービスチーム**を支援するものであるべきだ。
* シームレスで直感的なコマース体験により、顧客は追加購入のために再び来店し、Liferayのマーチャンダイジングとディスカウント機能により、**注文のサイズと頻度を増加させることができます**。
* 最後に、Liferay DXPはお客様の**効率的な成長と拡張**を支援します。
