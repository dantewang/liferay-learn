# デジタル・コマース・サイト

デジタル・コマース・サイトは、人々がインターネット上で物理的な商品、サービス、デジタル製品を購入することを可能にする。

## ターゲット・オーディエンス

デジタル・コマースの購買層は通常、マーケティングとセールスの両組織に属する。 多くの統合要件では、ほとんどの場合、IT部門が大きく関与する：

* CMO
* CRO
* 部長、取締役、副社長、マネージャー
  * マーケティング
  * デジタル・エクスペリエンス
  * デジタル・コマース
  * 事業開発、
  * デジタル・チャンネル
  * eコマース
* CIO／IT部門責任者

## 主なメリット

デジタルコマースサイトの基盤としてLiferay DXPを使用することで、見込み客と話す際に強調したいいくつかの重要な利点が得られます：

* 効果的なデジタル・コマース・サイトは、 **、オンライン・ショッピングを容易にするものでなければならない**。 顧客が適切な製品を見つけ、フィット感や互換性を確認できるようサポートする。 Liferay DXPは、強力な検索機能とともに、コンテンツ管理とコマース機能のネイティブな統合によって価値を提供します。
* 優れたデジタル・コマース・サイトは、単なる商品カタログではない。 また、 **、複雑な注文規則や複雑な商品であっても、**。 これは特にB2Bでよくあることだが、B2Cの要因になることもある。 一旦注文が入ると、 **フォローアップが簡素化され、** 注文と出荷のステータスが可視化される。
* 営業チームの中には、デジタル・コマース・サイトを競争相手と見なす者もいる。 しかし、デジタル・コマースは、 **、セルフサービスによって簡単に達成できる平凡な作業から解放し、より価値の高い作業や対話に集中できるようにすることで、営業チームやサービスチーム**。
* シームレスで直感的なコマース体験により、顧客は追加購入のために再び来店し、Liferayのマーチャンダイジングとディスカウント機能により、 **注文サイズと注文頻度を増やすことができます**。
* 最後に、Liferay DXP は顧客 **の成長と効率的な拡張を支援します**。
