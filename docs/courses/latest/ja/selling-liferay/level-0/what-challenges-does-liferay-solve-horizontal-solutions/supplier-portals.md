# サプライヤー・ポータル

サプライヤー・ポータルは、企業が商品やサービスのサプライヤーとつながり、協力することを可能にする。  サプライヤー・ポータルのユーザーの多くは、複雑なサプライチェーンを持つメーカーである。 どのような業種の企業でも、ある程度はサプライヤーに頼らざるを得ないが、成長するにつれてサプライヤーとの関係管理が難しくなるため、サプライヤー・ポータルから利益を得ることができる。

## ターゲット・オーディエンス

サプライヤー・ポータル・プロジェクトは通常、調達部門が主導する。 特にERPとの）統合要件があるため、財務部門とIT部門のリーダーも深く関わっている：

* 調達部長
* CFO/VP ファイナンス
* CIO／IT部門責任者
