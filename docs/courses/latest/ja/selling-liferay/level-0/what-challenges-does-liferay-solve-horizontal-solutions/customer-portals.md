# カスタマーポータル

カスタマーポータルは、顧客がベンダーから購入した製品やサービスに関するすべての関連情報にアクセスできる単一のポイントを顧客に提供する。

その鍵となるのは、組織の顧客／エンドユーザーが、製品やサービスにアクセスするためにソリューションを利用することである。

## ターゲット・オーディエンス

カスタマーポータルの主要な購買層は、カスタマーサクセス、セールス、オペレーション、マーケティングです。 IT部門が関与することもある：

* カスタマー・サクセス／エクスペリエンス部門責任者
* オペレーション部長
* CFO/VP ファイナンス
* マーケティング担当CMO/VP
* CIO／IT部門責任者

## 主なメリット

各組織にはそれぞれ独自のニーズがあるが、カスタマーポータルの主な利点は、すべてのオーディエンスに響くはずである：

* 顧客ポータル **オンボーディング** を合理化し、顧客が関係を最大限に活用できるようパーソナライズされた体験を提供する。
* 顧客ポータル **顧客** にアカウント管理機能とセルフサービスを提供し、日常的な購入、支払い、アカウント情報の管理など、一般的なビジネス・タスクを処理できるようにする。
* カスタマー・ポータルは、 **顧客が答えを見つけるのを助け、** 自らの問題を解決することができる。
* 顧客は自分の都合でビジネスを進められることを期待しており、顧客ポータルは **24時間365日の可用性を提供する**を支援する。
