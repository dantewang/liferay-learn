# 外部Webサイト

外部Webサイト（またはコーポレートサイト）は、企業のサービスについての情報を見込み顧客に提供することで、新たなビジネスチャンスを拡大します。 Liferay DXPは、このような深くパーソナライズされたオムニチャネルエクスペリエンスを構築するのに適しています。

## ターゲットオーディエンス

外部Webサイトの購買層は、最高マーケティング責任者（CMO）や最高収益責任者（CRO）のような、上位レベルの関係者です。 より大規模で多様性のある企業では、IT部門だけでなく、マーケティング、デジタルエクスペリエンス、事業開発、デジタルチャネル部門のリーダーも含まれます。

* CMO
* CRO
* 以下それぞれの担当の部長、ディレクター、バイスプレジデント、マネージャー
  * マーケティング
  * デジタルエクスペリエンス
  * 事業開発
  * デジタルチャネル
* 最高情報責任者（CIO）／IT部門責任者

## 主なメリット

Liferay DXPで外部Webサイトを構築する理由には、次のようなものがあります。

* Liferay DXPは、直感的なコンテンツ管理ツールと強力なセグメンテーション機能を備えており、 **パーソナライズされたエクスペリエンス*****を通じて*****新規顧客を獲得** できます。
* デザインシステムとエクスペリエンス管理機能により、顧客はあらゆる **デジタルタッチポイント** で、 **ブランドを管理** できます。
* ローコードやノーコードによるアプリケーション構築、データモデリングやプロセス管理などの機能により、開発者はプラットフォームの **設定** 、 **カスタマイズ** 、 **拡張** を行うことができます。
* Liferay DXPのアナリティクスおよび最適化ツールにより、 **コンテンツパフォーマンスに関するインサイト** を引き出すことができます。
