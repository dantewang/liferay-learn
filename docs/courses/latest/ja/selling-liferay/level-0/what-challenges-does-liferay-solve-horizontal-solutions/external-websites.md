# 外部ウェブサイト

外部ウェブサイト（または公開ウェブサイト）は、組織が提供するものについての情報を見込み客に提供することで、新たな機会を促進するのに役立つ。 Liferay DXPは、このような深くパーソナライズされたオムニチャネル体験を構築するのに適しています。

## ターゲット・オーディエンス

外部ウェブサイトの購買層は、最高マーケティング責任者（CMO）や最高収益責任者（CRO）のような、より高いレベルの利害関係者です。 より大規模で多様性のある組織では、IT部門だけでなく、マーケティング、デジタル・エクスペリエンス、ビジネス開発、デジタル・チャネル部門のリーダーも含まれる：

* CMO
* CRO
* 部長、取締役、副社長、マネージャー
  * マーケティング
  * デジタル・エクスペリエンス
  * 事業開発
  * デジタル・チャンネル
* CIO／IT部門責任者

## 主なメリット

Liferay DXPを使って外部ウェブサイトを構築する理由には、次のようなものがあります。

* Liferay DXPは、**パーソナライズされた体験**を通じて**新規顧客を獲得**するための直感的なコンテンツ管理ツールと強力なセグメンテーション機能を提供します。
* デザインシステムとエクスペリエンスマネジメント機能により、顧客はあらゆる**デジタルタッチポイント**で**ブランドをコントロールすることができます。
* ローコード/ノーコードのアプリケーション構築、データモデリング、プロセス管理機能により、開発者はプラットフォームを**構成**、**カスタマイズ**、**拡張**することができます。
* Liferay DXPは、**コンテンツパフォーマンスの**洞察を明らかにするための分析と最適化ツールを提供します。
