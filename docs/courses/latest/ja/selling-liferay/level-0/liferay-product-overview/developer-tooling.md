# 開発者向けツール

Liferayは、標準的なビルドスクリプト、軽量なCLIユーティリティ、Eclipseベースの完全なIDEを含む様々な開発者ツールを提供します。 Liferayの開発者ツールは、初心者から長年コードを書いてきたエキスパートまで、すべての開発者に役立ちます。
