# 分析と最適化

改善点を特定するアナリティクスにより、デジタル・エクスペリエンスの成果を最適化する。 顧客がどのようにコンテンツに接するかを理解し、より適切で、カスタマイズされた体験を提供する。 内蔵のダッシュボードとレポーティングは、ビジネスチームが関連するメトリクスを掘り下げて結果を監視し、改善領域を特定するのに役立ちます。
