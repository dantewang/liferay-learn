# デザイン・システムとエクスペリエンス・マネジメント

デザイン・システムとエクスペリエンス・マネジメントは密接に連携しており、デザイナーはサイト制作者がサイトやページを構築するための基礎を作ることができる。 デザインシステムを使えば、開発者でないユーザーでも、ほとんど、あるいは潜在的に開発者が関与することなく、Liferayアプリケーションのルックアンドフィールを作成することができます。 一方、開発者は、すぐに使える機能を拡張することができ、それを開発者以外のユーザーが使用することで、エンドユーザー体験をさらに向上させることができる。

![Liferayデザインシステムは、サイトのデザインと構成を箱から出してすぐにコントロールできます。](./design-systems-and-experience-management/images/01.png)
