# コラボレーションと知識の共有

ユーザーが自分自身でより早く答えを見つけ、組織的なコラボレーションを通じてより複雑な課題を解決できるようにします。 Liferay DXPのコラボレーションとナレッジ共有ツールは、顧客、パートナー、従業員が自ら必要な情報を見つけられるようにします。

![Liferayは、すぐにリッチなブログ体験を提供します。](./collaboration-and-knowledge-sharing/images/01.png)
