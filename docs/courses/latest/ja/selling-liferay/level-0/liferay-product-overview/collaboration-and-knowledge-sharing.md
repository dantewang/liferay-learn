# コラボレーションとナレッジの共有

ユーザーが自分自身でより早く答えを見つけ、組織的なコラボレーションを通じてより複雑な課題を解決できるようにします。 Liferay DXPのコラボレーションとナレッジ共有ツールにより、顧客、パートナー、従業員は自ら必要な情報を見つけられるようになります。
