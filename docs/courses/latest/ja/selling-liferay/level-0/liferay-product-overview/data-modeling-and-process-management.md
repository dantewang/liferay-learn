# データモデリングとプロセス管理

ビジネスユーザーは、ビジネスアプリケーションやフォームに必要なフィールドの迅速かつ容易な定義、基礎的な機能の活用によるデータ入力の検証、および動的なフォーム入力を可能にする条件付きルールの定義が可能になります。 ユーザーが必要な業務タスクの定義、提出、レビュー及び承認が可能なワークフローエンジンにより、ビジネスプロセスを容易に実行できます。
