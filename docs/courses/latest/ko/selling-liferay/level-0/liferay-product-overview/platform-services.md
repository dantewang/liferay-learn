# 플랫폼 서비스

Liferay의 플랫폼 서비스는 관리자가 Liferay 구현의 모든 측면을 구성 및 관리하고 여기에 배포된 응용 프로그램을 구축, 구성 및 관리할 수 있는 도구와 인터페이스를 제공합니다.
