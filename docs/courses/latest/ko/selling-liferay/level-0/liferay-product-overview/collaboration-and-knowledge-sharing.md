# 협업 및 지식 공유

사용자가 스스로 질문에 대한 답변을 더 빠르게 얻고 체계적인 협업을 통해 더 복잡한 문제를 해결할 수 있도록 지원합니다. Liferay DXP의 협업 및 지식 공유 도구는 고객, 파트너 및 직원이 스스로 필요한 정보를 찾을 수 있도록 지원합니다.
