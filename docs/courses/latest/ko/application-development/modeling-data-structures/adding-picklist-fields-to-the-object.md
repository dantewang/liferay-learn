# 개체에 선택 목록 필드 추가

배포자 응용 프로그램 개체에 선택 목록 필드를 추가하는 작업은 몇 가지 추가 구성 옵션이 있지만 다른 필드 유형과 동일한 프로세스를 따릅니다.

1. *전역 메뉴* ( ![Global Menu](../../images/icon-applications-menu.png) )을 열고 *제어판* 탭으로 이동한 다음 *개체*클릭합니다.

1. Distributor Application 개체 편집을 시작하고 *Fields* 탭으로 이동합니다.

1. *Add* (![Add Button](../../images/icon-add.png))을 클릭하고 세부 정보를 입력한 후 *Save*클릭합니다.

   | 분야     | 가치             |
   |:------ |:-------------- |
   | 라벨     | 사업 유형          |
   | 필드 이름  | `businessType` |
   | 유형     | 선택목록           |
   | 선택목록   | 사업 유형          |
   | 필수적인   | 틀린             |
   | 상태로 표시 | 틀린             |

   ![비즈니스 유형 선택 목록 필드를 추가합니다.](./adding-picklist-fields-to-the-object/images/01.png)

1. 분포 지역 필드를 추가합니다.

   | 분야    | 가치                    |
   |:----- |:--------------------- |
   | 라벨    | 유통 지역                 |
   | 필드 이름 | `distributionRegions` |
   | 유형    | 다중 선택 선택 목록           |
   | 선택목록  | 유통 지역                 |
   | 필수적인  | 틀린                    |

1. 비즈니스 유통 채널 필드를 추가합니다.

   | 분야    | 가치                             |
   |:----- |:------------------------------ |
   | 라벨    | 사업 유통채널                        |
   | 필드 이름 | `businessDistributionChannels` |
   | 유형    | 다중 선택 선택 목록                    |
   | 선택목록  | 유통채널                           |
   | 필수적인  | 틀린                             |

1. 관심 있는 주문 유형 필드를 추가합니다.

   | 분야    | 가치                     |
   |:----- |:---------------------- |
   | 라벨    | 관심 있는 주문 유형            |
   | 필드 이름 | `orderTypesOfInterest` |
   | 유형    | 다중 선택 선택 목록            |
   | 선택목록  | 상거래 주문 유형              |
   | 필수적인  | 틀린                     |

1. 관심 제품 필드를 추가합니다.

   | 분야    | 가치                   |
   |:----- |:-------------------- |
   | 라벨    | 관심상품                 |
   | 필드 이름 | `productsOfInterest` |
   | 유형    | 다중 선택 선택 목록          |
   | 선택목록  | 제품 유형                |
   | 필수적인  | 틀린                   |

1. 예상 연간 구매량 필드를 추가합니다.

   | 분야     | 가치                              |
   |:------ |:------------------------------- |
   | 라벨     | 예상 연간 구매량                       |
   | 필드 이름  | `estimatedAnnualPurchaseVolume` |
   | 유형     | 선택목록                            |
   | 선택목록   | 연간 구매량                          |
   | 필수적인   | 틀린                              |
   | 상태로 표시 | 틀린                              |

1. 제품 라벨링 필드를 추가합니다.

   | 분야     | 가치                |
   |:------ |:----------------- |
   | 라벨     | 제품 라벨링            |
   | 필드 이름  | `productLabeling` |
   | 유형     | 선택목록              |
   | 선택목록   | 제품 라벨             |
   | 필수적인   | 틀린                |
   | 상태로 표시 | 틀린                |

   ![배포자 애플리케이션에 7개의 선택 목록 필드를 추가합니다.](./adding-picklist-fields-to-the-object/images/02.png)

사용자는 배포자 신청서를 작성할 때 각 필드에 대해 사전 정의된 옵션 중에서 선택할 수 있습니다. 맛있는 분재는 이러한 데이터 포인트를 사용하여 각 비즈니스 기회의 가치를 평가할 수 있습니다.

![사용자는 미리 정의된 옵션 중에서 선택합니다.](./adding-picklist-fields-to-the-object/images/03.png)

축하해요! 배포자 애플리케이션 개체에 필요한 모든 필드를 추가했습니다. 이제 개체의 양식 레이아웃과 테이블 보기 디자인을 시작할 수 있습니다.
