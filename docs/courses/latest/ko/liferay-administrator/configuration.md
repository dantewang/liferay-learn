# 설정

이야기는 Liferay가 설치되거나 LXC 또는 LXC-SM에서 인스턴스가 생성된 후에 시작됩니다. Clarity Vision Solutions의 IT 관리자는 귀하의 도움을 받아 시스템을 구성하고 사용자를 등록하여 웹 사이트와 콘텐츠를 생성할 수 있도록 해야 합니다.

범위 또는 수준의 계층 구조에서 많은 설정을 구성할 수 있습니다. 보다 세부적인 범위 설정은 그 위의 설정보다 우선 적용됩니다. 예를 들어 사이트 수준의 구성 세트는 시스템 수준이나 인스턴스 수준의 구성보다 우선합니다. 이렇게 하면 범위(예: 인스턴스 또는 사이트) 소유자가 재정의할 수 있는 적절한 기본값을 설정할 수 있습니다. 자세한 내용은 [구성 범위 이해](https://learn.liferay.com/w/dxp/system-administration/configuring-liferay/understanding-configuration-scope) 를 참조하세요.

완성된 구성에는 다음이 포함됩니다.

* 업로드된 파일을 검사하는 바이러스 백신 소프트웨어
* 두 개의 내부 사이트(파트너를 위한 B2B 사이트 및 내부 인트라넷)에 대한 별도의 가상 인스턴스
* 가상 인스턴스에 대한 별도의 사용자 인증 설정
* 연결된 메일 서버

첫 번째 단계는 시스템 설정 구성입니다.
