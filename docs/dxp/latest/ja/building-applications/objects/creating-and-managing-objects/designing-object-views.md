# オブジェクトビューの設計

{bdg-secondary}`Liferay DXP 7.4 U10以降およびPortal 7.4 GA14以降で利用可能`

オブジェクトビューは、オブジェクトのアプリケーションページにエントリーを表示するためのテーブルを定義します。 デフォルトでは、Liferayは公開されたオブジェクトごとにデフォルトビューを自動的に生成します。 このビューでは、すべてのエントリーフィールドがアルファベット順に並べられています。 ただし、カスタムビューを設計して、テーブルに含まれるオブジェクトフィールドとその順序を決定することはできます。

![オブジェクトエントリーを表示するためのカスタムビューを設計します。](./designing-object-views/images/01.png)

```{note}
カスタムビューは、カスタムオブジェクトにのみ追加できます。 システムオブジェクトに追加することはできません。
```

次の手順に従って、カスタムオブジェクトビューを設計します。

1. **グローバルメニュー**（![Global Menu](../../../images/icon-applications-menu.png)）を開き、 ［**コントロールパネル**］ タブをクリックして、 ［**オブジェクト**］ に進みます。

1. 目的の **オブジェクト** の編集を開始します。

   ```{note}
   下書きオブジェクトと公開オブジェクトの両方にビューを追加できます。
   ```

1. ［**Views**］ タブに移動し、 **追加** ボタン（![Add Button](../../../images/icon-add.png)）をクリックします。

1. **名前** を入力し、 ［**保存**］ をクリックして新しい空白のビューを作成します。 これで、フィールドを追加できます。

   ![空白のビューを作成します。](./designing-object-views/images/02.png)

1. **ビュー** をクリックして、編集を開始します。

1. ［**Basic Info**］ タブで、ビューを*デフォルトデフォルトとして使用するかどうかを決定します。

   ```{note}
   これにより、ビューがオブジェクトのアプリケーションページで使用されるようになります。 デフォルトとしてカスタムレイアウトが選択されていない場合、オブジェクトは自動的に生成されたレイアウトを使用します。
   ```
</p>

   ![ビューをデフォルトとして設定します。](./designing-object-views/images/03.png)</li>

1

［**View Builder**］ タブに移動し、 **追加** ボタン（![Add Button](../../../images/icon-add.png)）をクリックします。

1

ビューテーブルに含めるすべてのフィールドをチェックして、 ［**保存**］ をクリックします。

   選択した各フィールドは、アプリケーションページテーブルの列として使用されます。

   使用可能なオプションには、[カスタムフィールド](./adding-fields-to-objects.md)と一般的なメタデータフィールド（ID、作成者、作成日、編集日時、ワークフローステータスなど）の両方が含まれます。

   ```{note}
   ビューは、複数のエントリーを含む列には対応していません（たとえば、*多対多*の関連フィールド）。
   ```

   ![フィールドを任意の順序で配置します。](./designing-object-views/images/04.png)

1

フィールドをドラッグアンドドロップして、順序を並べ替えます。

1

［**保存**］ をクリックします。</ol>

保存すると、カスタムビューはオブジェクトエントリーの表示に使用されます。

## ビューのデフォルトのソート順の設定

{bdg-secondary}`Liferay DXP 7.4 U18以降およびPortal 7.4 GA18以降で利用可能`

次の手順に従って、オブジェクトビュー列のデフォルトの順序/並べ替えを設定します。

1. **グローバルメニュー**（![Global Menu](../../../images/icon-applications-menu.png)）を開き、 ［**コントロールパネル**］ タブをクリックして、 ［**オブジェクト**］ に進みます。

1. 目的の **オブジェクト** の編集を開始し、 ［**Views**］ タブに移動して、目的の **ビュー** を選択します。

1. ［**Default Sort**］ タブに移動し、 **追加** ボタン（![Add Button](../../../images/icon-add.png)）をクリックします。

   ![［Default Sort］タブに移動し、追加ボタンをクリックします。](./designing-object-views/images/05.png)

1. **列** と **ソート** 順序（つまり、昇順または降順）を選択します。

   ![列とソート順を選択します。](./designing-object-views/images/06.png)

1. ［**Save**］ をクリックします。

オブジェクトエントリーは、デフォルトのソート順に従ってソートされるようになりました。

複数のデフォルトのソートを作成する場合は、それらを再配置して、ビューに適用されるソートを決定できます。 ビューは、最初に配置されたソートを使用します。

![ソート順を調整して、どちらを使用するかを決定します。](./designing-object-views/images/07.png)
