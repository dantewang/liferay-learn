# 一般設定リファレンス

HTML、Paragraph、Separator、Socialの各フラグメントを除き、すべてのフラグメントにGeneral設定があります。 これらのオプションの中には、標準的なもの（例えば、可視性やフレームなど）もあれば、各フラグメントに固有のものもあります。 以下の参考資料では、「General」設定のフラグメントのみを対象としています。

以下の手順で、フラグメントの一般設定にアクセスします：

1. フラグメントをサポートするページまたはテンプレートの編集を開始します。

1. サイドバーメニューの **ブラウザ** パネル（![Browser](../../../../../images/icon-hierarchy.png)）を開き、目的のフラグメントを選択します。

   または、ページやテンプレート内の目的のフラグメントをクリックします。

1. ページ要素サイドパネルの「**一般**」タブをクリックします。

![フラグメントを選択すると、ページエレメントのサイドパネルにある一般設定にアクセスできます。](./general-settings-reference/images/01.png)

## 標準の一般設定

### フラグメントを非表示

| 項目         | 説明                                                                                                                                           |
|:---------- |:-------------------------------------------------------------------------------------------------------------------------------------------- |
| フラグメントを非表示 | ビューポートによってフラグメントを非表示にする場合は、チェックを入れてください。 この設定がページビューポートとどのように相互作用するかについて詳しくは、[フラグメントの閲覧の構成](../configuring-fragment-visibility.md)を参照してください。 |

### フレーム

| 項目            | 説明                                                                                                                           |
|:------------- |:---------------------------------------------------------------------------------------------------------------------------- |
| 幅/高さ          | フラグメントのフレームのデフォルトの高さと幅を決定します。                                                                                                |
| 幅の最小値/幅の最大値   | 異なるビューポートに表示されたときのフレームの最小幅と最大幅を決定します。                                                                                        |
| 高さの最小値/高さの最大値 | 異なるビューポートに表示されたときのフレームの高さの最小値と最大値を決定します。                                                                                     |
| オーバーフロー       | フラグメントコンテンツがそのフレームをオーバーフローすることを許容するかどうかを決定する。 デフォルトでは、フラグメントのコンテンツはそのフレームに限定されます。 オーバーフローが許可されている場合は、その表示方法を決定できます（スクロールなど）。 |

| 項目                                                                       | 説明                                                                                      |
|:------------------------------------------------------------------------ |:--------------------------------------------------------------------------------------- |
| リンク                                                                      | コンテナがURL、ページ、またはマップされたURLにリンクするかどうかを決定します。この設定により、以下のフィールドのどれが利用できるかが決まります。             |

| ページ (**ページ用**)                                                             | コンテナのLiferayページを選択します。                                                                  |
| 項目(**マップされたURLの場合**)                                                       | ドキュメントとメディア、Webコンテンツ、ブログ、またはカテゴリから項目を選択します。 選択した項目によって、マッピング用のフィールドが提供されます。             |
| フィールド(**for Mapped URL**)                                                  | コンテナにマップするURLフィールドを選択します。 有効なフィールドを選択すると、 **URL** フィールドに自動的に入力されます。                         |

| コンテンツの表示                                                                 | コンテナのコンテンツをブロック、Flex Row、Flex Columnのいずれで表示するかを決定します。Flexオプションには追加の設定があります。             |
| フレックスラップ (**for Flex Row or Flex Column**) {bdg-secondary}`7.4 U19+/GA19+` | フレックスの行と列の折り返しを設定します：折り返しなし、折り返しあり、折り返し反転あり。                                            |
| アライメント項目 (**for Flex Column**) {bdg-secondary}`7.4 U19+/GA19+`             | 列の要素がどのように整列されるかを決定する：スタート、センター、エンド、ストレッチ、ベースラインのいずれかを選択します。                            |
| ジャスティファイコンテンツ（**for Flex Row**） {bdg-secondary}`7.4 U19+/GA19+`            | 行内の要素の両端揃えを決定します：開始」、「中央」、「終了」、「間」、「周囲」のいずれかを選択します。                                     |
| コンテナの幅                                                                   | コンテナで可変を使用するか固定幅を使用するかを決定します。                                                           |

## フォームフラグメントのオプション

{bdg-secondary}`Liferay 7.4 U45+/GA45+で利用可能`

フォームフラグメントには、独自の構成オプションがあります。 アクセスするには、フラグメントをオブジェクトフィールドにマッピングする必要があります。

### フォームコンテナー

| 項目                                        | 説明                                                                                              |
|:----------------------------------------- |:----------------------------------------------------------------------------------------------- |
| コンテンツの種類                                  | カスタムオブジェクトを選択します。 フォームコンテナとそのフラグメントを、選択されたオブジェクトにマッピングします。 フォームコンテナは、一度に1つのオブジェクトにしかマッピングできません。 |
| 成功メッセージ                                   | フォーム送信の成功メッセージを現在のページに埋め込むか、ユーザーをLiferayのページや外部URLにリダイレクトするかを決定します。                             |
| サクセステキスト (**for Embedded**)                 | ローカライズ可能な成功メッセージを入力する。                                                                          |
| Preview Success State (**for Embedded**)    | 埋め込まれたサクセスメッセージをプレビューします。                                                                       |
| コンテンツの表示                                  | コンテナのコンテンツをブロック、Flex Row、Flex Columnのいずれで表示するかを決定します。Flexオプションには追加の設定があります。                     |
| Flex Wrap (**for Flex Row or Flex Column**) | フレックスの行と列の折り返しを設定します：折り返しなし、折り返しあり、折り返し反転あり。                                                    |
| アイテムの整列（**for Flex Column**)                | 列の要素がどのように整列されるかを決定する：スタート、センター、エンド、ストレッチ、ベースラインのいずれかを選択します。                                    |
| コンテンツのジャスティファイ（**for Flex Row**）。           | 行内の要素の両端揃えを決定します：開始」、「中央」、「終了」、「間」、「回り」のいずれかを選択します。                                             |
| コンテナの幅                                    | コンテナで可変を使用するか固定幅を使用するかを決定します。                                                                   |
