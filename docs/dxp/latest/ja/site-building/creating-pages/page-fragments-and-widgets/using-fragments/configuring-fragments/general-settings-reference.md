# 一般設定リファレンス

HTML、Paragraph、Separator、および Social フラグメントを除くすべてのフラグメントに General 設定があります。 これらのオプションには、標準的なもの（例：visibility、frame）もあれば、各フラグメントに固有のものもある。 以下の参考資料は、「一般」設定のフラグメントのみです。

フラグメントの一般設定にアクセスするには、次の手順に従います。

1. フラグメントに対応したページやテンプレートの編集を開始します。

1. サイドバーメニューから **ブラウザ** パネル (![Browser](../../../../../images/icon-hierarchy.png)) を開き、目的のフラグメントを選択します。

   または、ページやテンプレート内の目的のフラグメントをクリックします。

1. ページ要素サイドパネルの「**一般**」 タブをクリックします。

![フラグメントを選択すると、ページ要素のサイドパネルにある一般設定にアクセスできます。](./general-settings-reference/images/01.png)

## 標準の一般設定

### フラグメントを非表示

| 項目         | 説明                                                                                                                                                  |
|:---------- |:--------------------------------------------------------------------------------------------------------------------------------------------------- |
| フラグメントを非表示 | チェックボックスをオンにすると、異なるビューポートでフラグメントを非表示にすることができます。 この設定がページビューポートとどのように相互作用するかについて詳しくは、[フラグメントの閲覧の構成](../configuring-fragment-visibility.md)を参照してください。 |

### フレーム

| 項目            | 説明                                                                                                                           |
|:------------- |:---------------------------------------------------------------------------------------------------------------------------- |
| 幅/高さ          | フラグメントのフレームのデフォルトの高さと幅を決定します。                                                                                                |
| 幅の最小値/幅の最大値   | 異なるビューポートに表示されたときのフレームの最小幅と最大幅を決定します。                                                                                        |
| 高さの最小値/高さの最大値 | 異なるビューポートに表示されたときのフレームの高さの最小値と最大値を決定します。                                                                                     |
| オーバーフロー       | フラグメントコンテンツがそのフレームをオーバーフローすることを許可するかどうかを決定する。 デフォルトでは、フラグメントのコンテンツはそのフレームに制限されます。 オーバーフローが許可されている場合は、その表示方法を決定できます（スクロールなど）。 |

## コンテナオプション

| 項目                                                                        | 説明                                                                                      |
|:------------------------------------------------------------------------- |:--------------------------------------------------------------------------------------- |
| リンク                                                                       | コンテナがURL、ページ、マップされたURLのいずれにリンクするかを決定します。この設定により、以下のどのフィールドが利用できるかが決定されます。               |
| URL (**for URL**)                                                           | コンテナのローカライズ可能なURLを入力します。 他の言語用のURLを設定するには、まず編集バーでページやテンプレートの言語を切り替えます。 次に、目的のURLを入力します。 |
| ページ (**for Page**)                                                          | コンテナのLiferayページを選択します。                                                                  |
| 項目(**for Mapped URL**)                                                      | ドキュメントとメディア、Webコンテンツ、ブログ、またはカテゴリから項目を選択します。 選択した項目によって、マッピング用のフィールドが提供されます。             |
| フィールド(**for Mapped URL**)                                                   | コンテナにマップするURLフィールドを選択します。 有効なフィールドを選択すると、 **URL** フィールドに自動的に入力されます。                         |
| URL (**for Mapped URL**)                                                    | このフィールドには、選択した **フィールド** URLが自動的に入力されます。                                                    |
| 新しいタブで開く                                                                  | リンクを新しいタブで開くかどうかを決定します。                                                                 |
| コンテンツの表示                                                                  | コンテナのコンテンツをブロック、Flex Row、Flex Columnのいずれで表示するかを決定する。Flexオプションには、追加の設定がある。               |
| Flex Wrap (**for Flex Row or Flex Column**) {bdg-secondary}`7.4 U19+/GA19+` | フレックスの行と列の折り返しを設定します。折り返しなし、折り返し、折り返し反転。                                                |
| Align Items (**for Flex Column**) {bdg-secondary}`7.4 U19+/GA19+`           | 列の要素がどのように整列されるかを決定します。開始、中央、終了、伸張、ベースライン。                                              |
| Justify Content (**for Flex Row**) {bdg-secondary}`7.4 U19+/GA19+`          | 行内の要素の両端揃えを決定します。開始］、［中央］、［終了］、［間］、［回り込み］のいずれかを指定します。                                   |
| コンテナの幅                                                                    | コンテナで可変を使用するか固定幅を使用するかを決定します。                                                           |

## フォームフラグメントのオプション

{bdg-secondary}`利用可能な Liferay 7.4 U45+/GA45+`

フォームフラグメントには、独自の設定オプションがあります。 これらにアクセスするには、フラグメントをオブジェクトフィールドにマッピングする必要があります。

### フォームコンテナー

| 項目                                        | 説明                                                                                                    |
|:----------------------------------------- |:----------------------------------------------------------------------------------------------------- |
| コンテンツタイプ                                  | カスタムオブジェクトを選択します。 これは、フォームコンテナとそのフラグメントを選択されたオブジェクトにマッピングするものです。 フォームコンテナは、一度に1つのオブジェクトにしかマッピングできません。 |
| 成功メッセージ                                   | フォーム送信の成功メッセージを現在のページに埋め込むか、ユーザーをLiferayのページまたは外部URLにリダイレクトするかを決定します。                                 |
| 成功テキスト (**for Embedded**)                   | ローカライズ可能な成功メッセージを入力してください。                                                                            |
| Preview Success State (**for Embedded**)    | 埋め込まれた成功メッセージをプレビューします。                                                                               |
| コンテンツの表示                                  | コンテナのコンテンツをブロック、Flex Row、Flex Columnのいずれで表示するかを決定する。Flexオプションには、追加の設定がある。                             |
| Flex Wrap (**for Flex Row or Flex Column**) | フレックスの行と列の折り返しを設定します。折り返しなし、折り返し、折り返し反転。                                                              |
| アイテムの整列 (**for Flex Column**)               | 列の要素がどのように整列されるかを決定します。開始、中央、終了、伸張、ベースライン。                                                            |
| コンテンツの両端揃え (**for Flex Row**)               | 行内の要素の両端揃えを決定します。開始］、［中央］、［終了］、［間］、［回り込み］のいずれかを指定します。                                                 |
| コンテナの幅                                    | コンテナで可変を使用するか固定幅を使用するかを決定します。                                                                         |
