# コンテンツページへの要素の追加

[コンテンツページ](../adding-pages/adding-a-page-to-a-site.md)を作成したら、ウィジェットとフラグメントを追加して、ページのコンテンツと構造を定義することができます。

要素を追加する、

1. コンテンツページに移動し、 **Edit**( ![Edit Button](../../../images/icon-edit.png) ) をクリックします。 これにより、 [コンテンツページエディタ](./content-page-editor-ui-reference.md)が開きます。

1. **Fragments and Widgets**( ![Fragments and Widgets](../../../images/icon-plus.png) ) サイドパネルを開き、必要な要素をページにドラッグします。

   フラグメントやウィジェットのタブをブラウズしたり、検索フィールドを使って特定の要素を見つけることができます。

   ![編集サイドバーからフラグメントやウィジェットを追加できます。](adding-elements-to-content-pages/images/01.png)

1. (オプション) 要素を追加したら、 **ブラウザ** パネル ( ![Browser](../../../images/icon-hierarchy.png) ) を開いて、ページに追加したすべての要素を表示および管理します。

   Liferay 7.4 U37/GA37+では、ブラウザパネルで要素をドラッグ＆ドロップして再編成したり、要素をダブルクリックして名前を変更することができます。

   ![ブラウザーパネルでページに追加されたすべての要素を表示し、管理します。](adding-elements-to-content-pages/images/02.png)

1. 設定サイドバーで各ページ要素を選択し、設定します。 詳しくは、[フラグメントの設定](../page-fragments-and-widgets/using-fragments/configuring-fragments.md)を参照してください。

   ![設定サイドバーで各ページ要素を選択し、設定します。](adding-elements-to-content-pages/images/03.png)

1. ［**Publish**］ をクリックします。

   デフォルトでは、公開すると変更がすぐに公開されます。 ただし、 [ワークフロー](../../../process-automation/workflow/introduction-to-workflow.md) を使用してレビューおよび発行プロセスを指示している場合は、［発行］をクリックすると定義された手順が開始されます。

```{note}
ページを編集している間、Liferayは自動的に進捗を下書きとして保存します。
```
