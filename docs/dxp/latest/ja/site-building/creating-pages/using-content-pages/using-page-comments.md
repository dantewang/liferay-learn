 # ページコメントの使用

コンテンツページを作成する際、Liferayのページコメント機能を使ってチームとコラボレーションすることができます。 コメントはLiferay DXP 7.3+ですぐに有効になります。

![ページのコメントを使ってチームメイトと協力しよう。](./using-page-comments/images/01.png)

## ページコメントの追加

1. ページ要素（フラグメントやウィジェットなど）を含むコンテンツページに移動し、 **Edit** ボタン(![Edit Button](../../../images/icon-edit-pencil.png))をクリックして、ページの編集を開始します。

1. サイドバーメニューの **コメント** パネル (![Comments Icon](../../../images/icon-comments-w.png)) を開き、希望のフラグメントまたはウィジェットを選択します。

   または、必要なページ要素を選択し、 **コメント** アイコンをクリックすることもできます。 コメントパネルが開きます。

   ![ページ要素のコメントアイコンをクリックします。](./using-page-comments/images/02.png)

   ```{note}
   コメントを[フラグメントのサブ要素](../page-fragments-and-widgets/using-fragments/configuring-fragments/fragment-sub-elements-reference.md)、コンテナ、グリッドに追加することはできません。
   ```

1. サイドパネルにコメントを入力し、 ［**コメント**］ をクリックします。

コメントが追加された後、 **Actions** ボタン (![Actions Button](../../../images/icon-actions.png)) からコメントを編集または削除することができます。 ユーザーはそれに返信を追加することもできます。

![コメントを編集、削除、または返信します。](./using-page-comments/images/03.png)

```{note}
コメントを削除すると、コメントへのすべての返信が削除されます。
```

## ページコメントの確認

コメントがページ・フラグメントまたはウィジェットに追加されると、コメント・サイドパネルでそれらを確認することができます。

![コメント・サイドパネルでページのコメントを確認する。](./using-page-comments/images/04.png)

共同作業する場合、 ［**解決する**］ ボタン（![Resolve Button](../../../images/icon-resolve.png)）をクリックして、コメントを解決済みとしてマークできます。 デフォルトでは、解決されたコメントとその返信は非表示になります。 ただし、 ［**解決済みコメントの表示**］ をオンにすると、解決済みのコメントを表示できます。

![コメントを解決します。](./using-page-comments/images/05.png)

解決されたコメントを表示しているときに、緑色の ［**解決する**］ ボタンをクリックしてコメントを再度開くことができます。

![解決されたコメントを表示して再度開きます。](using-page-comments/images/06.png)
