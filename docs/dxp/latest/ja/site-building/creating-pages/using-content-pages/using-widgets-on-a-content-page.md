# コンテントページでのウィジェットの使用
<!--TASK: Reconsider-->
ウィジェットセクションは、ウィジェットページの [追加メニュー](../using-widget-pages/adding-widgets-to-a-page.md)と同じように機能します。 使用可能なウィジェットの完全なリストが表示され、それらをページに追加できます。 主な違いは、ウィジェットの主な設定オプションだけがコンテンツページで利用できることです。

```{note}
ウィジェットページでは、ウィジェットは [render-weight](https://resources.learn.liferay.com/reference/latest/en/dxp/definitions/liferay-portlet-app_7_4_0.dtd.html#render-weight) に従ってレンダリングされますが、コンテンツページではそうではありません。 代わりに、ページ上に表示される順番（つまり、左から右、上から下）でレンダリングされる。
```

## コンテントページにウィジェットを追加する

1. [コンテンツページに要素を追加している間](./adding-elements-to-content-pages.md)、 [ウィジェットパネル](./content-page-editor-ui-reference.md) を開き、利用可能なウィジェットをブラウズするか、名前でウィジェットを検索します。

1. ウィジェットを配置したいレイアウトの列と行にドラッグします。

ウィジェットはウィキの表示や、アセットパブリッシャーのようなダイナミックパブリッシングツールになります。 ウィジェットで表示するコンテンツは、長文のテキストや画像ギャラリーなど、さまざまなものがあります。

## コンテントページでのウィジェットの構成

1. ウィジェットにカーソルを合わせます。

1. **Actions**(![Actions menu icon](../../../images/icon-actions.png)) をクリックし、 **Configuration** を選択します。 ここからは、ウィジェットに応じていくつかのオプションがあります。

   ![コンテンツページのウィジェットは、オプションメニューから設定できます。](./using-widgets-on-a-content-page/images/01.png)

1. ［**保存**］ をクリックして変更を適用します。

## コンテントページでのウィジェット権限の構成

Liferay Portal 7.3 GA2から、コンテンツページからウィジェットのパーミッションを設定できるようになりました。 以下の手順に従って、コンテンツページのウィジェットパーミッションを設定してください：

1. ウィジェットにカーソルを合わせます。

1. **Actions**(![Actions menu icon](../../../images/icon-actions.png)) をクリックし、 **Permissions** を選択する。

1. 表示される新しいウィンドウで、ロールのアクセス許可をチェック／解除する。

1. ［**Save**］ をクリックして変更を適用します。

```{note}
 [ページテンプレート](../adding-pages/creating-a-page-template.md) に基づいてページを作成すると、権限もコピーされます。 マスターページのパーミッションは、ページではなく、[マスターページ](../defining-headers-and-footers/managing-master-page-templates.md)で設定されます。
```

## コンテンツページのウィジェットのエクスポート/インポート

1. ウィジェットにカーソルを合わせます。

1. **Actions**(![Actions menu icon](../../../images/icon-actions.png)) をクリックし、 **Export / Import** を選択します。

1. エクスポートまたはインポートのタブを選択し、それぞれのプロセスの設定を開始します。

## コンテンツページにウィジェット構成テンプレートを設定する

1. ウィジェットにカーソルを合わせます。

1. **Actions**(![Actions menu icon](../../../images/icon-actions.png)) をクリックし、 **Configuration Templates** を選択します。

1. **Save Current Configuration as Template** をクリックして、ウィジェットの現在の設定を保存します。

1. テンプレートの名前を入力し、 ［**保存**］ をクリックします。

## ウィジェットのコンテントページの制限

コンテンツページでウィジェットの基本的な設定オプションとパーミッションにアクセスできますが、ウィジェットページでのみ利用可能なオプションもあります：

****ネストされたアプリケーション** : ウィジェットの中のウィジェットは、ウィジェットページでのみサポートされています。

****ルック＆フィールコンフィギュレーション** : ウィジェットページで、ウィジェットのアクション(![Actions icon](../../../images/icon-actions.png))を通じて、ルック＆フィールコンフィギュレーションメニューにアクセスできます。 コンテンツのルック＆フィールはテーマで定義するか、フラグメントを使用するので、これはコンテンツページのウィジェットには使用できません。
