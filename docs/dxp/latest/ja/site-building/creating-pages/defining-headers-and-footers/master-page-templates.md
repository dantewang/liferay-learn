# マスターページテンプレート

{bdg-secondary}`Liferay DXP/Portal 7.3以降で利用可能`

マスターページテンプレートは、フラグメントを使用してすべてのページに共通する要素を定義する方法を提供します（例：以前テーマによって提供されたヘッダーとフッター）。 サイト全体で同じマスターページテンプレートを使うことも、ページごとに独自のテンプレートを用意することもできます。 各ページとページテンプレートには、関連するマスターページテンプレートが必要です。

デフォルトでは、すぐに使えるBlankマスターページテンプレートは共通要素を提供していません。

![空白のマスター・ページ・テンプレートは、ページやページ・テンプレートのデフォルトです。](./master-page-templates/images/01.png)

## マスターページテンプレートの作成

マスターページテンプレートの作成は、コンテンツページの作成プロセスと似ている。 マスターページテンプレートは、コンテンツページのように、任意の数のページフラグメントを含むことができます。 ただし、マスターページテンプレートは、ドロップゾーンと呼ばれる、ページの断片を配置できる場所を指定する重要な領域が1つ定義されている点が異なります。 このエリアは削除できませんが、ページレイアウトのどこかに配置することは可能です。 また、ドロップゾーンに追加できるページフラグメントを定義することもできます。 マスターページテンプレートの典型的な使用例は、ヘッダー、フッター、ドロップゾーンを持つことです。 詳しくは、 [マスターページテンプレートの作成](./creating-a-master-page-template.md) を参照してください。

![マスターページテンプレートの典型的な使用例には、ヘッダー、ドロップゾーン、フッターがある。](./master-page-templates/images/02.png)
