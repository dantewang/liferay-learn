# ページフラグメントとウィジェット

<!--TASK: Develop into-->
Liferayは、サイトページとページテンプレートを構築するためのページフラグメントとウィジェットを提供します。

```{note}
フラグメントはウィジェット・ページでは使用できません。 コンテンツページ、マスターページテンプレート、ページテンプレート、表示ページテンプレートでのみ利用可能です。
```
