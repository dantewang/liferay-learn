# Liferay ソフトウェアのバージョン

Liferay 7.4から、Liferayはアップデートと修正を提供するためにローリングリリースモデルを採用しました。 つまり、Liferay 7.4は当分の間、多くのリリースがある長寿リリースということです。 さらに、Liferay DXP 7.4 U92から、Liferay DXPは四半期ごとのリリーススケジュールに移行しました。 バージョンを示すために、Liferay PortalとLiferay DXPは以下のスキームを使用します：

| ソフトウェア         | バージョン         |
|:-------------- |:------------- |
| Liferay Portal | 7.4.3.XX GAXX |
| Liferay DXP    | 年・四半期         |

## Liferay ポータルのバージョン管理

以下は、Liferay Portalの仮想的なバージョン例です：

```
Liferay Portal 7.4.3.30 GA30
Liferay Portal 7.4.3.31 GA31
...
Liferay Portal 7.4.3.100 GA100
```

## Liferay DXP バージョン管理

以下はLiferay DXPバージョンの仮想的な例です：

```
Liferay DXP 2023.Q3
Liferay DXP 2023.Q4
...
Liferay DXP 2025.Q1
```

```{note}
Liferay DXP 7.4 U92以前のバージョンは、Liferay DXP 7.4.13 Update 92、Liferay DXP 7.4.13 Update 91などの形式をとっています。
```

## 旧バージョン

古いLiferayのリリースは、3桁の数字からなるメジャーリリース、マイナーリリース、メンテナンスリリースという特定の構造に従っています。 例えば、6.1.2. これらの数字はリリースの種類を表しています：メジャー、マイナー、メンテナンス。

1桁目の数字（例：6.x.xから7.x.x）または2桁目の数字（例：7.2.xから7.3.x）の変更は、メジャーリリースまたはマイナーリリースを表します。 リリースには、機能の変更、アーキテクチャの変更、APIの変更、新機能の追加などが含まれます。

3桁目の数字の変更（例：6.2.1から6.2.2）は、メンテナンスリリースを表します。 リリースには、セキュリティの修正や信頼性の向上などが含まれるかもしれない。

旧バージョン用のパッチや修正パックも用意されている。 詳しくは [パッチタイプについて](../maintaining-a-liferay-installation/patching-dxp-7-3-and-earlier/understanding-patch-types-for-dxp-7-3-and-earlier.md) をご覧ください。
