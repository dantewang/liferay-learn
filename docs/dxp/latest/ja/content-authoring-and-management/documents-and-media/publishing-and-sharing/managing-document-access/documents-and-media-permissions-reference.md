# ドキュメントとメディアの権限リファレンス

{bdg-secondary}`Liferay 7.4 U76+/GA76+`.

[アプリケーション](#general-permissions) および [リソース](#resource-permissions) の権限を使用して、ドキュメントおよびメディア資産へのアクセスを管理します。 ロールベースのアクセスコントロールにより、意図したユーザのみがドキュメントの作成、閲覧、ダウンロード、管理を行えるようにすることができます。

これらのパーミッションは、パーミッションのサイドパネルにある「コンテンツとデータ」の「ロール」アプリケーションで付与することができます。

![Grant Documents and Media permissions in the Roles application under Content & Data in the permissions side panel.](./documents-and-media-permissions-reference/images/01.png)

または、Documents and Mediaアプリケーションで個々のリソース権限を割り当てることもできます。

![Assign individual resource permissions in the Documents and Media application.](./documents-and-media-permissions-reference/images/02.png)

詳細は、 [ロール権限の定義](../../../../users-and-permissions/roles-and-permissions/defining-role-permissions.md) を参照してください。

## 通常権限

Generalパーミッションは、Documents and Mediaアプリケーション自体へのアクセスを許可し、 [リソース関連のパーミッションは許可しません。](#resource-permissions) 。

| 権限                   | 説明                                                                                                       |
| :------------------- | :------------------------------------------------------------------------------------------------------- |
| サイトとアセットライブラリ管理にアクセス | サイトメニュー( ![サイトメニュー](../../../../images/icon-product-menu.png))またはアセットライブラリからドキュメントとメディアアプリケーションにアクセスします。 |
| 設定                   | ドキュメントとメディアの設定オプションを表示および変更します。                                                                          |
| 権限の設定                | ドキュメントとメディアのアプリケーション権限を表示および変更します。                                                                       |
| 詳細設定                 | 該当なし                                                                                                     |
| 表示                   | 書類とメディアの申し込みページを見る                                                                                       |

## リソース権限

リソース権限は、Documents and Mediaアプリケーションのリソースの表示と操作へのアクセスを許可します。 これらの権限の中には、 [エンティティを作成するためのもの](#documents) もあれば、既存のエンティティに作用するためのものもある（たとえば、 [フォルダ](#documents-folder) 、 [文書](#document) ）。

### ドキュメントとメディア

| 権限              | 説明                                       |
| :-------------- | :--------------------------------------- |
| ドキュメントを追加する     | ドキュメントとメディア」にドキュメントを追加する。                |
| ドキュメントタイプを追加する  | ドキュメントとメディアでドキュメントタイプを追加します。             |
| フォルダを追加する       | ドキュメントとメディアにフォルダを追加する。                   |
| リポジトリを追加する      | ドキュメントとメディアにリポジトリを追加する。                  |
| ショートカットを追加する    | ドキュメントとメディアにファイルや外部ビデオへのショートカット（リンク）を追加。 |
| メタデータセットの追加     | ドキュメントとメディアにメタデータセットを追加する。               |
| 権限の設定           | リソース（フォルダやファイルなど）の作成と管理に関する権限の表示と変更。     |
| 変更通知を受け取る（購読する） | リソース・イベントのお知らせをEメールで受け取る。                |
| 更新              | ドキュメントとメディアでリソースを編集する。                   |
| 表示              | ドキュメントとメディアでリソースを見る                      |

### ドキュメントとメディアフォルダー

| 権限              | 説明                                             |
| :-------------- | :--------------------------------------------- |
| アクセス            | フォルダの内容にアクセスする。                                |
| ドキュメントを追加する     | フォルダにドキュメントを追加する。                              |
| ショートカットを追加する    | ロールに閲覧権限があるフォルダ内のファイルへのショートカット（リンク）を追加します。     |
| サブフォルダを追加する     | フォルダにサブフォルダを追加                                 |
| 削除              | フォルダをごみ箱に移動する。                                 |
| 権限の設定           | フォルダパーミッションの表示と変更                              |
| 変更通知を受け取る（購読する） | フォルダにファイルが追加されたり、変更されたときにEメールで通知を受け取るように登録します。 |
| 更新              | フォルダの編集や移動                                     |
| 表示              | フォルダを見る                                        |

### ドキュメント

| 権限            | 説明                                                                                                                            |
| :------------ | :---------------------------------------------------------------------------------------------------------------------------- |
| コメントを追加する     | 文書にコメントを追加する。                                                                                                                 |
| 削除            | 文書をごみ箱に移動する。                                                                                                                  |
| コメントを削除する     | ドキュメントからコメントを削除する。                                                                                                            |
| ダウンロードする      | 資料をダウンロードする                                                                                                                   |
| チェックアウトを無効にする | 他のユーザーによってチェックアウトされた場合でも、ドキュメントに変更を加えることができます。                                                                                |
| 権限の設定         | ドキュメントの権限の表示と変更。                                                                                                              |
| 更新            | ドキュメントの編集や移動                                                                                                                  |
| コメントを更新する     | 文書コメントを編集する。                                                                                                                  |
| 表示            | 資料を見る この権限を持つユーザーは、すべてのドキュメント・メタデータ（タイトル、説明、アダプティブ・メディア・イメージ、プレビュー、サムネイルなど）、および添付のドキュメント・タイプやメタデータ・セットで定義されているものを閲覧することができます。 |

```{important}
Liferay 7.4 U75/GA75以前のバージョンでは、ビューパーミッションはアセットをダウンロードする権限をユーザーに与えます。 7.4 U76+/GA76+では、Documents and Mediaに個別のダウンロード権限が含まれているため、表示とダウンロードのアクセスを別々に管理できます。 アップグレードすると、Liferayはビュー権限を持つすべてのロールにダウンロード権限を自動的に割り当てます。 デフォルトでは、認証されていないユーザーを含むすべてのユーザーがファイルをダウンロードできます。 ダウンロードへのアクセスを制限したい場合は、ファイルをダウンロードできないロールからこの権限を削除する必要があります。
```
### ドキュメントとメディアのショートカット

| 権限    | 説明                    |
| :---- | :-------------------- |
| 削除    | ショートカットをごみ箱に移動する。     |
| 権限の設定 | ショートカットのパーミッションの表示と変更 |
| 更新    | ショートカットの編集や移動。        |
| 表示    | ショートカットを見る            |

### ドキュメントタイプ

| 権限    | 説明                           |
| :---- | :--------------------------- |
| 削除    | ドキュメントタイプを永久に削除します。          |
| 権限の設定 | ドキュメントタイプのアクセス許可を表示および変更します。 |
| 更新    | 文書タイプを編集する。                  |
| 表示    | ドキュメントの種類を見る                 |

### ドキュメント・メタデータ

| 権限    | 説明                       |
| :---- | :----------------------- |
| 削除    | メタデータ・セットを永久に削除する。       |
| 権限の設定 | メタデータ・セットのパーミッションの表示と変更。 |
| 更新    | メタデータ・セットを編集する。          |
| 表示    | メタデータ・セットを見る。            |
