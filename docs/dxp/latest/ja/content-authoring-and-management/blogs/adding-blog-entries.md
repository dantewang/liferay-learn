# ブログエントリの追加

ブログのエントリーはサイトを対象としている。 デフォルトでは、認証されたユーザーのみがブログエントリを作成できます。 詳細は、[Blogs Permissions Reference](./blog-permissions-reference.md)を参照してください。

[ブログエントリ](#creating-an-entry) を作成し、 [ページ](./displaying-blogs.md)に表示することができます。

```{tip}
ブログ [widget](../../site-building/creating-pages/using-widget-pages/adding-widgets-to-a-page.md) を使用することは、ページにブログ・エントリーを表示する最も簡単な方法の一つです。
```

## エントリの作成

1. **サイトメニュー**( ![Product Menu](../../images/icon-product-menu.png) ) を開き、 **コンテンツ & データ** &rarr; **ブログ** にアクセスする。

1. Entriesタブで、 **Add**( ![Add icon](../../images/icon-add.png) ) をクリックして、新しいブログ・エントリーを追加する。

   ![ブログ・エントリー・フォーム・エディター](./adding-blog-entries/images/01.png)

1. エントリーのカバー画像としてアップロードするファイルをドラッグ＆ドロップするか、選択します。 デフォルトでは、このカバー画像はブログエントリの要約の一部として公開されます。

1. ブログエントリの［title］を入力します。

1. エントリにオプションで［subtitle］を入力します。

1. ［**Content**］ フィールドにエントリのコンテンツを入力します。 さまざまなテキスト編集機能の詳細については、 [ブログエントリエディターのリファレンス](./blog-entry-editor-reference.md) を参照してください。

### ブログエントリーの分類

<!-- ```{note} Available in Liferay DXP 7.3+. This section must be updated to reflect the new by-default categories and vocabularies in 7.3``` -->

ブログのエントリーを分類することで、ユーザーがサイトのコンテンツを見つけやすくなります。

カテゴリー（公開および内部）、タグ、またはエントリーをアセットに関連付けるなどして、ブログエントリーを分類しましょう。 カテゴリと語彙について詳しくは、 [コンテンツのカテゴリとボキャブラリの定義](../tags-and-categories/defining-categories-and-vocabularies-for-content.md) を、タグの使用について詳しくは、 [Tagging Content and Mananing Tags](../tags-and-categories/tagging-content-and-managing-tags.md) を参照してください。

カテゴリーとタグを使ってエントリーを分類する、

1. カテゴライズのセクションを展開する。

1. **Select** をクリックして、既存のカテゴリーまたはタグを選択します。

   または、タグフィールドにタグ名を入力し、リターンキーを押して新しいタグを作成することもできます。

資産をエントリーに関連付けること、

1. 関連資産」セクションを展開し、 「**選択**」 をクリックして資産を選択し、エントリーにリンクします。

### エントリの表示方法の設定

ブログエントリーとその表示方法を設定するには、 **設定** と **表示ページ** セクションを展開し、以下の設定を調整します、

| 設定            | Description                                                                                                                                    |
|:------------- |:---------------------------------------------------------------------------------------------------------------------------------------------- |
| URL           | エントリーのURLを設定します： **自動** 、エントリーのタイトルに基づいてURLが設定されます。 **カスタムフレンドリーURLのカスタム** 。                                                                        |
| 概要            | エントリーのアブストラクトを指定する。 デフォルトでは、Blogsアプリケーションは最初の400語を使用します。 **Custom Abstract** 手動での説明を求めるプロンプトが表示されます。                                             |
| 画像（小）         | エントリーのサムネイルとして表示する小さな画像を追加します。                                                                                                                 |
| 公開開始日時        | エントリーに使用する表示日付を設定する。                                                                                                                           |
| メール項目の更新通知を送る | エントリーの更新に関するコメントを送信する。                                                                                                                         |
| pingbackを許可   | あなたの投稿に他のウェブサイトのコンテンツへのリンクが含まれており、そのウェブサイトがピンバックをサポートしている場合、そのウェブサイトはあなたのサイトからの通知または「ping」を受け取ります。 ピンバックは、あなたがリンクしたURLへのXML-RPCリクエストを使用します。    |
| 表示ページテンプレート   | ブログ・エントリーを表示するために、デフォルトの [表示ページ・テンプレート](../../site-building/displaying-content/using-display-page-templates.md)、特定のもの、または表示ページ・テンプレートなしを選択します。 |

![ブログエントリーを作成する際、設定パネルを使用して、ブログエントリーをいつ、どこに表示するか、エントリーの抄録に何を使用するかを制御します。](./adding-blog-entries/images/02.png)

### ブログエントリーの公開

ブログエントリーの編集、カテゴリー分け、設定が終わったら、 **Publish** をクリックして公開します。

仲間のサイトメンバーは、コメントを追加したり、エントリを評価したり、他のソーシャルメディアサイトで共有したりできます。

![ブログエントリが公開されました。](./adding-blog-entries/images/03.png)
