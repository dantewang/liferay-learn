# ブループリントの検索

{bdg-link-primary}`[LES Subscription](../activating-liferay-enterprise-search.md)`
{bdg-secondary}`DXP 7.4 U5+; 7.3 U6+`

検索ブループリントは、Search Experiencesアプリケーションの最初の機能です。 これを使えば、次のことができる。

- Liferayのデフォルトの検索ページの動作を微調整します
- 検索ページの動作を完全にオーバーライドします
- 検索ユーザーの属性に応じたコンテキストアウェアな検索結果の構成
