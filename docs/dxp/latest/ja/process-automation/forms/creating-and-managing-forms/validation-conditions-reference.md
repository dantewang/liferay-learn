## 数値フィールド検証条件

| 条件        | 説明                             |
|:--------- |:------------------------------ |
| 以上もしくは等しい | この条件は、値が指定された値以上かどうかを確認します。    |
| 以上        | この条件は、値が指定された値より大きいかどうかを確認します。 |
| 等しくない     | この条件は、値が指定された値と等しいかどうかを確認します。  |
| 以下もしくは等しい | この条件は、値が指定された値以下かどうかを確認します。    |
| 以下        | この条件は、値が指定された値より小さいかどうかを確認します。 |
