# アプリケーションの構築

自分が知っているツールとフレームワークを使用してLiferay DXP用のすばらしいアプリケーションを構築する方法はたくさんあります。 Liferay DXP 7.4には、コードを記述したりモジュールをデプロイしたりせずにビジネスアプリケーションを作成するために使用できる [オブジェクト](building-applications/objects.md) が導入されています。
