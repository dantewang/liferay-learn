# 검색 페이지 템플릿 사용

기본 검색 페이지는 전역 범위 페이지 템플릿으로 뒷받침됩니다. 수동으로 생성된 검색 페이지는 동일한 템플릿을 기준으로 사용할 수 있습니다. 즉시 사용 가능한 검색 페이지 템플릿에는 검색 표시줄, 검색 결과, 검색 제안, 검색 옵션 및 여러 검색 패싯 위젯이 포함되어 있습니다.

![검색 페이지 템플릿을 사용하면 시작하고 실행할 수 있습니다.](./using-a-search-page-template/images/01.png)

템플릿은 두 가지 방법으로 사용할 수 있습니다. [템플릿을 사용하여 생성된 검색 페이지를 관리](#using-a-page-template-to-manage-search-pages) 하거나 [템플릿과 독립적으로 구성된 기본 검색 페이지를 생성](#using-a-page-template-to-create-a-baseline-search-page) 합니다.

## 페이지 템플릿을 사용하여 검색 페이지 관리

페이지 템플릿을 사용하여 만든 검색 페이지 관리는 검색 페이지를 구성할 때 *변경 사항 상속* 기능을 활성화하여 수행할 수 있습니다.

템플릿을 사용하여 검색 페이지를 만들고 템플릿에 대한 변경 사항을 상속하도록 하려면 다음을 수행하십시오.

1. [사이트 메뉴](../../../getting-started/navigating-dxp.md#site-menu)을 사용하여 원하는 사이트로 이동합니다.

1. 사이트 메뉴에서 *사이트 빌더* &rarr; *페이지* 로 이동합니다.

1. 페이지 페이지에서 *추가* 버튼(![Add](../../../images/icon-add.png))을 클릭하여 최상위 페이지를 만듭니다.

1. 검색 페이지 템플릿을 선택합니다. 기본 검색 페이지 템플릿을 사용하려면 *전역 템플릿* &rarr; *검색*을 클릭하십시오. 새 검색 페이지의 이름을 입력하고 *저장*을 클릭합니다.

1. 변경 사항 상속을 활성화하려면 페이지 구성 내에서 _Inherit Changes_ (변경 사항 상속 1) 옵션을 전환합니다.

    ![이 기능을 활성화하려면 변경 사항 상속을 클릭하십시오.](./using-a-search-page-template/images/02.png)

    ```{tip}
    새 페이지의 경우 페이지를 처음 만들 때 이를 활성화할 수 있습니다. 검색 템플릿을 기반으로 하는 기존 페이지의 경우 사이트 메뉴 → *사이트 작성기* → *페이지*로 이동한 다음 작업 메뉴에서 *구성*을 클릭합니다.
    ```

    ```{warning}
    페이지가 페이지 템플릿에서 변경 사항을 상속하도록 설정된 경우 페이지 구성 내의 일부 설정을 사용할 수 없게 됩니다.
    ```

이제 페이지 템플릿이 업데이트될 때마다 검색 페이지가 업데이트됩니다.

## 페이지 템플릿을 사용하여 기본 검색 페이지 만들기

검색 페이지를 구축하기 위한 시작점으로 페이지 템플릿을 사용한 다음 기본 페이지 템플릿에 적용될 수 있는 변경 사항과 별도로 페이지를 조정할 수도 있습니다.

페이지 템플릿에서 추가 변경 사항을 상속하지 않는 템플릿을 사용하여 검색 페이지를 만들려면 다음을 수행합니다.

1. [사이트 메뉴](../../../getting-started/navigating-dxp.md#site-menu)을 사용하여 원하는 사이트로 이동합니다.

1. 사이트 메뉴에서 *사이트 빌더* &rarr; *페이지* 로 이동합니다.

1. 페이지 페이지에서 *추가* 버튼(![Add](../../../images/icon-add.png))을 클릭하여 최상위 페이지를 만듭니다.

1. 검색 페이지 템플릿을 클릭하고 기본 검색 페이지 템플릿을 사용하려면 *전역 템플릿* &rarr; *검색*을 클릭합니다. 새 검색 페이지의 이름을 입력하고 *저장*을 클릭합니다.

1. 페이지 구성에서 _Inherit Changes_ 옵션을 선택하지 않은 상태로 둡니다.

이제 템플릿을 기반으로 페이지가 생성되며 페이지 템플릿에 대한 변경 사항을 수신하지 않고도 추가로 조정하고 구성할 수 있습니다.

## 기본 검색 페이지 템플릿 사용

기본적으로 모든 위젯은 _Barebone_ Application Decorator를 사용합니다. 위젯에 렌더링할 콘텐츠가 없으면 위젯 본문이 숨겨집니다. 마우스를 위젯 위로 가져가면 헤더 및 위젯 옵션 아이콘이 표시됩니다. 이 때문에 검색 페이지를 방문할 때 완전히 렌더링된 특정 위젯을 볼 수 없습니다.

![베어본 모양과 느낌의 검색 페이지입니다.](./using-a-search-page-template/images/03.png)

반대로 검색 위젯을 페이지에 수동으로 추가하면 표시할 콘텐츠가 없는 경우에도 더 많은 위젯을 표시하는 _Borderless_ 데코레이터(기본값)를 사용합니다.

![테두리 없는 모양과 느낌의 검색 페이지입니다.](./using-a-search-page-template/images/04.png)

테두리 없는 데코레이터로 페이지의 모양과 느낌을 구성하면 이제 범주 패싯이 표시됩니다.
