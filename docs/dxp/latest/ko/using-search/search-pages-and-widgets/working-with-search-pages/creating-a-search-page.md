# 검색 페이지 만들기

기존 검색 페이지 템플릿을 기반으로 하지 않도록 선택한 경우 사이트에 대한 검색 페이지를 만들고 구성할 수 있습니다. 이 접근 방식은 사이트 검색 페이지의 공통 요소를 다른 사이트의 검색 페이지와 공유하지 않으려는 경우에 적합합니다.

이 단계에서는 기본 사이트에서 수동으로 구성된 검색 환경으로 전환하는 방법을 보여줍니다.

1. 기존 검색 페이지가 있는 사이트를 사용하는 경우 먼저 기본 검색 페이지를 삭제하십시오. 기본 사이트 메뉴로 이동하여 *Site Builder* &rarr; *Pages*를 클릭합니다. 검색 페이지의 작업 메뉴 아이콘(![Actions](../../../images/icon-actions.png))을 클릭하고 *삭제*를 선택합니다. 페이지 삭제를 확인합니다.

    삭제되면 검색 표시줄이 사이트 관리자에게만 표시되는 경고 메시지로 바뀝니다.

    ![검색 창 위젯에 대해 검색 페이지를 구성해야 합니다.](./creating-a-search-page/images/01.png)

1. 새 페이지를 만듭니다. 숨기거나 탐색에 추가하십시오(기본 검색 페이지는 탐색에서 숨겨짐).

    미리 구성된 검색 페이지를 시작점으로 사용하려면 검색 페이지 템플릿을 사용하세요. 사이트 페이지 추가 양식에서 템플릿을 찾습니다. *Global Templates*아래에 있습니다.

    ![검색 페이지 템플릿을 사용하여 바로 생성을 시작하세요.](./creating-a-search-page/images/02.png)

1. 템플릿이 지원하지 않는 페이지를 만드는 경우 필요한 모든 위젯을 추가하고 구성합니다. 위젯 추가 메뉴의 검색 섹션에서 사용 가능한 모든 검색 위젯을 찾을 수 있습니다. 페이지에 원하는 대로 배치합니다.

1. 페이지 상단에 검색 표시줄을 구성하여 새 검색 페이지의 친숙한 URL을 가리키도록 합니다.

    검색창 위젯의 옵션 메뉴(![Options](../../../images/icon-widget-options.png))를 클릭합니다.

    *구성* 을 클릭하고 대상 페이지를 검색 페이지의 친숙한 URL로 설정합니다.

    *저장*을 클릭합니다.

검색 페이지가 실행 중입니다.
