# 검색 제안 활성화

초기 쿼리가 개선될 수 있을 때 사용자에게 검색어를 제안할 수 있습니다. 맞춤법 검사 설정을 통해 관리자는 사용자가 많은 결과를 반환하지 않는 검색어를 입력하는 경우(예: 철자가 약간 틀린 단어) 검색을 개선하라는 메시지가 표시되도록 관리자가 검색 애플리케이션을 구성할 수 있습니다.

맞춤법 검사 설정을 구성하려면:

1. 먼저 맞춤법 검사 색인을 재색인해야 합니다. **Control Panel** &rarr; **Configuration** &rarr; **Search** 로 이동하여 Index Actions 화면을 엽니다.

    **모든 맞춤법 검사 색인 다시 색인화** 옆에 있는 **실행** 을 클릭하십시오.

1. 하나의 **제안** 위젯이 검색 페이지에 있는지 확인하십시오( [검색 페이지 템플릿](../working-with-search-pages/using-a-search-page-template.md)을 사용하는 경우 기본적으로 추가됨).

1. **제안** 위젯 구성 화면을 엽니다. 위젯 옵션 버튼(![Options](../../../images/icon-widget-options.png))을 클릭하고 **구성** 를 선택합니다.

    ![사용자 입력 실수를 허용하고 사용자를 결과로 이끄는 데 도움이 되도록 제안 설정을 구성합니다.](./enabling-search-suggestions/images/01.png)

1. 원하는 대로 구성하고 **저장** 을 클릭합니다.

## 제안 구성 참조

제안 위젯에 사용할 수 있는 세 가지 기본 설정이 있습니다.

**검색 결과 수가 임계값을 충족하지 않는 경우 "...을(를) 사용하셨습니까?"를 표시합니다.** 검색 결과가 최소 수(기본값 50)를 반환하지 않는 경우 철자가 검사된 검색 쿼리를 사용자에게 제공합니다.

**관련 쿼리 표시:** 검색 결과 수가 지정된 임계값(기본값 50)을 충족하지 않는 경우 최대 대체 쿼리 수(기본값 10)를 표시합니다.

**성공적인 쿼리를 기반으로 새 관련 쿼리 추가:** 사용자에게 제안으로 표시될 수 있도록 최소 수의 결과(기본적으로 50개)를 생성하는 경우 사용자의 검색 쿼리를 인덱싱합니다. 관련 검색어 표시 설정이 활성화된 경우 충분한 결과를 생성하지 않는 유사한 검색어에 대한 관련 검색어로 사용됩니다.
