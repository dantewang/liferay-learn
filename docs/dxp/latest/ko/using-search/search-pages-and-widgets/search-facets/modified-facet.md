# 수정 페이셋

수정된 패싯은 특정 기간 동안 생성되거나 수정된 결과 및 검색된 키워드와 일치하는 결과로 검색 결과를 좁힙니다.

![수정된 패싯 결과의 예.](modified-facet/images/01.png)

미리 구성된 기간을 선택하는 것 외에도 사용자는 날짜 선택기를 사용하여 시작 및 종료 날짜를 지정하여 사용자 지정 범위를 선택할 수 있습니다.

## 수정된 패싯 구성

수정된 패싯을 구성하려면 패싯의 옵션 메뉴(![Click on the options icon of the search bar.](../../../images/icon-app-options.png))를 열고 **구성** 클릭하십시오.

![구성 옵션을 클릭합니다.](modified-facet/images/02.png)

**디스플레이 설정:** **기본값** , **컴팩트 레이아웃** , **라벨 레이아웃** 중에서 선택합니다. 기본 레이아웃은 각 용어 옆에 체크박스를 표시하지만 압축 레이아웃은 그렇지 않습니다. 레이블 레이아웃은 각 용어에 대해 클릭할 수 있는 작은 레이블을 표시합니다.

고급 구성 섹션에는 다음과 같은 추가 옵션이 있습니다.

**빈도 임계값:** 패싯 용어 목록에 나타나는 용어에 필요한 최소 빈도를 설정합니다. 예를 들어 패싯의 빈도 임계값이 `3`로 설정된 경우 일치하는 결과가 2개인 용어는 패싯의 용어 결과 목록에서 생략됩니다.

**용어 주문 기준:** Liferay 7.4 U63+/GA63+의 경우 패싯 용어 정렬 전략(용어 빈도 내림차순 또는 용어 빈도 오름차순)을 선택할 수 있습니다. 용어 빈도 내림차순은 가장 일치하는 용어부터 가장 적게 일치하는 용어까지 패싯 용어를 정렬합니다. 기본적으로 패싯 용어는 타이머 기간 구성 순서대로 정적으로 나타납니다.

범위 구성에서 시간 범위를 수정, 삭제 또는 생성할 수 있습니다.

사용 가능한 시간 범위 별칭에는 다음이 포함됩니다.

    past-hour
    past-24-hours
    past-week
    past-month
    past-year

기본적으로 모든 시간 범위는 `*`로 끝나며 이제 **로 평가됩니다** . 예를 들어 지난 주 범위는 다음과 같습니다.

    [past-week TO *]

그러나 지금까지 범위를 종료하는 데 제한되지 않습니다. `*`대신 다른 시간 범위 별칭을 끝점으로 지정할 수 있습니다.

예를 들어 12개월 전부터 1개월 전까지 범위를 설정하려면

1. 기존 범위 중 하나에서 더하기 버튼을 클릭합니다.

1. **Past 1-12 Months** 레이블을 지정합니다.

1. 범위 값을 지정하십시오. 
   
        [past-year to past-month]

이렇게 하면 수정된 패싯 용어로 대체 시간 범위를 사용할 때 많은 유연성이 제공됩니다.

완료되면 **저장** 버튼을 클릭합니다.
