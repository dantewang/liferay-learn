# 사이트 페이셋

사이트 패싯은 검색 결과를 특정 사이트에 있는 항목으로 좁힙니다. 검색된 키워드와 일치하는 콘텐츠가 있는 각 사이트는 패싯 용어로 나타납니다.

![사이트 패싯 결과의 예.](site-facet/images/01.png)

```{important}
사이트 패싯이 여러 사이트를 표시하려면 *Everything*을 검색하도록 검색 표시줄을 구성해야 합니다. Everything을 검색하지 않으면 현재 사이트만 검색되고 사이트 패싯에는 아무것도 표시되지 않습니다. 이 경우 사이트 패싯이 페이지에서 숨겨집니다.

Everything을 검색하도록 전역적으로 포함된 페이지 상단 검색 표시줄을 구성하면 모든 페이지에 포함된 검색 표시줄만 구성되는 것이 아닙니다. 또한 페이지 상단 검색 표시줄의 구성이 검색 페이지의 검색 표시줄 구성을 재정의하기 때문에 검색 페이지의 검색 표시줄이 모든 항목을 검색하도록 합니다. 사이트의 다른 검색창 위젯에도 동일하게 적용되지 않습니다. 이들 각각은 원하는 대로 구성해야 합니다.
```

전역 검색 표시줄이 비활성화된 경우 모든 것을 검색하도록 검색 페이지의 검색 표시줄 위젯을 구성합니다.

검색 범위를 구성하려면

1. 검색 표시줄의 옵션 아이콘(![Options icon](../../../images/icon-app-options.png))을 클릭하고 **구성** 클릭합니다.

   ![구성 옵션을 클릭합니다.](site-facet/images/02.png)

1. Scope 옵션을 **Everything** 로 설정합니다.

   ![드롭다운 메뉴에서 모든 항목을 선택합니다.](site-facet/images/03.png)

   완료되면 **저장** 버튼을 클릭합니다.

## 사이트 패싯 구성

사이트 패싯을 구성하려면 패싯의 옵션 메뉴(![Click on the options icon of the search bar.](../../../images/icon-app-options.png))를 열고 **구성** 클릭하십시오.

![구성 옵션을 클릭합니다.](site-facet/images/04.png)

**디스플레이 설정:** **기본값** , **컴팩트 레이아웃** , **라벨 레이아웃** 중에서 선택합니다. 기본 레이아웃은 각 용어 옆에 체크박스를 표시하지만 압축 레이아웃은 그렇지 않습니다. 레이블 레이아웃은 각 용어에 대해 클릭할 수 있는 작은 레이블을 표시합니다.

고급 구성 섹션에는 다음과 같은 추가 옵션이 있습니다.

**사이트 매개변수 이름:** 패싯에 대한 URL에서 매개변수 이름을 변경합니다. 기본값은 **사이트** 입니다.

**최대 용어:** 패싯에 대해 발견된 일치 용어 수에 관계없이 표시할 패싯 용어의 최대 수를 설정합니다.

**빈도 임계값:** 패싯 용어 목록에 나타나는 용어에 필요한 최소 빈도를 설정합니다. 예를 들어 패싯의 빈도 임계값이 3으로 설정된 경우 두 개의 일치하는 결과가 있는 용어는 용어 결과 목록에 나타나지 않습니다.

**빈도 표시:** 용어 빈도를 표시할지 여부를 선택합니다.

**용어 주문 기준:** Liferay 7.4 U54+/GA54+의 경우 용어 빈도 내림차순(기본값), 용어 빈도 오름차순, 용어 값 오름차순 또는 용어 값 내림차순과 같은 패싯 용어 정렬 전략을 선택할 수 있습니다. 기본 정렬 순서는 패싯 용어가 가장 많이 일치하는 것부터 가장 적게 일치하는 것까지입니다. 용어 값 옵션은 패싯 용어를 사전순으로 정렬합니다.

완료되면 **저장** 버튼을 클릭합니다.
