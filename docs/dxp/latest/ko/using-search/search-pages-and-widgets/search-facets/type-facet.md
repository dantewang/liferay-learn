# 유형 페이셋

유형 패싯은 검색 결과를 특정 자산 유형과 관련된 항목으로 좁힙니다. 검색된 키워드와 일치하는 콘텐츠가 있는 각 유형은 패싯 용어로 나타납니다.

![유형 패싯 결과의 예.](type-facet/images/01.png)

기본적으로 사용 가능한 모든 자산 유형은 패싯 용어로 표시됩니다.

* Blogs입장
* 캘린더 이벤트
* Documents and Media Document
* 문서 도서관 폴더
* 동적 데이터 목록 레코드
* 양식 레코드
* 기술 자료 문서
* 전보국 메시지
* 페이지
* 사용자
* 웹 내용
* 웹 내용 급식
* Wiki페이지

## 유형 패싯 구성

유형 패싯을 구성하려면 패싯의 옵션 메뉴(![options icon](../../../images/icon-app-options.png))를 열고 **구성** 클릭하십시오.

![구성 옵션을 클릭합니다.](type-facet/images/02.png)

**디스플레이 설정:****기본값** , **컴팩트 레이아웃** , **라벨 레이아웃** 중에서 선택합니다. 기본 레이아웃은 각 용어 옆에 체크박스를 표시하지만 압축 레이아웃은 그렇지 않습니다. 레이블 레이아웃은 각 용어에 대해 클릭할 수 있는 작은 레이블을 표시합니다.

고급 구성 섹션에는 다음과 같은 추가 옵션이 있습니다.

**유형 매개변수 이름:** 패싯에 대한 URL에서 매개변수 이름을 변경합니다. 기본값은 **유형** 입니다.

**빈도 임계값:** 패싯 용어 목록에 나타나는 용어에 필요한 최소 빈도를 설정합니다. 예를 들어 패싯의 빈도 임계값이 3으로 설정된 경우 두 개의 일치하는 결과가 있는 용어는 용어 결과 목록에 나타나지 않습니다.

**빈도 표시:** 용어 빈도를 표시할지 여부를 선택합니다.

**용어 주문 기준:** Liferay 7.4 U60+/GA60+의 경우 용어 빈도 내림차순(기본값), 용어 빈도 오름차순, 용어 값 오름차순 또는 용어 값 내림차순과 같은 패싯 용어 정렬 전략을 선택할 수 있습니다. 기본 정렬 순서는 패싯 용어가 가장 많이 일치하는 것부터 가장 적게 일치하는 것까지입니다. 용어 값 옵션은 패싯 용어를 사전순으로 정렬합니다.

**현재 및 사용 가능:** 패싯에 표시되는 자산 유형을 선택합니다. 유형을 제거하려면 클릭하고 강조 표시하여 현재 섹션에서 선택합니다. 오른쪽 화살표를 클릭하고 자산 유형을 **현재** 에서 **사용 가능** 이동합니다. 자산 유형을 현재 섹션으로 이동하여 추가합니다.

![자산 유형을 추가하거나 제거하려면 오른쪽 및 왼쪽 화살표를 사용하십시오.](type-facet/images/03.png)

완료되면 **저장** 버튼을 클릭합니다.
