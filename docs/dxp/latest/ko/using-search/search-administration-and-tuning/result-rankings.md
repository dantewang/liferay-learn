# 결과 순위

> **가입자**

```{note}
이 기능은 [Elasticsearch에서만](../installing-and-upgrading-a-search-engine/solr/solr-limitations.md) 작동합니다.
```

결과 순위는 특정 키워드에 대해 반환된 결과 목록을 변경하기 위한 것입니다.

결과 순위로 다음을 수행할 수 있습니다.

* 결과 목록 상단에 고정할 결과를 지정합니다.
* 검색에 대한 특정 결과를 숨깁니다.
* 일반적으로 특정 키워드에 대해 반환되지 않는 결과를 추가합니다.
* 끌어서 놓기 인터페이스로 고정된 결과를 재정렬합니다.

```{note}
결과 순위는 최신 검색 위젯(검색 표시줄, 검색 결과 등)을 사용하는 검색에만 영향을 미칩니다. 사용자 정의한 순위는 레거시 검색 포틀릿 결과 또는 개별 애플리케이션 검색 표시줄에 적용되지 않습니다.
```

## 요구 사항 및 제한 사항

결과 순위는 Elasticsearch를 검색 엔진으로 사용할 때만 지원됩니다. [Elasticsearch 시작하기](../installing-and-upgrading-a-search-engine/elasticsearch/getting-started-with-elasticsearch.md)에서 Elasticsearch 설치에 대해 알아보세요.

기존 순위 항목은 이름을 바꿀 수 없습니다. 이름을 바꾸려면 다른 이름으로 순위를 다시 만들어야 합니다. 자세한 내용은 [LPS-96357](https://issues.liferay.com/browse/LPS-96357) 을 참조하십시오.

## 결과 순위 생성 및 관리

새로운 결과 순위를 생성하려면,

1. 전역 메뉴(![Global Menu](../../images/icon-applications-menu.png))에서 *애플리케이션* &rarr; *검색 조정* &rarr; *결과 순위*로 이동합니다.

   ![애플리케이션 메뉴에서 결과 순위 섹션으로 이동합니다.](result-rankings/images/01.png)

1. 추가 아이콘(![Click on the add icon](../../images/icon-add.png))을 클릭하여 새 결과 순위를 추가합니다.

1. 사용하려는 키워드 또는 구문을 입력합니다. *결과 사용자 지정*을 클릭합니다.

1. 검색 쿼리가 실행되고 결과가 나타납니다.

   결과 순위는 별칭 필드에 단어나 구를 추가하여 추가 키워드에 적용할 수 있습니다. *을 클릭합니다.* 을 입력하거나 쉼표를 사용하여 추가 별칭을 입력합니다.

   ![별칭 필드에 추가 키워드를 추가합니다.](result-rankings/images/02.png)

   별칭을 제거하려면 키워드 옆에 있는 *X* 을 클릭합니다.

1. 특정 결과를 고정하려면 _옵션_ 아이콘(![Click on the options icon.](../../images/icon-options.png))을 클릭하고 *결과 고정*을 클릭합니다. 결과 위에 마우스를 놓고 핀 아이콘을 클릭할 수도 있습니다.

   ![고정 결과 링크를 클릭합니다.](result-rankings/images/03.png)

   둘 이상의 결과가 고정되면 고정된 결과를 재정렬할 수 있습니다. 재정렬하려는 항목의 왼쪽 가장자리로 커서를 이동합니다. 항목을 클릭한 상태에서 결과 목록 위 또는 아래로 드래그합니다.

   ![고정된 결과의 왼쪽 가장자리를 잡고 위 또는 아래로 끕니다.](result-rankings/images/04.png)

   드래그하면 선택한 항목이 이동할 수 있는 직사각형 상자로 나타납니다. 강조 표시된 파란색 선은 항목 삽입 지점을 나타냅니다.

1. 특정 결과를 숨기려면 _옵션_ (![Click on the options icon.](../../images/icon-options.png)) &rarr; *결과 숨기기*를 클릭하십시오. 결과 위에 마우스를 놓고 결과 숨기기 아이콘을 클릭할 수도 있습니다.

   ![결과 숨기기 링크를 클릭합니다.](result-rankings/images/03.png)

   숨겨진 결과 목록은 결과의 *Hidden* 탭을 클릭하면 확인할 수 있습니다. 숨겨진 결과를 다시 표시하려면 _옵션_ (![Click on the options icon.](../../images/icon-options.png)) &rarr; *결과 표시*를 클릭합니다. 결과 위에 마우스를 놓고 결과 표시 아이콘을 클릭할 수도 있습니다.

1. 추가 결과를 추가하려면 _결과 추가_을 클릭하십시오. 새 창에서 다른 검색어를 사용하여 추가 결과를 찾습니다. 추가하려는 결과를 확인 표시하고 _추가_ 버튼을 클릭합니다.

   ![추가 결과를 검색하고 추가할 결과를 체크 표시합니다.](result-rankings/images/05.png)

   이제 결과 목록에 방금 추가한 추가 결과가 표시됩니다.

 1. 결과 순위 구성을 완료하고 *저장*을 클릭합니다.

 1. 원하는 경우 특정 결과 순위를 비활성화하도록 선택할 수 있습니다. 클릭 _옵션_ (![Click on the options icon.](../../images/icon-options.png)) &rarr; *비활성화*.

   ![검색 쿼리를 비활성화하려면 비활성화를 클릭합니다.](result-rankings/images/06.png)

   결과 순위를 다시 활성화하려면 _옵션_ (![Click on the options icon.](../../images/icon-options.png)) &rarr; *활성화*를 클릭하십시오.

   결과 순위를 영구적으로 삭제하려면 _옵션_ (![Click on the options icon.](../../images/icon-options.png)) &rarr; *삭제*를 클릭하십시오.

## 결과 순위 범위 및 권한

결과 순위 구성은 가상 인스턴스 범위에서 발생하므로 주의해야 할 범위 지정 및 권한 동작이 있습니다.

고정된 결과의 범위는 무시됩니다. 사이트 A에 존재하는 고정된 결과는 검색 창 범위가 검색 *모든 항목*이 아닌 *이 사이트* 로 설정되어 있어도 항상 사이트 B의 검색에 나타납니다.

결과 순위 검색은 전역입니다. 결과 순위 관리에서 결과를 검색하면 모든 사이트의 관련 결과가 반환됩니다.

권한은 평소와 같이 적용됩니다. 사용자에게 자산을 볼 수 있는 권한이 없는 경우 자산을 고정해도 해당 사용자의 검색 결과에 나타나지 않습니다.

## 결과 순위 별칭 대 동의어 집합

결과 순위와 동의어 집합 간의 유사성으로 인해 약간의 혼란이 발생할 수 있습니다. [동의어 집합 ](./synonym-sets.md) 은 추가(동의어) 키워드와 일치하는 결과를 포함하도록 검색을 확장하므로 동의어와 일치하는 항목이 있는 경우 더 많은 결과가 반환됩니다.

결과 순위는 특정 순위 조정이 적용된 키워드입니다. 검색된 키워드만 결과에 매칭되고 이후 고정, 숨김, 재정렬 및 추가 결과가 적용됩니다. 물론 동일한 검색어 집합을 포함하는 동의어 집합을 구성한 Result Ranking으로 추가하면 기능을 함께 사용할 수 있습니다. 모든 동의어는 사용자 지정 순위가 맨 위에 적용된 동일한 결과 집합을 반환합니다.
