# Elasticsearch 커넥터 구성 참조

> 여기에 있는 구성 정보는 Liferay Portal 7.2-7.4 CE 및 Liferay DXP 7.2-7.4용 Elasticsearch 6 및 Elasticsearch 7 커넥터의 최신 버전(번들 또는 Marketplace를 통해)에 적용됩니다. 필요한 경우 정확한 GA/서비스 팩/수정 팩 및 Marketplace 버전에 대한 적절한 정보가 제공됩니다.

Elasticsearch에 대한 연결은 주로 시스템 설정의 _Elasticsearch 6/7_ 구성 항목에서(또는 [해당 구성 파일](#configuration-files-and-system-settings-entries)을 통해) 정의됩니다. Liferay 7.3+에서는 [공장 구성](../../../system-administration/configuring-liferay/configuration-files-and-factories/using-factory-configuration.dll)을 통해 Elasticsearch에 대한 여러 연결을 정의할 수 있습니다. md) _Elasticsearch 연결_. 두 항목 모두 [시스템 설정](../../../system-administration/configuring-liferay/system-settings.md) 또는 [OSGi 구성 파일](../../../system-administration/configuring-liferay/configuration-files-and-factories/using-configuration-files.md)을 통해 구성할 수 있습니다. 구성 파일은 프로덕션 환경에 권장되는 접근 방식입니다.

## 구성 파일 및 시스템 설정 항목

| 서버 연결                          | <div style="width:380px">시스템 설정 항목/구성 파일</div>                                                                                                                                                                                                                                              |
| :----------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| liferay 7.2.x<br />엘라스틱서치 6.x    | Elasticsearch 6<br />`com.liferay.portal.search.elasticsearch6.configuration.ElasticsearchConfiguration.config`                                                                                                                                                                             |
| liferay 7.2.x<br />엘라스틱서치 7.x    | Elasticsearch 7<br />`com.liferay.portal.search.elasticsearch7.configuration.ElasticsearchConfiguration.config`                                                                                                                                                                             |
| liferay 7.3+<br />엘라스틱서치 7.x/8.x | Elasticsearch 7<br />`com.liferay.portal.search.elasticsearch7.configuration.ElasticsearchConfiguration.config`<br /><br />Elasticsearch 연결(공장)<br />`com.liferay.portal.search.elasticsearch7.configuration.ElasticsearchConnectionConfiguration-[connectionId].config` |

Liferay 7.3 이상에는 Elasticsearch 연결이라는 추가 연결 구성 항목이 있습니다. 이를 사용하여 Elasticsearch에 대한 연결을 정의할 수 있지만 하나의 연결만 구성하는 경우 기본 Elasticsearch 7 구성 항목을 계속 사용할 수 있습니다. 7.3 이상에서 여러 연결을 사용하는 경우 그에 따라 이름이 지정된 파일로 연결을 정의하십시오.

```
com.liferay.portal.search.elasticsearch7.configuration.ElasticsearchConnectionConfiguration-[connectionId].config
```

Elasticsearch 6에서 보안을 구성하는 경우 별도의 Liferay 구성(및 LES 구독)이 필요합니다. 자세한 내용은 [Elasticsearch 보안](securing-elasticsearch.md) 참조하세요.

구성 파일을 `[Liferay_Home]/osgi/configs`에 배포하면 리스너가 구성을 자동으로 감지하여 데이터베이스에 기록합니다.
