# Liferay 소프트웨어 버전

Liferay 7.4부터 Liferay는 업데이트 및 수정 사항 제공을 위한 롤링 릴리스 모델을 채택했습니다. 이는 Liferay 7.4가 가까운 미래에 많은 릴리스를 통해 오랫동안 지속되는 릴리스임을 의미합니다. 또한 Liferay DXP 7.4 U92부터 Liferay DXP는 분기별 출시 일정으로 전환되었습니다. 버전을 표시하기 위해 Liferay Portal 및 Liferay DXP는 다음 구성표를 사용합니다.

| 소프트웨어          | 버전            |
| :------------- | :------------ |
| Liferay Portal | 7.4.3.XX GAXX |
| Liferay DXP    | 연도.분기         |

## Liferay 포털 버전 관리

다음은 몇 가지 가상의 Liferay Portal 버전 예입니다.

```
Liferay Portal 7.4.3.30 GA30
Liferay Portal 7.4.3.31 GA31
...
Liferay Portal 7.4.3.100 GA100
```

## Liferay DXP 버전 관리

다음은 몇 가지 가상의 Liferay DXP 버전 예입니다.

```
Liferay DXP 2023.Q3
Liferay DXP 2023.Q4
...
Liferay DXP 2025.Q1
```

```{note}
Liferay DXP 7.4 U92 이전 버전은 Liferay DXP 7.4.13 업데이트 92, Liferay DXP 7.4.13 업데이트 91 등의 형식을 취했습니다.
```

## 이전 버전

이전 Liferay 릴리스는 세 자리 숫자로 구성된 주요, 부, 유지 관리 릴리스의 특정 구조를 따릅니다. 예를 들어 6.1.2. 이 숫자는 릴리스 유형(major.minor.maintenance)을 나타냅니다.

첫 번째 숫자(예: 6.xx에서 7.xx로) 또는 두 번째 숫자(예: 7.2.x에서 7.3.x)의 변경은 주요 또는 부 릴리스를 나타냅니다. 릴리스에는 기능 변경, 아키텍처 변경, API 변경, 새로운 기능 추가 등이 포함될 수 있습니다.

세 번째 숫자의 변경(예: 6.2.1에서 6.2.2)은 유지 관리 릴리스를 나타냅니다. 릴리스에는 보안 수정 사항이나 안정성 개선 사항이 포함될 수 있습니다.
