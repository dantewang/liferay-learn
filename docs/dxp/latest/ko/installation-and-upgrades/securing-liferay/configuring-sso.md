# 싱글 사인온 구성

Liferay Portal/DXP는 토큰 기반 솔루션에서 OpenID Connect 및 SAML과 같은 표준에 이르기까지 다양한 인증 방법을 지원합니다. 사용 가능한 모든 옵션을 통해 자신에게 적합한 옵션을 찾을 수 있습니다.
