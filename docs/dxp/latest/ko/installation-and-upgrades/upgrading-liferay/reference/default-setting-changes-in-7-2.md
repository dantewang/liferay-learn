# 7.2의 기본 설정 변경 사항

대부분의 새 버전의 Liferay에는 기본 설정에 대한 변경 사항이 포함되어 있습니다. 이전 버전의 기본값을 사용하는 경우 변경 사항을 검토하고 이전 버전의 기본값을 유지하거나 새 기본값을 수락할지 결정해야 합니다.
