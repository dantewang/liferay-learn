# 보기에 필터 추가

{bdg-secondary}`라이프레이 7.4 U49+/GA49+`

사용자 정의 보기를 디자인할 때 필터 패싯으로 표시되는 필드를 결정할 수 있습니다. 또한 사용자가 개체의 응용 프로그램 페이지를 방문할 때마다 보기 테이블에 자동으로 적용되는 기본 필터를 설정할 수 있습니다.

다음과 같이하세요:

1. 원하는 보기 편집을 시작하고 **필터** 탭으로 이동합니다.

1. **추가**(![추가 버튼](../../../../images/icon-add.png))를 클릭합니다.

1. 필터링 기준으로 사용할 **필드** 를 선택하세요.

   워크플로 상태, 생성 날짜, 수정 날짜, 선택 목록 필드 또는 관계 필드를 선택할 수 있습니다.

1. 필터 유형을 선택합니다: **포함** 또는 **제외** . 이는 기본 필터에 다음 값을 가진 항목이 포함되거나 제외되는지 여부를 결정합니다.

1. (선택 사항) 기본 필터로 사용할 필드 값을 입력합니다.

   ![Select a field, filter type, and default values.](./adding-filters-to-views/images/01.png)

1. **저장** 을 클릭하세요.

1. 위 프로세스를 반복하여 뷰에 추가 필드 필터를 추가합니다.

   ![Add multiple field filters to the view.](./adding-filters-to-views/images/02.png)

1. **저장** 을 클릭하세요.

일단 저장되면 사용자는 추가된 필드를 사용하여 개체 항목을 필터링할 수 있으며 기본 필터는 자동으로 적용됩니다.

![Users can filter entries using the added fields.](./adding-filters-to-views/images/03.png)
