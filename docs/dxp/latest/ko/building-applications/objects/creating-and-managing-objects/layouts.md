# 배치

{bdg-secondary}`라이프레이 7.4+`

개체 레이아웃은 항목을 생성하고 편집할 때 필드와 관계가 나타나는 방식을 정의합니다. 게시된 모든 개체에는 개체의 필드가 사전순으로 나열되는 단일 탭이 있는 자동 생성 레이아웃이 포함됩니다. 사용자 정의 개체에 대한 [자신만의 레이아웃을 디자인](./layouts/designing-object-layouts.md)하여 해당 필드와 관계가 표시되는 방식을 결정할 수 있습니다. 7.4 U31+ 및 GA31+의 경우 최종 사용자가 개체 항목에 태그와 범주를 적용할 수 있도록 탭에 [분류 도구](./layouts/adding-categorization-to-fields-tabs.md)를 추가할 수도 있습니다.

```{important}
자동 생성된 레이아웃은 객체 관계를 표시하지 않습니다. 관계를 표시하려면 사용자 지정 레이아웃을 만들어야 합니다.
```
