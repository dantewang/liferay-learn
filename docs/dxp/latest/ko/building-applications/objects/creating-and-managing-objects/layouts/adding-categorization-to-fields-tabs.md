# 필드 탭에 분류 추가

{bdg-secondary}`라이프레이 7.4 U31+/GA31+`

[필드 탭](./designing-object-layouts.md#fields-tabs) 에 분류 블록을 추가하면 최종 사용자가 객체 항목에 태그와 범주를 적용할 수 있습니다. 다음과 같이하세요:

1. **전역 메뉴**(![전역 메뉴](../../../../images/icon-applications-menu.png))를 열고 **제어판** 탭으로 이동한 후 **개체를 클릭합니다.** .

1. 원하는 객체 정의 편집을 시작합니다.

   ```{note}
   시스템 객체는 사용자 지정 레이아웃을 지원하지 않습니다. 자세한 내용은 [시스템 객체 확장](../extending-system-objects.md)을 참조하세요.
   ```

1. 세부정보 탭에서 분류를 활성화하고 **저장** 을 클릭하세요.

   ![Enable categorization.](./adding-categorization-to-fields-tabs/images/01.png)

1. **레이아웃** 탭을 클릭하고 원하는 레이아웃을 선택합니다.

1. **레이아웃** 탭으로 이동하여 원하는 [필드 탭](./ designing-object-layouts.md#fields-tabs) , **분류 추가** 를 선택합니다.

   ![Click the Actions button and select Add Categorization.](./adding-categorization-to-fields-tabs/images/02.png)

   그러면 분류 도구에 액세스하기 위한 새 블록이 탭에 추가됩니다.

   ```{important}
   분류 블록은 레이아웃당 하나의 탭에만 추가할 수 있습니다.
   ```

1. 블록이 **접을 수** 있는지 확인합니다.

   ![Determine whether the block is collapsible.](./adding-categorization-to-fields-tabs/images/03.png)

1. **저장** 을 클릭하세요.

이제 개체 항목을 생성하거나 편집할 때 사용자는 태그와 범주에 액세스하고 적용할 수 있습니다.

![Access and apply tags and categories to the object's entries.](./adding-categorization-to-fields-tabs/images/04.png)

```{note}
가시성 및 권한은 분류 블록이 표시되는 방식에 영향을 줄 수 있습니다.
```
