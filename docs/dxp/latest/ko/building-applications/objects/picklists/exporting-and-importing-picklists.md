# 선택 목록 내보내기 및 가져오기

Liferay의 내보내기/가져오기 프레임워크를 사용하여 환경 간에 선택 목록을 JSON 파일로 전송할 수 있습니다.

## 선택 목록 내보내기

1. **전역 메뉴**(![전역 메뉴](../../../images/icon-applications-menu.png))를 열고 **제어판** 탭으로 이동한 다음 **선택 목록** 을 클릭합니다.

1. 원하는 목록에 대해 **작업**(![작업 버튼](../../../images/icon-actions.png))을 클릭하고 **JSON으로 내보내기** 를 선택합니다.

   그러면 선택 목록 정의와 항목이 JSON 파일로 다운로드됩니다.

   ![Click the Actions button for the desired list and select Export as JSON.](./exporting-and-importing-picklists/images/01.png)

## 선택 목록 가져오기

1. **전역 메뉴**(![전역 메뉴](../../../images/icon-applications-menu.png))를 열고 **제어판** 탭으로 이동한 다음 **선택 목록** 을 클릭합니다.

1. 응용 프로그램 표시줄에서 **작업**(![작업 버튼](../../../images/icon-actions.png))을 클릭하고 **선택 목록 가져오기** 를 선택합니다.

   ![Click the Action button in the Application Bar and select Import Picklist.](./exporting-and-importing-picklists/images/02.png)

1. 모달 창에서 목록 이름을 입력하고 가져올 선택 목록 JSON을 선택합니다.

1. **가져오기** 를 클릭하세요.

   ![Enter a name for the list and select a picklist to import.](./exporting-and-importing-picklists/images/03.png)
