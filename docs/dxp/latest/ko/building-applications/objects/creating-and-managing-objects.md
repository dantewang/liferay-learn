{bdg-secondary}`사용 가능한 Liferay DXP/Portal 7.4+`

Liferay Objects는 코드를 작성하지 않고도 비즈니스 요구 사항을 충족하는 [](./creating-and-managing-objects/creating-objects.md) 맞춤형 애플리케이션을 만들기 위한 편리한 UI를 제공합니다. 개체를 사용하여 시스템 개체와 사용자 지정 개체를 모두 관리하고 확장할 수도 있습니다. 여기에는 [필드](./creating-and-managing-objects/fields.md), [관계](./creating-and-managing-objects/relationships.md), [작업](./creating-and-managing-objects/actions.md), [레이아웃](./creating-and-managing-objects/layouts.md), [보기](./creating-and-managing-objects/views.md), [유효성 검사](./creating-and-managing-objects/validations.md)및 [사용자 정의 상태](./creating-and-managing-objects/fields/adding-and-managing-custom-states.md)추가가 포함됩니다. 이러한 작업 및 개체 정의와 관련된 기타 작업을 수행하는 데 필요한 권한에 대해 알아보려면 [개체 응용 프로그램 권한](./objects-application-permissions.md) 참조하십시오.

```{note}
시스템 개체는 개체 프레임워크와 통합된 기본 Liferay 응용 프로그램인 반면 사용자 지정 개체는 사용자가 만들고 게시한 정의입니다.
```
