# 사용자 정의 필드 추가

Liferay의 많은 자산과 리소스는 사용자 지정 필드로 확장할 수 있습니다. 사용자 정의 필드를 추가하면 사이트를 더 잘 제어하고 유연하게 사용할 수 있습니다. 사용자 지정 필드를 사용할 수 있는 모든 위치는 다음과 같습니다.

* Blogs입장
* 캘린더 이벤트
* Documents and Media Document
* 문서 도서관 폴더
* 기술 자료 문서
* 기술 자료 폴더
* 전보국 종류
* 전보국 메시지
* 조직
* 페이지
* 역할
* 사이트
* 사이트 내비게이션 메뉴 항목
* 사용자
* 사용자 그룹
* 웹 내용
* 웹 내용 급식
* Wiki페이지

사용자 정의 필드를 사용하는 데는 여러 가지 이유가 있습니다. 여기에 한 가지 예가 있습니다.

**최대 역할 사용 사례**: 사이트의 특정 역할에 할당할 수 있는 사용자 수를 제한한다고 가정합니다. 이를 위해 역할에 대해 *max-users* 사용자 정의 필드를 생성할 수 있습니다. 그런 다음 개발자에게 설정된 제한에 도달하면 더 많은 사용자가 추가되는 것을 추적하고 방지하는 로직을 생성하도록 요청하십시오.

## 사용자 정의 필드 추가

위의 예제 사용 사례에 대해 사용자 정의 필드를 만드는 방법을 보려면 다음 단계를 따르십시오.

1. 제어판 → 구성 → 사용자 정의 필드로 이동합니다.

    ![제어판의 구성 섹션에서 사용자 정의 필드로 이동합니다.](adding-custom-fields/images/01.png)

1. 아래로 스크롤하여 *역할*링크를 클릭합니다.

1. ** 추가 아이콘(![Click on the add icon to add a custom field.](../../images/icon-add.png))을 클릭하여 사용자 정의 필드를 추가합니다.

1. 선택할 수 있는 여러 필드 유형이 있습니다.

    ![선택할 수 있는 다양한 필드 유형이 있습니다.](adding-custom-fields/images/02.png)

    * **텍스트 및 숫자**
        * **텍스트 영역**: 이 필드 유형은 더 긴 텍스트 행을 수집할 수 있습니다.
        * **입력 필드**: 이 필드 유형은 텍스트 문자열, 십진수 또는 정수를 수집할 수 있습니다.

    * **선택**
        * **드롭다운**: 이 필드 유형은 선택할 항목의 드롭다운 목록을 제공할 수 있습니다. 목록은 텍스트, 십진수 또는 정수일 수 있습니다.
        * **확인란**: 이 필드 유형은 확인란을 표시합니다.
        * **라디오**: 이 필드 유형은 두 개 이상의 라디오 버튼 선택을 보여줍니다.

    * **다른이들**
        * **지리적 위치**: 이 필드 유형은 지리적 위치 정보를 수집합니다.
        * **날짜**: 이 필드 유형은 날짜 및 시간 정보를 수집합니다.
        * **참/거짓**: 이 필드 유형은 참 또는 거짓 선택을 표시합니다.

1. 위 사용 사례의 경우 필드 유형으로 *입력 필드* 을 선택합니다. *필드 이름* 을 입력하고 데이터 유형을 정수로 선택합니다.

    ![개발자가 이 이름을 사용하여 액세스해야 하므로 사용자 정의 필드에 기억할 수 있는 이름을 지정하십시오.](adding-custom-fields/images/03.png)

    *저장* 버튼을 클릭하여 변경 사항을 저장합니다.

    ```{note} 
    필드 이름은 데이터베이스에 저장된 식별자이며 개발자가 사용자 정의 필드에 액세스하는 데 사용하는 이름입니다.
    ```

사용자 지정 필드를 만든 후에는 해당 필드를 만든 리소스의 UI에서 찾을 수 있습니다. 리소스로 이동하면 사용자 정의 필드가 UI에 표시됩니다.

예를 들어 *max-users* 사용자 정의 필드는 이제 모든 역할에 대해 표시됩니다.

1. 제어판 → 사용자 → 역할로 이동합니다.

1. 역할에 대한 옵션 아이콘(![Click on the options icon to add a custom field.](../../images/icon-options.png))을 클릭하고 편집을 클릭합니다.

    ![사용자 정의 필드는 리소스 UI에 표시되지 않습니다.](adding-custom-fields/images/04.png)

    맨 아래에 있는 *Max Users* 필드를 확인하십시오.

## 사용자 정의 필드 편집

편집할 사용자 정의 필드 이름 옆에 있는 옵션 아이콘(![Click on the options icon for the custom field.](../../images/icon-options.png))을 클릭합니다. 다음 옵션이 나타납니다.

![옵션 아이콘을 클릭하면 다양한 선택이 제공됩니다.](adding-custom-fields/images/05.png)

**편집:** 사용자 정의 필드의 다양한 측면을 변경할 수 있지만 사용자 정의 필드 이름이나 사용자 정의 필드 유형은 변경할 수 없습니다.

![사용자 정의 필드를 변경하려면 편집을 클릭하여 새 창을 엽니다.](adding-custom-fields/images/06.png)

가지고 있는 구성 옵션은 선택한 필드 유형에 따라 달라집니다.

**권한:** 다른 사용자가 사용자 정의 필드를 편집하거나 볼 수 있도록 권한을 수정할 수 있습니다.

**삭제:** 사용자 정의 필드를 삭제할 수 있습니다.
