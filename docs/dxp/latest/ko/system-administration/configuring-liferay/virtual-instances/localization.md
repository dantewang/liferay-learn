# 지역화

현지화 범주에는 다음 항목이 포함됩니다.

| 설정                   | 목적                                                   |
|:-------------------- |:---------------------------------------------------- |
| 언어                   | 기본 인스턴스 언어와 인스턴스에서 사용할 수 있는 언어를 설정합니다.               |
| 시간대                  | 인스턴스의 시간대 설정                                         |
| 언어 선택기(WIDGET SCOPE) | 언어 선택기 위젯의 인스턴스 범위 기본값을 설정합니다. 위젯 자체에서 이 설정을 재정의합니다. |

현지화 설정에 액세스하려면 제어판을 열고 *구성* &rarr; *인스턴스 설정*로 이동한 다음 *플랫폼* 섹션에서 *현지화* 범주를 선택합니다.

![인스턴스 범위에 대한 기본 언어 및 표준 시간대를 구성합니다.](./localization/images/01.png)

필요에 따라 왼쪽 및 오른쪽 화살표 버튼을 사용하여 언어를 현재(구성 및 활성)에서 사용 가능(비활성이지만 구성 가능)으로 또는 그 반대로 이동합니다. 위쪽 및 아래쪽 화살표 버튼을 사용하여 원하는 순서대로 현재 언어를 정렬합니다.
