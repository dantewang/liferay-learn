# 가상 인스턴스 추가

가상 인스턴스를 추가하기 위해 Liferay DXP에서 어떤 작업을 수행하기 전에 설치된 Liferay DXP 시스템을 가리키도록 네트워크의 도메인 이름을 구성해야 합니다.

![기본적으로 하나의 가상 인스턴스가 있습니다.](./adding-a-virtual-instance/images/01.png)

도메인 이름이 준비되면 가상 인스턴스를 만듭니다.

1. **제어판** &rarr; **구성** &rarr; **가상 인스턴스** 로 이동합니다.

1. **추가** 버튼(![Add](../../../images/icon-add.png))을 클릭합니다. 그러면 **New Instance** 양식이 열립니다.

1. 다음과 같이 새 인스턴스 양식을 작성하십시오.

![가상 인스턴스 추가 양식을 작성합니다.](./adding-a-virtual-instance/images/02.png)

**웹 ID:** 인스턴스 ID를 지정합니다. 도메인 이름을 사용하는 것은 일반적인 규칙입니다.

**가상 호스트:** 네트워크에서 구성한 도메인 이름을 설정합니다. 사람들이 도메인 이름을 사용하면 자동으로 이 인스턴스로 연결됩니다.

**메일 도메인:** 가상 인스턴스에 대한 메일 호스트의 도메인 이름을 입력합니다. 이메일 알림은 이 도메인을 사용하는 인스턴스에서 전송됩니다.

**최대 사용자:** 가상 인스턴스가 지원하는 최대 사용자 계정 수를 설정합니다. 무제한 사용자를 지원하려면 `0` 을 입력합니다.

**활성:** 가상 인스턴스를 활성 또는 비활성으로 설정합니다. 비활성 가상 인스턴스는 관리자를 포함하여 누구도 액세스할 수 없습니다.

양식 작성을 마쳤으면 **저장** 을 클릭합니다.

이제 도메인 이름을 사용하여 인스턴스로 이동할 수 있습니다. Liferay DXP를 새로 설치한 것처럼 보입니다. 이것은 새로운 가상 인스턴스입니다! 원하는 방식으로 구성할 수 있습니다.
