# 사용자

사용자 구성은 개인 메뉴의 모양과 느낌, 기본 사용자 연결 및 새 사용자가 계정을 만들 때 사용할 수 있는 필드를 정의합니다.

사용자 설정에 액세스하려면 제어판을 열고 *구성* &rarr; *인스턴스 설정*로 이동한 다음 *PLATFORM* 섹션에서 *사용자* 범주를 선택합니다.

사용자 인스턴스 구성에는 세 가지 항목이 있습니다.

| 설정        | 목적                                                  |
|:--------- |:--------------------------------------------------- |
| 개인 메뉴     | 개인 애플리케이션의 모양과 느낌 및 개인 메뉴가 컨트롤 메뉴에 표시되는지 여부를 구성합니다. |
| 과태 사용자 협회 | 일부 사이트, 역할 및 사용자 그룹에 자동으로 멤버십 부여                    |
| 분야        | 사용자 양식에서 일부 선택적 필드 활성화 및 비활성화                       |

## 개인 메뉴

개인 메뉴 구성 항목은 개인 응용 프로그램이 현재 사이트와 동일한 모양과 느낌을 사용하는지 또는 대신 내 대시보드 페이지의 모양과 느낌을 사용해야 하는지 여부를 지정합니다. *컨트롤 메뉴* 에 표시 옵션을 켜거나 꺼서 컨트롤 메뉴에 개인 메뉴를 표시할지 여부를 지정할 수도 있습니다.

![인스턴스 범위에서 개인 메뉴를 구성합니다.](./users/images/01.png)

## 과태 사용자 협회

기본 사용자 연결 구성 항목은 모든 사용자가 기본적으로 할당되는 사이트, 조직 사이트, 역할 및 사용자 그룹을 정의합니다. 기본적으로 새 사용자는 사용자 역할만 받습니다. 사용자 그룹은 사용자의 개인 사이트를 페이지로 미리 채우고 역할 및 권한을 할당하며 사이트 구성원을 관리할 수 있습니다. 사용자가 계정을 생성한 후 이 구성을 업데이트하는 경우 *기존 사용자에게 적용* 확인란을 클릭하여 기존 사용자를 업데이트합니다. 변경 사항은 다음에 사용자가 로그인할 때 적용됩니다.

![모든 인스턴스의 새 사용자에 대한 기본 연결을 지정합니다.](./users/images/02.png)

## 분야

필드 구성 항목에는 사용자 추가/편집 양식에서 아래 나열된 필드를 활성화/비활성화하기 위한 설정이 포함되어 있습니다.

* 화면 이름 자동 생성(기본적으로 비활성화됨)
* 생일 필드(기본적으로 활성화됨)
* 성별 필드(기본적으로 비활성화됨)
