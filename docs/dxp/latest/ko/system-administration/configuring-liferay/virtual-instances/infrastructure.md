# 인프라스트럭처

인프라 범주에는 가져오기/내보내기 프로세스 중에 유효성을 검사할 콘텐츠 유형과 가져오기/내보내기 프로세스 중에 임시 LAR 파일을 삭제할지 여부를 지정하는 설정이 포함되어 있습니다.

![가상 인스턴스 범위에서 가져오기 내보내기 설정을 구성합니다.](./infrastructure/images/01.png)

이러한 설정에 액세스하려면 제어판을 열고 *구성* &rarr; *인스턴스 설정* 로 이동한 다음 플랫폼 제목 아래에서 *인프라* 을 선택합니다.

| 설정                                  | 목적                                                                                                                                                         |
|:----------------------------------- |:---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 파일 항목 확인                            | 선택한 경우 일부 유형의 텍스트 콘텐츠를 게시하거나 가져오면 Liferay DXP가 콘텐츠에서 문서 링크를 확인하고 각 링크가 유효한 문서를 가리키는지 확인합니다.                                                                |
| 저널 피드 참조 유효성 검사                     | 선택한 경우 일부 유형의 텍스트 콘텐츠를 게시하거나 가져오면 Liferay DXP가 콘텐츠에서 웹 콘텐츠 피드에 대한 참조를 확인하고 각 참조가 유효한 피드를 가리키는지 확인합니다.                                                      |
| 레이아웃 참조 유효성 검사                      | 선택한 경우 일부 유형의 텍스트 콘텐츠를 게시하거나 가져오면 Liferay DXP 가상 호스트를 참조하는 링크의 유효성 검사가 트리거되어 각 링크가 포털 내의 유효한 페이지를 가리키도록 합니다. 가상 호스트에 Liferay DXP 외부 페이지가 있는 경우 선택을 취소하십시오. |
| 누락된 참조 유효성 검증                       | 선택하면 게시 또는 가져오기에서 누락된 참조에 대한 확인을 트리거합니다. 선택하지 않으면 누락된 참조 유효성 검사를 건너뛰고 기존 대상이 없는 참조를 게시하거나 대상 사이트로 가져올 수 있습니다.                                              |
| 실패한 스테이징 게시 프로세스 중에 임시 LAR을 삭제합니다.  | 실패한 스테이징 게시 프로세스 중에 임시 LAR을 삭제하지 않으려면 선택을 취소하십시오. 원격 스테이징 컨텍스트에서 이는 스테이징 환경에만 적용됩니다.                                                                       |
| 성공적인 스테이징 게시 프로세스 중에 임시 LAR을 삭제합니다. | 성공적인 스테이징 게시 프로세스 중에 임시 LAR을 삭제하지 않으려면 선택을 취소하십시오. 원격 스테이징 컨텍스트에서 이는 스테이징 환경에만 적용됩니다.                                                                      |
| 원격 사이트의 가상 호스트를 사용하십시오.             | 원격 환경의 가상 호스트는 스테이징된 사이트에서 원격 라이브 사이트에 액세스할 때 사용됩니다.                                                                                                       |
