# 이메일 설정

이메일 구성은 가상 인스턴스 범위에서 이메일 알림을 구성하는 곳입니다.

이러한 옵션에 액세스하려면

1. **전역 메뉴**(![Global Menu](../../../images/icon-applications-menu.png))을 열고 **제어판** &rarr; **구성** &rarr; **인스턴스 설정** 으로 이동합니다.

1. 플랫폼에서 **이메일** 선택합니다.

구성 옵션이 왼쪽에 나타납니다.

| 설정                                             | 목적                                          |
|:---------------------------------------------- |:------------------------------------------- |
| [계정 생성 알림](#account-created-notification)      | 계정 생성에 대한 이메일 알림을 활성화/비활성화하고 알림 템플릿을 결정합니다. |
| [이메일 발송자](#email-sender)                       | 이메일 템플릿의 발신자 이름과 주소를 설정합니다.                 |
| [이메일 확인 알림](#email-verification-notification)  | 사용자에게 이메일 주소를 확인하도록 요청하는 이메일 템플릿을 결정합니다.    |
| [우편물 주인 이름](#mail-host-names)                  | 인스턴스와 연결된 메일 호스트를 입력합니다.                    |
| 메일 설정                                          | 메일 서버를 인스턴스에 연결합니다.                         |
| [암호는 통보를 변화했다](#password-changed-notification) | 비밀번호 변경을 위한 이메일 템플릿을 결정합니다.                 |
| [암호는 통보를 변화했다](#password-reset-notification)   | 비밀번호 재설정 요청을 위한 이메일 템플릿을 결정합니다.             |

```{note}
이메일 알림을 보내려면 설치에 [메일 서버에 대한 연결 구성](../../../installation-and-upgrades/setting-up-liferay/configuring-mail.md)이 있어야 합니다. 인스턴스 범위에서 설정된 메일 서버 구성은 서버 수준 구성을 재정의합니다. 

각 알림 템플릿에 대해 페이지 하단에서 사용 가능한 템플릿 변수를 볼 수 있습니다. 이러한 변수를 사용하여 이메일을 동적으로 채웁니다.
```

## 계정 생성 알림

이러한 템플릿은 계정을 만든 후 사용자에게 보내는 이메일을 정의합니다. 한 템플릿에는 암호 설정 링크가 포함되어 있지만 다른 템플릿에는 포함되어 있지 않습니다.

이 알림을 활성화하려면 상단의 **활성화** 확인란을 사용하십시오.

![계정 생성 이메일 알림 구성](./email-settings/images/01.png)

## 이메일 발송자

이메일 발신자는 이메일 알림을 위한 가상 인스턴스의 관리 이름과 이메일 주소를 지정합니다. `[$FROM_NAME$]` 및 `[$FROM_ADDRESS$]` 변수를 사용하여 알림 템플릿에 이러한 값을 포함할 수 있습니다. [포털 속성](https://resources.learn.liferay.com/reference/latest/en/dxp/propertiesdoc/portal.properties.html) 의 기본값은 `admin.email.from.name` 및 `admin.email.from.address`입니다. 이 이름과 이메일 주소는 가상 인스턴스에서 보낸 모든 이메일 메시지의 **From** 필드에 나타납니다.

![시스템 이메일의 보낸 사람 이름과 주소를 구성합니다.](./email-settings/images/02.png)

## 이메일 확인 알림

이 템플릿은 이메일 주소를 확인하라는 메시지가 표시될 때 사용자에게 전송되는 이메일을 정의합니다.

![사용자의 이메일 주소를 확인합니다.](./email-settings/images/03.png)

## 우편물 주인 이름

여기에서 가상 인스턴스에 대해 조직에서 소유한 추가 메일 호스트 이름을 지정할 수 있습니다. 한 줄에 하나의 메일 호스트 이름을 입력하되 가상 인스턴스 구성에 지정된 이름은 입력하지 마십시오.

![가상 인스턴스에 대한 추가 메일 호스트 이름을 구성합니다.](./email-settings/images/04.png)

## 암호는 통보를 변화했다

이 템플릿은 사용자에게 비밀번호가 변경되었음을 알리는 이메일을 정의합니다.

![비밀번호가 변경되면 사용자에게 알립니다.](./email-settings/images/05.png)

## 암호는 통보를 변화했다

이 템플릿은 비밀번호 재설정 요청을 위한 것입니다.

![비밀번호가 재설정되면 사용자에게 알립니다.](./email-settings/images/06.png)
