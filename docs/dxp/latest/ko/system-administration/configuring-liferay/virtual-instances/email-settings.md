# 이메일 설정

이메일 구성은 가상 인스턴스 범위에서 이메일 알림을 구성하는 곳입니다. 제어판을 열고 **구성** &rarr; **인스턴스 설정** 로 이동한 다음 플랫폼 섹션에서 **이메일** 범주를 선택합니다. 이메일 구성에는 다음 6개 항목이 포함됩니다.

| 설정           | 목적                                       |
|:------------ |:---------------------------------------- |
| 계정 생성 알림     | 새 계정을 만들 때 사용자에게 이메일을 보낼지 여부와 내용을 설정합니다. |
| 이메일 발송자      | 이메일 템플릿의 **From** 이름 및 주소 설정               |
| 이메일 확인 알림    | 사용자에게 이메일 주소 확인을 요청하는 템플릿 만들기            |
| 우편물 주인 이름    | 인스턴스와 연결된 메일 호스트를 입력하세요.                 |
| 메일 설정        | 인스턴스에 대한 메일 서버 설정                        |
| 암호는 통보를 변화했다 | 비밀번호가 변경되면 사용자에게 알림                      |
| 암호는 통보를 변화했다 | 비밀번호 재설정 요청이 수신되면 사용자에게 알림               |

```{note}
이메일 알림을 보내려면 설치에 [메일 서버에 대한 연결 구성](../../../installation-and-upgrades/setting-up-liferay/configuring-mail.md)이 있어야 합니다. 인스턴스 범위에서 설정된 메일 서버 구성은 서버 수준 구성을 재정의합니다. 
```

## 계정 생성 알림

계정 생성 알림 항목은 사용자가 새 계정을 생성할 때 사용자에게 알리기 위해 전송되는 메시지에 대해 텍스트 본문에 포함된 새 사용자의 비밀번호를 포함하거나 포함하지 않는 이메일 템플릿을 정의합니다. 구성 항목 맨 위에 있는 **사용** 확인란을 통해 이 이메일을 보낼지 여부를 지정할 수 있습니다. 이메일 템플릿을 구축하는 데 도움이 되도록 "용어 정의" 제목 아래 구성 하단에 나열된 템플릿 변수를 사용합니다.

![계정 생성 이메일 알림 구성](./email-settings/images/01.png)

## 이메일 발송자

Email Sender 항목은 이메일 템플릿에서 각각 `[$FROM_NAME$]` 및 `[$FROM_ADDRESS$]` 변수로 선언된 이메일 알림을 위한 가상 인스턴스의 관리 이름 및 주소를 지정합니다. 기본적으로 `admin.email.from.name` 및 `admin.email.from.address` [포털 속성](https://learn.liferay.com/reference/latest/en/dxp/propertiesdoc/portal.properties.html#Admin%20Portlet) 에서 가져옵니다. 이 이름과 이메일 주소는 가상 인스턴스에서 보낸 모든 이메일 메시지의 **From** 필드에 나타납니다.

![시스템 이메일의 보낸 사람 이름과 주소를 구성합니다.](./email-settings/images/02.png)

## 이메일 확인 알림

이메일 확인 알림 항목은 이메일 주소를 확인하라는 메시지가 표시될 때 사용자에게 전송되는 메시지의 이메일 템플릿을 정의합니다. 이메일 템플릿을 구축하는 데 도움이 되도록 "용어 정의" 제목 아래 구성 하단에 나열된 템플릿 변수를 사용합니다.

![사용자의 이메일 주소를 확인하십시오.](./email-settings/images/03.png)

## 우편물 주인 이름

메일 호스트 이름 항목을 사용하여 가상 인스턴스에 대해 조직에서 소유한 추가 메일 호스트 이름을 지정하십시오. 한 줄에 하나의 메일 호스트 이름을 입력하되 가상 인스턴스 구성에 지정된 이름은 입력하지 마십시오.

![가상 인스턴스에 대한 추가 메일 호스트 이름을 구성합니다.](./email-settings/images/04.png)

## 암호는 통보를 변화했다

비밀번호 변경 알림 항목은 비밀번호가 변경되었을 때 사용자에게 알리기 위해 전송되는 메시지의 이메일 템플릿을 정의합니다. 이메일 템플릿을 구축하는 데 도움이 되도록 "용어 정의" 제목 아래 구성 하단에 나열된 템플릿 변수를 사용합니다.

![비밀번호가 변경되면 사용자에게 알립니다.](./email-settings/images/05.png)

## 암호는 통보를 변화했다

비밀번호 재설정 알림 항목은 비밀번호 재설정 요청이 있을 때 사용자에게 알리기 위해 전송되는 메시지의 이메일 템플릿을 정의합니다. 이메일 템플릿을 구축하는 데 도움이 되도록 "용어 정의" 제목 아래 구성 하단에 나열된 템플릿 변수를 사용합니다.

![암호 재설정이 요청되면 사용자에게 알립니다.](./email-settings/images/06.png)
