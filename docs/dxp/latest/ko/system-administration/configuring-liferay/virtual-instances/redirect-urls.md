# 리디렉션 URL

{bdg-secondary}`가능 7.4+`

특정 URL 또는 IP 주소로의 리디렉션을 허용하도록 Liferay 인스턴스를 구성할 수 있습니다. 구성하려면 **제어판** &rarr; **인스턴스 설정** &rarr; **콘텐츠 및 데이터** &rarr; **페이지** &rarr; **리디렉션 URL** 으로 이동합니다.

![인스턴스 설정에서 리디렉션 URL로 이동합니다.](./redirect-urls/images/01.png)

보안 모드에서 **도메인** 또는 **IP** 중에서 선택합니다. Liferay 인스턴스가 사용자를 허용된 도메인으로 리디렉션하도록 하려면 **도메인** 을 선택합니다. Liferay 인스턴스가 사용자를 허용된 IP 주소로 리디렉션하도록 하려면 **IP** 을 선택하십시오.

**Domain** 을 선택한 경우 **Allowed Domains** 아래에 허용된 도메인을 추가하고 **Save** 를 클릭합니다. 모든 도메인에 대한 리디렉션을 허용하려면 기존 항목을 모두 제거하십시오. 하위 도메인으로의 리디렉션을 허용하려면 선행 **_* 로 도메인을 지정하십시오.

**IP** 을 선택한 경우 Allowed IPs 아래에 허용된 IP 주소를 추가하고 **Save** 를 클릭합니다. 모든 IP 주소로의 리디렉션을 허용하려면 기존 항목을 모두 제거하십시오.
