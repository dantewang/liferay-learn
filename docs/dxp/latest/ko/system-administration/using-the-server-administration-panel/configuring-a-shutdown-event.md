# 종료 이벤트 구성

종료 탭에서 정상적인 종료 이벤트를 시작하여 사용자에게 임박한 종료를 경고하고 진행 중인 작업을 저장하도록 지시할 수 있습니다.

종료를 예약하면 Liferay DXP는 시스템 경고 메시지와 사용자 페이지 상단에 정의한 메시지를 표시합니다. 시간이 만료되면 모든 페이지에 맞춤 메시지가 표시됩니다. 서버가 다시 시작되면 액세스가 복원됩니다.

## 종료 이벤트 추가

종료 이벤트를 추가하려면 먼저 시스템 설정에서 비활성 요청 처리기 구성을 활성화합니다.

1. 제어판 &rarr; 구성 &rarr; 시스템 설정으로 이동하여 **비활성 요청 처리기** 을 검색합니다.

1. **비활성 요청 메시지 표시** 을 활성화하고 **저장** 을 클릭합니다.

이렇게 하면 종료가 발생한 후 Liferay DXP에서 어딘가를 탐색하려는 사용자에게 메시지를 표시할 수 있습니다. 이것이 없으면 그들은 완전히 빈 페이지를 보는 비우호적인 경험을 하게 될 것입니다.

이제 종료 자체를 구성합니다.

1. 제어판 &rarr; 구성 &rarr; 서버 관리로 이동하고 **종료** 탭을 클릭합니다.

1. 종료될 때까지의 시간(분)을 입력합니다.

1. 특별한 메시지를 입력하세요. 무엇을 입력하든 시스템 기본 메시지에 추가됩니다.

    `Maintenance Alert 4:48 PM EST The portal will shut down for maintenance in 5 minute(s). You will automatically be signed out at that time. Please finish any work in progress. After the maintenance has been completed, you will be able to successfully sign in. [YOUR MESSAGE DISPLAYS HERE]`

   또한 이 메시지는 사용자가 종료 후 포털 페이지로 이동하는 경우 표시되는 메시지입니다(**비활성 요청 메시지 표시** 설정을 활성화했기 때문).

1. **저장** 을 클릭합니다. 종료 이벤트에 대한 카운트다운이 즉시 시작됩니다.

![종료까지의 시간 및 경고 메시지는 종료 탭에서 구성할 수 있습니다.](./configuring-a-shutdown-event/images/01.png)

로그인한 모든 사용자는 페이지 상단에서 시스템 메시지와 추가된 사용자 지정 메시지를 즉시 볼 수 있습니다.

![임박한 종료에 대한 경고가 사용자에게 표시됩니다.](./configuring-a-shutdown-event/images/03.png)

시간이 만료되면 로그인한 사용자가 Liferay DXP의 페이지로 이동하려고 하면 정보가 없는 빈 페이지가 아니라 사용자 지정 메시지(아마도 나중에 다시 돌아오라는 초대 포함)가 표시됩니다.

![종료 후 사용자 지정 메시지가 사용자에게 표시됩니다.](./configuring-a-shutdown-event/images/02.png)

## 종료 이벤트 취소

종료가 발생하기 전에 언제든지 취소할 수 있습니다.

1. 제어판 &rarr; 구성 &rarr; 서버 관리로 이동하고 **종료** 탭을 클릭합니다.

   종료 이벤트가 예약된 경우 취소하는 한 가지 작업만 수행할 수 있습니다.

1. **종료 취소** 을 클릭합니다.

이제 예약된 종료 메시지가 사라지고 포털이 평소와 같이 작동합니다.
