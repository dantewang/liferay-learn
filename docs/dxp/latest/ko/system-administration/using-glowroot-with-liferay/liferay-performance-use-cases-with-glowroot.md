# Glowroot를 사용한 Liferay 성능 사용 사례

{bdg-secondary}`liferay DXP 2023.Q4+/포탈 7.4 GA100+`

[Glowroot](https://glowroot.org/) 사용하여 Liferay 설치의 성능 개선에 대해 진단하고 알려주세요. 다음은 가능한 사용 사례의 몇 가지 예입니다.

## 대규모 컬렉션 반환

Glowroot에서 트랜잭션 탭을 모니터링하면서 일부 느린 트랜잭션을 식별할 수 있습니다. 예를 들어, `blog-postings`에 대한 트랜잭션이 매우 오랜 시간(예: 30초)이 걸리는 것을 발견했습니다.

이 문제를 빠르게 해결하려면 'pageSize'를 맞춤설정하여 거래당 더 적은 수의 게시물을 반환하는 것입니다. 또는 필터를 적용하여 거래에 대한 특정 블로그 게시물만 반환할 수도 있습니다. Glowroot를 사용하여 다양한 트랜잭션의 응답 시간을 비교하세요.

## 사용자 정의 패싯 최적화

Liferay는 [사용자 정의 패싯](../../using-search/search-pages-and-widgets/search-facets/custom-facet.md) 지원하므로 사이트 검색을 최적화할 수 있습니다.

사용자 정의 패싯을 사용하면 여러 패싯 항목을 추가할 때 페이지 응답 시간이 느려질 수 있습니다. Glowroot를 사용하여 트랜잭션을 검사하고 검색 응답을 최적화합니다. 예를 들어 항목 수를 반환하면 속도가 느려질 수 있습니다. 개수 계산 호출을 제거하면 성능이 향상되고 로드 시간이 단축된다는 결론을 내릴 수 있습니다.
