# 감사 이벤트 검색 및 내보내기

감사 애플리케이션에는 간단하고 고급 검색 기능이 함께 제공됩니다. 간편검색을 이용하시려면 검색어를 입력하신 후 돋보기 아이콘을 클릭하시면 됩니다. 고급 검색을 사용하려면,

1. 검색 아이콘 옆에 있는 아래쪽 화살표를 클릭합니다.

1. 검색 필드에 데이터를 입력합니다.

   | 분야                                                | 묘사                                                                                                     |
   | :------------------------------------------------ | :----------------------------------------------------------------------------------------------------- |
   | 성냥                                                | 지정한 모든 필드 또는 단일 필드와 일치하는 항목을 검색합니다.                                                                    |
   | 사용자 아이디                                           | 검색할 사용자 ID입니다. 이는 일반적으로 감사하려는 일부 작업을 수행한 사용자입니다.                                                       |
   | 그룹 ID {bdg-secondary}`Liferay DXP 7.4 U81+/GA81+` | 가능한 경우 감사 이벤트의 사이트 컨텍스트를 표시하는 그룹 ID입니다.                                                                |
   | 사용자 이름                                            | 검색할 사용자 이름입니다. 이는 사용자 ID를 검색하는 것보다 쉬운 경우가 많습니다. 특히 해당 사용자 ID가 포함된 데이터베이스에 대한 액세스 권한이 없는 경우에는 더욱 그렇습니다. |
   | 자원 아이디                                            | 이 감사 레코드에서 수정되거나 표시된 리소스의 ID입니다.                                                                       |
   | 클래스 이름                                            | 이 감사 레코드에서 수정되거나 표시된 리소스의 이름입니다. 예를 들어, 누군가가 사용자 계정을 수정했는지 확인하기 위해 사용자 리소스를 검색할 수 있습니다.                |
   | 리소스 작업                                            | 리소스에 대해 수행되는 작업 중 하나는 추가, 할당, 삭제, 가장, 로그인, login_failure, 로그아웃, 할당 취소 또는 업데이트입니다. |
   | 개회 식별                                             | 검색할 세션 ID입니다. 이를 사용하여 웹 서버 로그의 세션 ID를 Liferay DXP의 활동과 연관시킬 수 있습니다.                                    |
   | 클라이언트 IP                                          | 감사하려는 활동을 수행한 클라이언트의 IP 주소입니다.                                                                         |
   | 클라이언트 호스트                                         | 감사하려는 활동을 수행한 클라이언트의 호스트 이름입니다.                                                                        |
   | 서버 이름                                             | 활동이 발생한 서버의 이름입니다. 클러스터를 사용하는 경우 클러스터의 각 구성원을 개별적으로 쿼리할 수 있습니다.                                        |
   | 서버 항구                                             | 활동이 발생한 서버 포트입니다. 동일한 머신에서 여러 VM의 수직 클러스터를 실행하는 경우 이 정보가 필요합니다.                                        |
   | 시작 날짜                                             | 검색하려는 날짜 범위의 하한입니다.                                                                                    |
   | 최후 날짜                                             | 검색하려는 날짜 범위의 상한입니다.                                                                                    |

1. 돋보기 아이콘을 클릭하세요.

## 감사 이벤트 보기

특정 감사 이벤트를 찾은 후 해당 이벤트를 클릭하면 자세한 정보를 얻을 수 있습니다. 각 이벤트는 다음 필드를 강조합니다.

* 이벤트 ID
* 날짜 생성
* 그룹 ID
* 자원 아이디
* 리소스 이름
* 리소스 작업
* 사용자 아이디
* 사용자 이름
* 클라이언트 호스트
* 클라이언트 IP
* 서버 이름
* 추가 정보

`추가 정보`는 리소스 작업 중에 엔터티와 관련된 데이터를 강조 표시합니다. 예를 들어 사용자는 생성 시 기본적으로 활성 상태입니다. 사용자 비활성화 시 감사 이벤트에 대한 추가 정보는 다음과 같습니다.

```json
{"attributes":[{"newValue":"false","name":"active","oldValue":"true"}]}
```

`이름` 필드에는 변경된 속성이 표시됩니다. `oldValue` 필드에는 업데이트 전 필드 값이 표시되고, `newValue` 필드에는 업데이트 후 필드 값이 표시됩니다.

## 감사 이벤트 내보내기

{bdg-secondary}`liferay DXP 7.4 U81+/GA81+`

타사 시스템에서 감사 이벤트를 분석하기 위해 Liferay는 감사 이벤트를 CSV로 내보내기를 지원합니다. 이렇게 하려면 글로벌 메뉴를 열고 _제어판_ &rarr; _감사_로 이동합니다. 페이지 상단의 _작업_(![작업](../../images/icon-actions.png)) 아이콘을 클릭하고 _감사 이벤트 내보내기_를 선택합니다.

그러면 Liferay 인스턴스에서 모든 감사 이벤트가 내보내집니다. 하위 집합을 내보내려면 단순 검색이나 고급 검색을 사용하여 결과 집합을 얻은 다음 내보내기 기능을 사용할 수 있습니다. 예를 들어, 특정 사용자의 `LOGIN` 이벤트만 보고 싶다면,

1. 검색창에 'login'을 입력하고 돋보기 아이콘을 클릭하세요.

1. 페이지 상단에 있는 _작업_(![작업](../../images/icon-actions.png)) 아이콘을 클릭합니다.

   ![Exporting audit events for user log in to CSV.](./searching-and-exporting-audits-events/images/01.png)

1. _감사 이벤트 내보내기_를 선택합니다.

![The CSV file contains all the information associated to login events audited by Liferay.](./searching-and-exporting-audits-events/images/02.png)

'CSV 로깅 메시지 포맷터' 구성을 변경하여 CSV 파일로 내보낸 열을 편집할 수 있습니다. 자세한 내용은 [CSV 로깅 메시지 포맷터](./audit-configuration-reference.md#csv-logging-message-formatter) 참조하세요.

```{warning}
CSV 내보내기에는 사용자가 제공한 입력이 포함됩니다. 스프레드시트 프로그램에서 열 때 주의하십시오.
```

```{important}
CSV로 내보내기 기능은 현재 릴리스 기능 플래그 뒤에 있습니다. 이를 사용하려면 이 `feature.flag.LPS-177194=true`를 `portal-ext.properties`에 추가하거나 `LIFERAY_FEATURE_PERIOD_FLAG_PERIOD__UPPERCASEL__UPPERCASEP__UPPERCASES__MINUS__NUMBER1__NUMBER7__NUMBER7__NUMBER1__NUMBER9__NUMBER4_=true`를 환경 변수로 구성에 추가하세요.
```
