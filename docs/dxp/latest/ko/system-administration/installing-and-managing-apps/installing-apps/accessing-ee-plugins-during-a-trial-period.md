# 평가판 기간 동안 EE 플러그인 액세스

평가 기간 동안 DXP를 평가하는 경우 특정 EE 플러그인에 액세스할 수 있습니다.

단계는 다음과 같습니다.

1. 제어판에서 **앱** &rarr; **스토어** 로 이동합니다.

1. liferay.com 계정으로 로그인합니다.

    ```{important}
    앱을 구매하려면 [liferay.com](https://www.liferay.com) 계정이 있어야 하며 Marketplace 사용 약관에 동의해야 합니다.
    ```

1. 스토어에 로그인한 후 **구매** 링크를 클릭한 다음 **EE** 탭을 클릭합니다.

EE 플러그인 목록이 나타납니다. 평가 기간 동안 이러한 DXP EE 플러그인에 액세스하고 사용할 수 있습니다.

```{note}
평가 기간 동안 EE 플러그인에 대한 플러그인 관리는 *제어판*에서만 사용할 수 있습니다. Marketplace 웹사이트의 *Purchased* -> *EE* 탭은 엔터프라이즈 구독을 구매한 후에만 사용할 수 있습니다.
```

```{important}
EE 플러그인 소스 코드는 공식 엔터프라이즈 구독을 얻은 후에만 사용할 수 있습니다.
```
