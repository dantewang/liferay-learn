# 감사 프레임워크

Liferay의 감사 프레임워크는 인스턴스의 사용자, 사용자 그룹, 조직, 역할, 다단계 인증 및 개체(정의, 필드, 작업, 레이아웃, 관계, 유효성 검사 및 보기)와 관련된 활동을 보여줍니다. **글로벌 메뉴**(![Global Menu](../images/icon-applications-menu.png))를 열고 **제어판** &rarr; **감사** 로 이동하여 액세스할 수 있습니다. 애플리케이션에 캡처된 이벤트의 검색 가능한 목록이 표시됩니다. 이벤트를 클릭하여 자세한 정보를 보거나, 목록을 찾아보거나 검색하거나, 감사 이벤트를 CSV로 내보낼 수 있습니다. 감사 프레임워크를 사용하면 Liferay 인스턴스에서 어떤 변경이 누가 수행했는지 빠르게 확인할 수 있습니다. 어떤 그룹에 권한 부여를 위임한 경우 이 기능을 사용할 가능성이 높습니다.
