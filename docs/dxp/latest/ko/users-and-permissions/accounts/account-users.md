# 계정 사용자

{bdg-secondary}`라이프레이 7.4 U55+/GA55+`

계정을 만든 후에는 기존 사용자를 수동으로 연결할 수 있습니다. 또는 계정 사용자 애플리케이션을 사용하여 사용자를 생성하고 계정과 동시에 연결할 수 있습니다. 여기에서 계정과 연결된 모든 사용자를 관리할 수도 있습니다.

```{note}
Liferay 7.4 U20/GA20 및 이전 버전의 경우 계정 사용자 애플리케이션을 통해서만 계정 사용자를 생성할 수 있습니다. 사용자 및 조직 애플리케이션에서 생성된 사용자를 계정과 연결할 수 없습니다.
```

## 계정 사용자 추가

1. **글로벌 메뉴**(![Global Menu](../../images/icon-applications-menu.png))를 열고 **제어판** &rarr; **계정 사용자** 로 이동합니다.

1. **추가**(![Add Button](../../images/icon-add.png))을 클릭합니다.

1. 사용자와 연결할 **계정** 을 선택합니다.

1. 이 새 사용자에 대해 필요한 정보를 입력합니다.

   ![계정 사용자 정보를 입력합니다.](./account-users/images/01.png)

1. **저장** 을 클릭합니다.

### 계정 사용자의 비밀번호 설정

사용자 계정을 만들 때 Liferay는 암호를 생성하고 구성된 [메일 서버](../../installation-and-upgrades/setting-up-liferay/configuring-mail.md)을 사용하여 이메일을 통해 사용자에게 보냅니다.

사용자 비밀번호를 수동으로 설정하려면,

1. 계정 사용자에서 원하는 사용자에 대해 **작업** 버튼(![Actions Button](../../images/icon-actions.png))을 클릭하고 **편집** 을 선택합니다.

1. 일반 탭에서 **비밀번호** 을 클릭하십시오.

1. 비밀번호를 입력합니다.

1. 암호 재설정이 필요한지 여부를 결정합니다.

1. **저장** 을 클릭합니다.

### 기존 사용자를 비즈니스 계정에 연결

1. **글로벌 메뉴**(![Global Menu](../../images/icon-applications-menu.png))을 열고 **제어판** &rarr; **계정** 로 이동합니다.

1. 편집할 계정을 선택하고 **작업** 버튼(![Actions Button](../../images/icon-actions.png)) &rarr; **사용자 관리** 를 클릭합니다.

1. **추가** 을 클릭하고 계정에 할당할 사용자를 선택합니다.

### 기존 사용자를 개인 계정에 연결

1. **글로벌 메뉴**(![Global Menu](../../images/icon-applications-menu.png))을 열고 **제어판** &rarr; **계정** 로 이동합니다.

1. 편집할 계정을 선택하고 **작업** 버튼(![Actions Button](../../images/icon-actions.png)) &rarr; **편집** 를 클릭합니다.

1. 사용자 섹션에서 **할당** 을 클릭하고 사용자를 선택합니다. **선택** 을 클릭합니다.

## 이메일로 새 사용자 초대

{bdg-secondary}`사용 가능 7.4 U55+/GA55+`

이메일로 사용자를 계정에 초대할 수 있습니다. Liferay는 새 사용자를 생성하고 계정과 연결하기 위해 지정한 사람에게 자동 초대 이메일을 보냅니다.

1. Liferay 인스턴스에 대해 이미 [메일](../../installation-and-upgrades/setting-up-liferay/configuring-mail.md) 을 구성했는지 확인하십시오.

1. **글로벌 메뉴**(![Global Menu](../../images/icon-applications-menu.png))을 열고 **제어판** &rarr; **계정** 로 이동합니다.

1. 사용자를 초대할 **계정** 을 선택하고 **사용자** 탭으로 이동합니다.

1. **추가**(![Add Button](../../images/icon-add.png))를 클릭하고 **사용자** 를 선택합니다.

1. 초대할 사람의 이메일 주소를 입력하고 Enter 키를 누릅니다. 원하는 경우 여러 주소를 추가할 수 있습니다.

1. (선택 사항) 사용자에게 할당할 계정 역할(예: 계정 관리자, 구매자, 주문 관리자 등)을 입력하고 선택합니다.

   ```{tip}
   Liferay는 초대 항목의 모든 이메일에 선택한 역할을 적용합니다. 원하는 경우 *항목 추가*를 클릭하여 이메일에 다른 역할을 할당하기 위한 추가 초대 항목을 만들 수 있습니다.
   ```

1. **초대** 을 클릭합니다.

   ![새 사용자의 이메일 주소와 역할을 입력합니다.](./account-users/images/02.png)

Liferay는 포함된 이메일 주소로 초대장을 보냅니다. 각 수신자는 이메일에서 계정 만들기를 클릭하여 Liferay 인스턴스에 새 사용자를 등록할 수 있습니다. 생성 시 사용자는 계정과 연결되고 계정 역할이 할당됩니다.

![그 사람은 새 사용자 계정을 만들라는 이메일 초대를 받습니다.](./account-users/images/03.png)

## 계정 사용자 편집

1. 계정 사용자에서 사용자를 선택하고 **작업** 버튼(![Actions Button](../../images/icon-actions.png))을 클릭합니다.

1. **편집** 을 클릭합니다.

다음 옵션을 사용할 수 있습니다.

| 일반 탭        | 묘사                      |
|:----------- |:----------------------- |
| Information | 이용자의 개인정보를 수정하거나 변경합니다. |
| 계정          | 사용자의 연결된 계정을 보거나 수정합니다. |
| 암호          | 사용자의 암호를 추가하거나 변경합니다.   |

| 연락처 탭 | 묘사                    |
|:----- |:--------------------- |
| 주소    | 사용자의 주소를 추가하거나 변경합니다. |
| 교신 정보 | 사용자의 연락처 정보 추가 또는 변경  |

| 기본 설정 탭    | 묘사                        |
|:---------- |:------------------------- |
| 경고 및 알림 전달 | 사용자의 알림 설정을 변경합니다.        |
| 전시 조정      | 사용자의 시간대 및 인사말 설정을 변경합니다. |

## 계정 사용자 비활성화 및 삭제

계정 사용자 삭제는 사용자를 비활성화한 다음 삭제하는 2단계 프로세스입니다. 이는 사용자를 비활성화하고 삭제하기 전에 해당 데이터를 검토할 시간이 여전히 있기 때문입니다(예: 법적 요구 사항 [](../managing-user-data.md)).

### 계정 사용자 비활성화

비활성화하면 계정 사용자가 로그인할 수 없지만 사용자의 데이터와 정보는 보존됩니다.

1. 계정 사용자에서 사용자를 선택하고 **작업** 버튼(![Actions Button](../../images/icon-actions.png))을 클릭합니다.

1. **비활성화** 을 클릭하고 **확인** 을 클릭하여 변경 사항을 확인합니다.

이제 계정 사용자가 비활성화되었습니다.

비활성화된 사용자를 보려면 **필터 및 주문** 드롭다운 메뉴를 클릭하고 상태별 필터에서 **비활성** 을 선택합니다. 사용자의 **작업** 버튼(![Actions Button](../../images/icon-actions.png))을 클릭하고 **활성화** 를 클릭하여 다시 활성화할 수 있습니다.

### 계정 사용자 삭제

1. 계정 사용자에서 **필터 및 주문** 드롭다운 메뉴를 클릭하고 상태별 필터링에서 **비활성** 을 클릭합니다.

1. 비활성 사용자의 **작업** 버튼(![Actions Button](../../images/icon-actions.png))을 클릭하고 선택 **삭제**

   ![비활성 사용자를 선택하고 삭제를 클릭합니다.](./account-users/images/04.png)

1. 팝업 양식에서 **확인** 을 클릭하여 결정을 확인합니다. 이제 계정 사용자가 삭제되었습니다.

## 계정 사용자 사칭

계정 사용자를 가장하여 그들이 보는 대로 시스템을 볼 수 있습니다. 이는 사용자가 제한된 데이터에 액세스할 수 없는지 확인하는 것과 같이 관리자가 볼 수 없는 권한 문제를 진단하는 데 사용할 수 있습니다. 이 권한이 있는 사용자만 다른 사용자를 가장할 수 있습니다.

1. 계정 사용자에서 사용자의 **작업** 버튼(![Actions Button](../../images/icon-actions.png))을 클릭합니다.

1. **가장 사용자** 을 선택합니다.

선택한 계정 사용자로 로그인한 새 창이 열립니다.
