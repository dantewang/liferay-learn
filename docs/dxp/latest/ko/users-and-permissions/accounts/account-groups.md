# 계정 그룹

{bdg-secondary}`가능 7.4+`

계정 그룹을 사용하여 필요에 따라 계정을 함께 그룹화합니다. 예를 들어 지역(미주, 아시아 태평양, 유럽 등)별로 계정을 그룹화합니다.

## 계정 그룹 만들기

1. 전역 메뉴(![Global Menu](../../images/icon-applications-menu.png))를 엽니다. **제어판** &rarr; **계정 그룹** 로 이동합니다.

1. **_** 추가 아이콘(![Add icon](../../images/icon-add.png))을 클릭하여 새 계정 그룹을 생성합니다.

1. 다음 페이지에서 계정 그룹에 이름과 선택적 설명을 제공합니다. **저장** 을 클릭합니다.

1. 페이지 상단의 **계정** 탭을 클릭합니다. **_** 추가 아이콘(![Add icon](../../images/icon-add.png))을 클릭하여 그룹에 계정을 할당합니다.

   ![추가 아이콘을 클릭하고 그룹에 할당할 계정을 선택합니다.](./account-groups/images/01.png)

   포함하려는 계정 옆의 체크박스를 선택합니다. 특정 계정을 찾으려면 검색창을 사용하세요. **할당** 을 클릭하여 변경 사항을 저장합니다.

## 계정 그룹 편집

1. 계정 그룹에서 수정하려는 계정 그룹의 **옵션** 아이콘(![Options icon](../../images/icon-actions.png))을 클릭합니다. **편집** 을 클릭하면 상세 페이지를 볼 수 있습니다.

1. 세부 정보 탭에서 이름 또는 설명을 보거나 변경합니다.

1. **계정** 탭을 클릭하여 그룹을 보거나 변경합니다. 그룹에 더 많은 계정을 할당하려면 **_** 추가 아이콘(![Add icon](../../images/icon-add.png))을 클릭합니다. 그룹에서 제거하려면 특정 계정의 **제거** 아이콘(![Remove icon](../../images/icon-delete.png))을 클릭합니다. **확인** 을 클릭하여 계정 제거를 확인합니다.

계정 그룹을 삭제하려면 삭제할 계정 그룹의 **옵션** 아이콘(![Options icon](../../images/icon-actions.png))을 클릭하세요. **확인** 을 클릭하여 결정을 확인합니다.
