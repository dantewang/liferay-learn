# 계정 주소

{bdg-secondary}`가능 7.4+`

계정 주소는 비즈니스 계정 또는 개인 계정과 연결된 청구지 또는 배송 주소입니다.

## 계정 주소 추가

1. 전역 메뉴(![Global Menu](../../images/icon-applications-menu.png))를 엽니다. *제어판* &rarr; *계정*로 이동합니다.

1. 이름을 클릭하여 계정을 선택하거나 계정에서 _옵션_ (![Options icon](../../images/icon-actions.png))을 클릭하고 _편집_을 클릭합니다.

1. _주소_ 탭을 클릭합니다. __ 추가 아이콘(![Add icon](../../images/icon-add.png))을 클릭하여 새 주소를 추가합니다.

1. 주소 추가 양식을 작성합니다. 주소가 청구지 주소인지, 배송지 주소인지 또는 둘 다인지 선택합니다. 이름, 유형, 국가, 거리 1, 도시 및 우편 번호는 필수 필드입니다.

   ![추가 주소를 입력하십시오.](./account-addresses/images/01.png)

   *저장* 을 클릭하여 주소를 저장합니다.

## 계정 주소 편집

1. _주소_ 탭에서 주소의 _옵션_ 아이콘(![Options icon](../../images/icon-actions.png))을 클릭하고 _편집_을 클릭합니다. 필터 도구를 사용하여 청구 주소 또는 배송 주소로 주소를 필터링합니다. 특정 주소를 빠르게 찾으려면 검색창을 사용하세요.

   ![주소의 옵션 아이콘을 클릭하고 수정을 클릭합니다.](./account-addresses/images/02.png)

1. 변경하고 *저장*을 클릭하십시오.

## 기본 계정 주소 설정

1. _세부 정보_ 탭에서 청구 또는 배송 아래의 *기본 주소 설정* 을 클릭합니다.

   ![청구 또는 배송에서 기본 주소 설정을 클릭합니다.](./account-addresses/images/03.png)

1. 새 창이 열립니다. 목록에서 주소를 선택하고 *저장*을 클릭합니다. 이제 기본 주소가 계정에 설정되었습니다.
