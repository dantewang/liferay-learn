# 계정 관리 위젯

{bdg-secondary}`가능 7.4+`

계정 관리 위젯을 사용하면 계정 정보를 쉽게 관리하고 액세스할 수 있습니다. 모든 페이지에 위젯을 배포합니다. 이것은 제어판을 볼 수 있는 권한이 없는 사용자에게 액세스를 제공하는 유용한 방법입니다.

## 페이지에 계정 관리 위젯 추가

계정 관리 위젯을 페이지에 추가하려면

1. 앱이 필요한 사이트 페이지로 이동합니다.

1. 페이지 오른쪽 상단에 있는 수정 아이콘(![Edit icon](../../images/icon-edit-pencil.png))을 클릭합니다.

1. 위젯의 계정 섹션에서 위젯을 찾습니다. 위젯을 페이지로 드래그합니다.

   ![위젯을 찾아 페이지로 끕니다.](./account-management-widget/images/01.png)

1. 이제 계정 목록이 위젯에 표시됩니다.

## 계정 관리 위젯 사용

계정 관리 위젯에는 다양한 계정 목록이 표시됩니다. 표시되는 내용은 현재 사용자가 액세스할 수 있는 계정에 따라 다릅니다. *필터 및 주문* 을 클릭하고 목록을 필터링하거나 검색 창을 사용하여 특정 계정을 빠르게 찾으십시오.

![계정 관리 위젯은 다른 계정을 표시합니다.](./account-management-widget/images/02.png)

위젯에는 다음 정보가 표시됩니다.

**이름:** 계정 이름

**ID:** 계정의 ID 번호

**조직:** 계정이 할당된 모든 조직

**유형:** 계정 유형(예: 비즈니스, 게스트, 사람)

**상태:** 계정이 활성 또는 비활성인지 여부

**선택됨:** 확인 표시는 현재 사이트에 대해 선택된 계정을 나타냅니다.

계정 사용자가 여러 계정에 액세스할 수 있는 경우 선택됨 확인 표시는 사용자가 사이트에서 현재 보고 있거나 관리하고 있는 계정을 나타냅니다. 이를 통해 가격 또는 주문 정보와 같은 각 계정에 대한 다양한 설정을 관리할 수 있습니다.

편집 또는 관리하려는 계정의 옵션 아이콘(![Options icon](../../images/icon-actions.png))을 클릭합니다. 다음 선택 사항을 사용할 수 있습니다.

**편집:** 계정을 변경합니다.

**사용자 관리:** 계정 사용자 관리

**조직 관리:** 계정이 할당된 조직을 관리합니다.

**비활성화:** 계정 비활성화

**삭제:** 계정 삭제
