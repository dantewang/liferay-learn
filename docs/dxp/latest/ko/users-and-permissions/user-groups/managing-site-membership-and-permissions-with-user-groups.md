# 사용자 그룹으로 사이트 구성원 및 권한 관리

사용자 그룹을 사용하여 사이트 구성원, 역할 및 권한을 관리할 수 있습니다.

## 사용자 그룹으로 사이트 구성원 관리

사용자 그룹의 주요 사용 사례 중 하나는 사이트 멤버십을 관리하는 것입니다. 사용자 그룹을 사이트에 할당하면 해당 그룹의 사용자는 즉시 해당 사이트의 구성원이 됩니다.

![멤버십의 사용자 그룹 탭에는 사이트에 현재 할당된 사용자 그룹이 표시됩니다.](./managing-site-membership-and-permissions-with-user-groups/images/02.png)

사이트에 사용자 그룹을 지정하려면 다음 단계를 따르십시오.

1. 메뉴(![Menu](../../images/icon-menu.png))를 열고 작업할 사이트를 선택한 다음 사이트 관리 메뉴를 엽니다.

1. 사이트 관리 메뉴에서 *People* &rarr; *Memberships*를 선택합니다.  그러면 사이트 구성원 화면이 열립니다.

   ![사람 메뉴에서 멤버십을 선택합니다.](./managing-site-membership-and-permissions-with-user-groups/images/01.png)

1. 멤버십에서 *사용자 그룹* 탭을 선택합니다. 이 탭은 사이트에 현재 할당된 모든 사용자 그룹을 표시합니다.

1. *추가* 버튼(![Add](../../images/icon-add.png))을 클릭하고 사이트에 할당할 사용자 그룹을 선택한 다음 *완료*를 클릭합니다.  이제 선택한 사용자 그룹이 사용자 그룹 탭에 나타납니다.

## 사용자 그룹으로 사이트 권한 관리

[사용자 그룹 만들기 및 관리](./creating-and-managing-user-groups.md) 에서는 포털에 대한 권한이 있어야 하는 사용자의 사용자 그룹에 일반 역할을 할당하는 방법을 설명합니다. 사용자 그룹은 종종 그룹의 사용자가 한 사이트에서만 활동할 수 있도록 생성됩니다. 일반 역할과 마찬가지로 해당 작업에 필요한 권한을 *사이트 역할*에 할당한 다음 해당 역할을 사용자 그룹에 할당하는 것으로 시작합니다.

사용자 그룹의 사용자에게 사이트 범위 권한을 부여하려면

1. [사용자 그룹 만들기](./creating-and-managing-user-groups.md#creating-a-user-group) .
1. 사용자 그룹을 사이트에 할당합니다( [위](#managing-site-membership-with-user-groups) 참조 ).
1. 사이트 역할을 생성하고 해당 권한을 정의합니다( [역할 및 권한](../roles-and-permissions.md)참조).
1. 사용자 그룹에 역할을 할당합니다.

이미 사이트의 구성원인 기존 사용자 그룹에 기존 사이트 역할을 할당하려면

1. 메뉴(![Menu](../../images/icon-menu.png))를 열고 작업할 사이트를 선택한 다음 사이트 관리 메뉴를 엽니다.

1. 사이트 관리 메뉴에서 *People* &rarr; *Memberships*를 선택합니다.

1. Memberships에서 *User Groups* 탭을 선택하여 사이트에 현재 할당된 사용자 그룹을 확인합니다.

1. 역할에 할당하려는 사용자 그룹에 대해 *동작* 버튼(![Actions](../../images/icon-actions.png))을 클릭하고 *역할 할당*을 선택합니다.

   ![사용자 그룹에 대한 역할 할당을 선택합니다.](./managing-site-membership-and-permissions-with-user-groups/images/03.png)

1. 역할 할당 대화 상자의 목록에서 역할을 선택하고 *완료*을 클릭합니다.
