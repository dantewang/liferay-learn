# 사용자 그룹 생성 및 관리

사용자 그룹은 특정 목적을 위해 생성된 사용자 목록입니다. 사용자 그룹은 [조직](../organizations/understanding-organizations.md)의 계층적 경계를 넘어 생성될 수 있습니다. 예를 들어, 관리자는 모든 구성원을 여러 사이트에 추가하기 위해 교사 사용자 그룹을 생성하고 이들 모두를 [일반 역할](../roles-and-permissions.md)에 할당하고 사용자 그룹의 모든 교사에 대한 공통 프로필 페이지 세트를 생성할 수 있습니다.

![사용자 그룹을 추가하려면 약간의 정보만 필요합니다.](./creating-and-managing-user-groups/images/01.png)

 여기에서 사용자 그룹으로 작업하여 가장 일반적인 사용 사례를 제공하는 방법을 배웁니다.

| 사용 사례          | 묘사                                                                                                                                                                                        |
|:-------------- |:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 사이트 회원 관리      | 사용자 그룹의 모든 사용자에게 사이트 멤버십을 부여합니다. 이전 예를 사용하여 Teachers 사용자 그룹을 Sites **University Employees** 및 **Students and Teachers Collaboration** 의 구성원으로 추가할 수 있습니다. 해당 사용자 그룹의 모든 사용자는 해당 사이트의 구성원이 됩니다. |
| 사용자의 개인 페이지 관리 | 사용자 그룹에 미리 정의된 페이지를 제공합니다. 예를 들어 교사 사용자 그룹을 생성하여 모든 교사의 개인 사이트에 있는 홈 페이지가 동일한 레이아웃과 애플리케이션을 갖도록 할 수 있습니다.                                                                                 |
| 권한 수집          | 조직을 공유하지 않는 사용자 그룹에 역할 및 권한을 할당합니다. 예를 들어, 대학 포털에서 부서(조직)와 독립적으로 모든 교사를 그룹화하기 위해 사용자 그룹을 만들 수 있습니다. 이렇게 하면 모든 교사에게 한 번에 하나 이상의 역할을 더 쉽게 할당할 수 있습니다.                                       |

## 사용자 그룹 만들기

사용자 그룹을 만들려면

1. 메뉴(![Menu](../../images/icon-menu.png))를 열고 **제어판** &rarr; **사용자** &rarr; **사용자 그룹** 을 선택합니다. 모든 기존 사용자 그룹이 여기 테이블에 나타납니다.

1. **추가** 버튼(![Add](../../images/icon-add.png))을 클릭하여 **새 사용자 그룹** 양식을 엽니다.

1. 사용자 그룹에 이름과 설명을 지정하십시오.

1. 사용자 그룹 구성원을 위한 내 프로필 및/또는 내 대시보드 페이지를 생성하려면 **내 프로필** 및 **내 대시보드** 선택기 메뉴에서 사이트 템플릿을 선택합니다.

1. **저장** 을 클릭합니다. 새 사용자 그룹이 테이블에 나타납니다.

새 사용자 그룹에는 추가할 때까지 사용자가 없습니다.

![방금 생성한 사용자 그룹이 이제 테이블에 나타납니다.](./creating-and-managing-user-groups/images/02.png)

## 사용자 그룹에 구성원 할당

사용자 그룹에 기존 사용자를 추가하려면

1. 사용자 그룹 애플리케이션을 엽니다(**제어판** &rarr; **사용자** &rarr; **사용자 그룹**). 사용 가능한 사용자 그룹이 나타납니다.

1. **작업**(![Actions](../../images/icon-actions.png)) &rarr; **사용자 그룹에 구성원 할당** 를 클릭합니다. 그룹의 기존 사용자가 테이블에 나타납니다.

1. **추가** 버튼(![Add](../../images/icon-add.png))을 클릭하면 선택할 수 있는 사용자 목록이 표시됩니다.

1. 목록에서 사용자를 선택한 다음 **추가** 을 클릭합니다. 이렇게 하면 선택한 사용자가 그룹에 추가되고 그룹의 사용자 목록이 표시됩니다.

![사용자 그룹에 추가할 사용자를 선택합니다.](./creating-and-managing-user-groups/images/03.png)

## 사용자 그룹 권한 구성

사용자 그룹 사용자는 시스템 전체 또는 특정 사이트에서 특정 작업을 수행해야 할 수 있습니다. 범위에 관계없이 작업에 필요한 권한을 역할에 할당한 다음 해당 역할을 사용자 그룹에 할당하여 액세스 권한을 부여합니다. 권한이 할당된 역할이 부여되면 사용자 그룹의 사용자는 작업을 시작할 수 있습니다.

포털 전체 범위에서 사용자 그룹 권한을 부여하려면 일반 역할이 필요합니다.

1. 일반 역할을 만들고 해당 권한을 정의합니다(자세한 내용은 [역할 및 권한](https://help.liferay.com/hc/ko/articles/360028819032-Roles-and-Permissions) 참조).
1. 역할이 생성되면 **제어판** &rarr; **사용자** &rarr; **역할** 에서 역할 제목을 클릭하여 사용자 그룹을 할당합니다.
1. **담당자** 탭을 엽니다.
1. **사용자 그룹** 패널을 엽니다.
1. 추가 아이콘(![Add](../../images/icon-add.png))을 클릭합니다.
1. 사용자 그룹을 선택하고 **추가** 을 클릭합니다.

![역할에 구성원을 추가할 때 사용자 그룹을 선택할 수 있습니다.](./creating-and-managing-user-groups/images/07.png)

사용자 그룹 및 사이트 범위 관리에 대한 자세한 내용은 [사용자 그룹으로 사이트 구성원 및 권한 관리](./managing-site-membership-and-permissions-with-user-groups.md)을 참조하십시오.

## 사용자 그룹 편집

**제어판** &rarr; **사용자** &rarr; **사용자 그룹** 에서 기존 사용자 그룹을 보고 편집할 수 있습니다.

![사용자 그룹 인터페이스를 사용하면 사용자 그룹을 쉽게 만들고 찾고 수정할 수 있습니다.](./creating-and-managing-user-groups/images/02.png)

사용자 그룹에서 사용자를 추가/제거하려면 다음 단계를 따르십시오.

1. 사용자 그룹의 현재 사용자 목록을 보려면 사용자 그룹의 이름 또는 설명을 클릭하십시오.

1. 사용자 그룹에서 사용자를 제거하려면 해당 사용자 옆에 있는 `-` 버튼을 클릭하십시오. 여러 사용자를 한 번에 제거하려면 각 사용자의 확인란을 선택한 다음 사용자 목록 위의 관리 표시줄에서 `-` 아이콘을 클릭합니다.

1. 사용자 그룹에 사용자를 추가하려면 **추가** 버튼(![Add](../../images/icon-add.png))을 클릭하고 사용자를 선택한 다음 **추가** 를 클릭합니다.

![사용자 목록을 통해 사용자 그룹의 구성원을 관리할 수 있습니다.](./creating-and-managing-user-groups/images/05.png)

다른 관리 옵션은 각 사용자 그룹의 작업 메뉴(![Actions](../../images/icon-actions.png))에서 사용할 수 있습니다.

**편집:** 사용자 그룹의 이름 또는 설명을 수정하거나 [사용자 그룹의 사이트](./user-group-sites.md)에 사용할 사이트 템플릿을 선택합니다.

**권한:** 사용자 그룹을 보고 관리할 수 있는 권한을 할당합니다.

**사용자 그룹 페이지 권한:** 사용자 그룹의 사이트 페이지를 관리하기 위한 권한을 할당합니다.

**페이지 관리:** 사용자 그룹의 사이트 페이지를 수동으로 관리합니다. 자세한 내용은 [사용자 그룹 사이트](./user-group-sites.md#creating-user-group-sites-manually) 을 참조하십시오.

**구성원 지정:** 사용자 그룹에서 사용자를 추가/제거합니다(위에 설명됨).

**삭제:** 사용자 그룹을 제거합니다. 사용자가 포함된 사용자 그룹은 삭제할 수 없습니다. 먼저 그룹에서 사용자를 제거해야 합니다.

사용자 그룹에 페이지가 있는 경우 옵션 **프로필 페이지로 이동** 및 **대시보드 페이지로 이동** 도 사용자 그룹의 작업 메뉴에 나타납니다. 이러한 링크 중 하나를 클릭하면 새 브라우저 창에서 해당 사이트가 열립니다. 자세한 내용은 [사용자 그룹 사이트](./user-group-sites.md) 을 참조하십시오.

![사용자 그룹의 작업 메뉴는 그룹 관리에 필요한 모든 기능을 제공합니다.](./creating-and-managing-user-groups/images/06.png)
