# 게시판 권한 참조

이 문서는 _Message Boards_ 위젯의 권한을 문서화하고 사용자 및 역할에 권한을 할당할 때 참조 역할을 합니다. 일반적인 DXP 역할 및 권한에 대해 자세히 알아보려면 [역할 및 권한](https://help.liferay.com/hc/articles/360017895212-Roles-and-Permissions) 을 참조하십시오.

## 홈 분류 권한

_게시판_ 권한 화면은 _게시판_ 기능에 대한 접근 권한을 부여하고 철회하는 화면입니다.

1. 호스트 사이트(예: Liferay DXP의 게스트 사이트)에 대한 _사이트 관리_ 로 이동합니다.
1. 클릭 _콘텐츠 & 데이터_ &rarr; _게시판_.
1. *옵션* 아이콘(![Options](./message-boards-permissions-reference/images/01.png))을 클릭한 다음 *홈 범주 권한을*합니다.

    ![사이트 역할에 권한 할당](./message-boards-permissions-reference/images/03.png)

권한을 통해 특정 역할을 가진 사용자는 다음 작업을 수행할 수 있습니다.

| 권한 이름            | 묘사                                |
|:---------------- |:--------------------------------- |
| **권한:**          | 권한을 보고 수정할 수 있는 권한을 부여합니다.        |
| **파일 추가:**       | 메시지에 파일을 첨부할 수 있는 권한을 부여합니다.      |
| **사용자 차단:**      | 사용자가 게시판에 참여하는 것을 금지하는 기능을 부여합니다. |
| **분류 등록:**       | 게시판에 새 카테고리를 추가합니다.               |
| **회신:**          | 기존 메시지에 응답합니다.                    |
| **댓글 잠금:**       | 스레드 메시지에 대한 추가 추가 또는 수정을 중지합니다.   |
| **구독:**          | 새 게시물과 수정된 게시물에 대한 알림을 받습니다.      |
| **보기:**          | 메시지 스레드의 모든 내용을 봅니다.              |
| **메세지 추가:**      | 새 스레드를 게시합니다.                     |
| **댓글 이동:**       | 스레드를 다른 범주 또는 하위 범주로 이동합니다.       |
| **댓글 우선 순위 변경:** | 스레드의 우선 순위를 수정합니다.                |

## 위젯 권한

메시지 보드 위젯을 페이지에 추가한 후 관리자는 해당 위젯 인스턴스의 일반 권한에 액세스할 수 있습니다. 그렇게 하려면 위젯의 *옵션* 메뉴(![Options](./message-boards-permissions-reference/images/02.png))를 선택하고 *권한*을 선택합니다.

![위젯 권한 할당](./message-boards-permissions-reference/images/04.png)

| 권한 이름        | 묘사                                |
|:------------ |:--------------------------------- |
| **권한:**      | 위젯의 권한을 보고 수정할 수 있는 권한을 부여합니다.    |
| **환경 설정:**   | 위젯의 기본 설정을 보고 수정할 수 있는 권한을 부여합니다. |
| **구성:**      | 위젯 구성을 보고 수정할 수 있는 권한을 부여합니다.     |
| **보기:**      | 위젯을 볼 수 있는 권한을 부여합니다.             |
| **페이지에 추가:** | 사이트 페이지에 위젯을 추가하는 기능을 부여합니다.      |

## 일반 카테고리 권한

이 권한 세트는 게스트 및 사이트 구성원이 범주에 대해 수행할 수 있는 변경 사항을 결정합니다.

![일반 카테고리 권한](./message-boards-permissions-reference/images/05.png)

| 권한 이름            | 묘사                              |
|:---------------- |:------------------------------- |
| **삭제:**          | 범주를 제거하십시오.                     |
| **권한:**          | 권한을 보고 수정합니다.                   |
| **파일 추가:**       | 메시지에 파일을 첨부하십시오.                |
| **회신:**          | 기존 메시지에 응답합니다.                  |
| **댓글 잠금:**       | 스레드 메시지에 대한 추가 추가 또는 수정을 중지합니다. |
| **수정:**          | 카테고리를 수정합니다.                    |
| **구독:**          | 새 게시물과 수정된 게시물에 대한 알림을 받습니다.    |
| **보기:**          | 카테고리를 봅니다.                      |
| **메세지 추가:**      | 새 스레드를 게시합니다.                   |
| **댓글 이동:**       | 스레드를 다른 범주 또는 하위 범주로 이동합니다.     |
| **하위 분류 추가:**    | 이 범주 내에 새 범주를 추가합니다.            |
| **댓글 우선 순위 변경:** | 스레드의 우선 순위를 수정합니다.              |

## 스레드 권한

 이 권한 세트는 사용자가 스레드에 대해 수행할 수 있는 변경 사항을 결정합니다.

| 권한 이름   | 묘사                   |
|:------- |:-------------------- |
| **삭제:** | 스레드를 제거하십시오.         |
| **권한:** | 스레드 권한을 부여/취소합니다.    |
| **수정:** | 스레드를 편집합니다.          |
| **구독:** | 스레드 활동에 대한 알림을 받습니다. |
| **보기:** | 스레드를 봅니다.            |
