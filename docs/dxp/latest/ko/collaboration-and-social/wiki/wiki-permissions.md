# Wiki 권한 참조

이 문서는 _Wiki_ 위젯의 권한을 문서화하고 사용자 및 역할에 권한을 할당할 때 참조 역할을 합니다. 일반적인 DXP 역할 및 권한에 대해 자세히 알아보려면 [역할 및 권한](https://help.liferay.com/hc/articles/360017895212-Roles-and-Permissions)참조하십시오.

## Wiki 노드 권한

1. _사이트 관리_ &rarr; _콘텐츠 & 데이터_ &rarr; _위키_로 이동합니다.
1. 노드 옆에 있는 _옵션_ (![Options](../../images/icon-options.png))(예: _새 노드_)을 클릭한 다음 _권한_클릭하십시오.

    ![Wiki 노드 권한 보기](./wiki-permissions/images/06.png)

사용 가능한 _위키 노드_ 권한은 다음과 같습니다.

![Wiki 노드 권한 보기](./wiki-permissions/images/05.png)

| 허가         | 묘사                                                                                                                                                                                                    |
|:---------- |:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 페이지 추가     | 사용자에게 위키 페이지를 노드에 추가할 수 있는 권한을 부여합니다.                                                                                                                                                                 |
| 가져오기       | 사용자에게 위키 페이지를 `.lar` 파일로 가져올 수 있는 기능을 부여합니다. [Wiki 내보내기 및 가져오기](../../site-building/creating-pages/page-fragments-and-widgets/using-widgets/configuring-widgets/exporting-importing-widget-data.md)참조 |
| 삭제         | 사용자에게 _위키 노드_을 삭제할 수 있는 권한을 부여합니다.                                                                                                                                                                    |
| 권한         | 사용자에게 위키 노드의 권한을 볼 수 있는 권한을 부여합니다.                                                                                                                                                                    |
| 수정         | 사용자에게 _Wiki 노드_을 수정할 수 있는 권한을 부여합니다.                                                                                                                                                                  |
| 부착을 추가하십시오 | 사용자에게 첨부 파일을 추가할 수 있는 권한을 부여합니다.                                                                                                                                                                      |
| 구독         | 사용자에게 노드를 구독할 수 있는 권한을 부여합니다.                                                                                                                                                                         |
| 보기         | 사용자가 _위키 노드_을 볼 수 있습니다.                                                                                                                                                                               |

## Wiki 페이지 권한

Wiki 페이지의 권한을 보려면:

1. Wiki 페이지에서 _세부 정보_클릭합니다.
1. _고급 작업_ 섹션에서 _권한_ 클릭합니다.

![Wiki 페이지 권한 보기](./wiki-permissions/images/04.png)

| 허가    | 묘사                                        |
|:----- |:----------------------------------------- |
| 대화 수정 | 사용자에게 댓글을 _위키 페이지_로 업데이트할 수 있는 권한을 부여합니다. |
| 삭제    | 사용자에게 _위키 페이지_을 삭제할 수 있는 권한을 부여합니다.       |
| 권한    | 사용자에게 페이지의 권한을 볼 수 있는 권한을 부여합니다.          |
| 대화 삭제 | 사용자에게 댓글을 삭제할 수 있는 권한을 부여합니다.             |
| 수정    | 사용자에게 _Wiki 페이지_을 수정할 수 있는 권한을 부여합니다.     |
| 보기    | 사용자가 _Wiki 페이지_을 볼 수 있습니다.                |
| 토론 추가 | 사용자가 _위키 페이지_에 댓글을 달 수 있습니다.              |

## 위키 위젯 권한

1. _Wiki_ 위젯이 배포된 사이트 페이지로 이동합니다.
1. _위키_ 위에 마우스를 놓고 클릭 _옵션_ (![Options](../../images/icon-widget-options.png))
1. _권한_클릭합니다.

![Wiki 위젯 권한 보기](./wiki-permissions/images/03.png)

| 허가           | 묘사                                           |
|:------------ |:-------------------------------------------- |
| 디스플레이 템플릿 추가 | Wiki 페이지에 표시 템플릿을 추가하는 기능을 부여합니다.            |
| 권한           | 사용자에게 _Wiki_ 위젯의 권한을 볼 수 있는 권한을 부여합니다.       |
| 환경 설정        | 사용자에게 _Wiki_ 위젯의 기본 설정을 볼 수 있는 권한을 부여합니다.    |
| 설정           | 사용자에게 _Wiki_ 위젯의 구성 메뉴에 액세스할 수 있는 권한을 부여합니다. |
| 보기           | 사용자가 _Wiki_ 위젯을 볼 수 있습니다.                    |
| 페이지에 추가      | 사이트 페이지에 _Wiki_ 위젯을 추가할 수 있는 기능을 부여합니다.      |

## 위키 권한

이러한 권한은 _사이트 관리_에서 찾을 수 있습니다.

1. _사이트 관리_ &rarr; _콘텐츠 & 데이터_ &rarr; _위키_로 이동합니다.
1. _옵션_ (![Widget Options](../../images/icon-options.png)) &rarr; _위키 권한_클릭 합니다 .

![위키 옵션 권한](./wiki-permissions/images/01.png)

사용 가능한 권한은 다음과 같습니다.

![위키 옵션 권한](./wiki-permissions/images/02.png)

| 허가    | 묘사                               |
|:----- |:-------------------------------- |
| 권한    | 사용자에게 페이지의 권한을 볼 수 있는 권한을 부여합니다. |
| 노드 추가 | 사용자에게 위키 노드를 추가할 수 있는 권한을 부여합니다. |
