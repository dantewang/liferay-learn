# 파일 업로드

문서 및 미디어를 사용하면 Liferay 인스턴스의 [파일 저장소](../../../system-administration/file-storage.md) 또는 연결된 리포지토리에 모든 유형의 파일을 업로드할 수 있습니다. 일단 업로드되면 필수 [권한](../publishing-and-sharing/managing-document-access/documents-and-media-permissions-reference.md) 을 가진 사용자는 업로드된 파일을 보고, 편집하고, 다운로드하거나 공유할 수 있습니다.

```{note}
사용자는 업로드 시 지원되는 자산에 자동으로 태그를 지정하도록 Liferay를 구성할 수 있습니다. 자세한 내용은 [자산 자동 태그 지정 구성](../../tags-and-categories/auto-tagging/configuring-asset-auto-tagging.md)을 참조하세요. 
```

```{important}
자동 바이러스 백신 검사를 활성화하여 업로드 시 파일을 검사할 수 있습니다. 자세한 내용은 [업로드된 파일에 대한 바이러스 백신 검사 활성화](../../../system-administration/file-storage/enabling-antivirus-scanning-for-uploaded-files.md)를 참조하십시오.
```

파일을 업로드하려면 먼저 사이트 또는 자산 라이브러리에서 문서 및 미디어 애플리케이션으로 이동합니다.

사이트의 문서 및 미디어에 액세스하려면 **사이트 메뉴**(![Site Menu](../../../images/icon-product-menu.png))을 열고 **콘텐츠 & 데이터** &rarr; **문서 및 미디어** 로 이동합니다.

![사이트 메뉴의 콘텐츠 & 데이터에서 문서 및 미디어를 클릭합니다.](./uploading-files/images/01.png)

자산 라이브러리의 문서 및 미디어에 액세스하려면 활성화된 라이브러리로 이동하고 **문서 및 미디어** 을 클릭합니다.

![자산 라이브러리에서 문서 및 미디어를 클릭합니다.](./uploading-files/images/02.png)

애플리케이션을 연 후 파일을 업로드하는 가장 빠른 방법은 파일을 원하는 폴더로 끌어다 놓는 것입니다. 이렇게 하면 파일이 게시되기 전에 업로드를 구성하라는 메시지 없이 즉시 파일 업로드가 시작됩니다. 파일 업로드가 완료되면 **작업** 버튼을 클릭하고 **편집** 을 선택하여 세부 정보를 편집할 수 있습니다.

![원하는 수의 파일을 원하는 폴더로 끌어다 놓습니다.](./uploading-files/images/03.png)

그러나 업로드하기 전에 파일 세부 정보를 정의하려면 다음 단계를 따르십시오.

1. **추가** 버튼(![Add Button](../../../images/icon-add.png))을 클릭하고 **파일 업로드** 또는 **여러 파일 업로드** 을 선택합니다.

   ![파일 업로드 또는 여러 파일 업로드를 선택합니다.](./uploading-files/images/04.png)

1. 파일을 지정된 드롭 영역으로 끌어다 놓거나 파일 선택기를 사용하여 파일을 찾습니다.

   **File Upload** 을 선택했다면 아래와 같이 업로드 화면이 나타납니다.

   ![업로드할 단일 파일을 선택합니다.](./uploading-files/images/05.png)

   **여러 파일 업로드** 을 선택한 경우 다음과 같은 업로드 화면이 나타납니다.

   ![업로드할 여러 파일을 끌어다 놓거나 선택합니다.](./uploading-files/images/06.png)

1. 업로드하기 전에 파일 세부 정보를 구성합니다. 자세한 내용은 [파일 업로드 구성 참조](#configuring-file-upload-reference) 을 참조하십시오.

1. 완료되면 **게시** 을 클릭하여 업로드 프로세스를 시작합니다.

## 파일 업로드 참조 구성

| 분야     | 단일 파일 업로드의 경우 | 여러 파일 업로드 | 묘사                                                                                                                                                                                         |
|:------ |:------------- |:--------- |:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 파일/파일들 | &#10004;      | &#10004;  | 업로드할 파일을 선택합니다. 기본적으로 개별 파일 업로드는 100MB를 초과할 수 없습니다.                                                                                                                                        |
| 제목     | &#10004;      |           | 업로드 후 파일에 사용되는 표시 제목을 설정합니다. 기본적으로 이 필드는 선택한 파일의 원래 이름으로 채워지지만 업로드 중이나 업로드 후에 변경할 수 있습니다.                                                                                                  |
| 파일 이름  | &#10004;      |           | 업로드된 파일의 새 이름을 설정합니다. 기본적으로 이 필드는 선택한 파일의 원래 이름으로 채워지지만 업로드 중이나 업로드 후에 변경할 수 있습니다.                                                                                                         |
| 묘사     | &#10004;      | &#10004;  | 업로드한 파일에 대한 파일 설명을 추가합니다.                                                                                                                                                                  |
| 페이지 표시 | &#10004;      | &#10004;  | 업로드된 파일에 대해 [디스플레이 페이지 템플릿](../../../site-building/displaying-content/using-display-page-templates.md) 을 선택합니다.                                                                            |
| 분류     | &#10004;      | &#10004;  | 업로드된 파일에 태그와 공개 또는 내부 카테고리를 적용합니다.                                                                                                                                                         |
| 만료 날짜  | &#10004;      | &#10004;  | 파일이 만료되거나 검토되어야 하는지 여부와 시기를 결정합니다. 기본적으로 파일은 만료되지 않거나 검토가 필요하도록 설정됩니다. 활성화된 경우 파일이 만료되거나 검토되어야 하는 날짜입니다. 자세한 내용은 [파일 만료 및 검토 날짜 사용](./using-file-expiration-and-review-dates.md) 을 참조하십시오. |
| 관련 애셋  | &#10004;      |           | 사이트 또는 연결된 자산 라이브러리에서 관련 자산 또는 파일을 선택합니다.                                                                                                                                                  |
| 권한     | &#10004;      | &#10004;  | 업로드된 파일에 대한 권한을 설정합니다. 이 설정은 어떤 사용자 역할이 파일을 볼 수 있고 파일과 관련된 다른 작업(예: 업데이트, 삭제)을 수행할 수 있는지 결정합니다.                                                                                            |
