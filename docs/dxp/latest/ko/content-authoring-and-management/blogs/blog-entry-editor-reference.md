# 블로그 항목 편집자 참조

블로그 항목 편집기는 텍스트 서식 지정, 이미지, 표 관리 및 기타 다양한 기능을 위한 상황에 맞는 편집 도구를 제공합니다. 사용자는 코드 보기로 전환하여 콘텐츠의 HTML을 직접 편집할 수 있습니다. 모든 초안 항목은 25초마다 자동으로 저장됩니다.

## 편집기의 텍스트 보기 사용

텍스트 보기는 텍스트 및 기타 유형의 콘텐츠를 입력하고 편집할 수 있는 WYSIWYG 편집기입니다.

콘텐츠 영역의 텍스트에 스타일을 지정하거나 서식을 지정하려면 해당 텍스트를 선택하세요. 다음 옵션이 포함된 도구 모음이 텍스트 위에 나타납니다.

| 기능      | 묘사                                                                                         |
| :------ | :----------------------------------------------------------------------------------------- |
| 텍스트 스타일 | 텍스트 스타일을 선택합니다. 일반은 일반적인 본문 텍스트이지만 다양한 제목 스타일, 경고 또는 오류 메시지 스타일, 코드 스타일 등 중에서 선택할 수도 있습니다. |
| 서체      | 굵게, 기울임꼴 또는 밑줄을 선택합니다.                                                                     |
| 리스트 스타일 | 번호 매기기 또는 글머리 기호 목록을 선택합니다.                                                                |
| 연결      | 선택한 텍스트를 특정 URL 또는 포털의 항목에 연결합니다.                                                          |
| Twitter | 선택한 텍스트를 트윗할 수 있는 링크를 생성합니다.                                                               |

![Formatting elements](./blog-entry-editor-reference/images/01.png)

항목의 콘텐츠 영역에 커서를 올리면 추가 아이콘(+)이 나타납니다. 추가 아이콘을 클릭하면 이미지, 비디오, 표 또는 수평선( ![Controls](../../images/icon-content-insert-controls.png) )을 삽입하기 위한 컨트롤이 표시됩니다.

| 기능       | 묘사                                                       |
| :------- | :------------------------------------------------------- |
| 심상 화랑 심상 | 블로그 항목에 이미지를 업로드합니다.                                     |
| 비디오      | 동영상 링크를 삽입하세요.                                           |
| 테이블      | 테이블을 삽입합니다.                                              |
| 선        | 간단하고 가벼운 가로선을 삽입하면 큰 블로그 항목의 콘텐츠 섹션을 구분하기 위해 콘텐츠에 표시됩니다. |

![Add elements to your blog entry](./blog-entry-editor-reference/images/02.png)

## 편집기의 코드 보기 사용

코드 보기로 전환하려면 소스 아이콘(</> )는 콘텐츠 영역에 커서를 놓으면 표시됩니다. 코드 보기의 오른쪽 상단에는 다음 버튼이 있습니다.

| 기능        | 묘사                                                     |
| :-------- | :----------------------------------------------------- |
| 전체 화면     | 왼쪽에는 HTML 코드가 표시되고 오른쪽에는 미리보기 창이 표시되는 이중 창 보기에서 작업하세요. |
| 어두운/밝은 테마 | 어두운 테마와 밝은 테마 간에 코드 편집기를 전환합니다.                        |
| 텍스트 보기    | 텍스트 보기로 다시 전환합니다.                                      |
