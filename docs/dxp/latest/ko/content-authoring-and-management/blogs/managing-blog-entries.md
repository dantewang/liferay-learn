# 블로그 항목 관리

블로거와 블로그 관리자는 사이트 관리 메뉴의 **블로그** 응용 프로그램을 사용하여 블로그 항목을 관리합니다. 이 앱에 액세스하려면 **제품 메뉴**(![제품 메뉴](../../images/icon-product-menu.png))를 열고 **콘텐츠 및 데이터** &rarr; **블로그** 를 선택하세요.

사이트의 블로그 항목을 나열하는 항목 탭이 기본적으로 선택됩니다. 관리 표시줄을 사용하여 사이트의 블로그 항목을 관리하세요.

## 블로그 항목 찾기 및 정렬

관리 표시줄에는 블로그 항목을 찾고 정렬하는 데 도움이 되는 도구도 포함되어 있습니다(예: 검색 표시줄을 사용하여 키워드로 파일 찾기).

검색창 왼쪽에 있는 정렬 버튼(![정렬 버튼](../../images/icon-sort.png))은 항목을 오름차순 또는 내림차순으로 정렬합니다.

다음 기준을 사용하여 항목을 정렬하려면 **필터 및 순서** 를 클릭하세요.

![Filter and Order menu](./managing-blog-entries/images/02.png)

| 여과기              | 묘사                 |
| :--------------- | :----------------- |
| (내비게이션으로 필터링) 모두 | 사이트의 모든 항목을 표시합니다. |
| (내비게이션으로 필터링) 광산 | 현재 사용자의 항목만 표시합니다. |
| (순서 기준) 제목       | 항목을 제목별로 정렬합니다.    |
| (정렬 기준) 표시 날짜    | 표시 날짜별로 항목을 정렬합니다. |

### 보기 유형

보기 유형 버튼은 추가 버튼 왼쪽에 있습니다. 사용자는 블로그 앱에서 블로그 항목을 표시하는 방법을 선택할 수 있습니다. 보기 유형 버튼의 아이콘은 선택한 보기 유형에 따라 다릅니다.

| 전망 유형 | 묘사                                                                                                                                                                           |
| :---- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 카드    | 작성자의 프로필 사진과 함께 블로그 항목의 카드형 렌더링을 표시합니다. 각 카드에는 항목의 타임스탬프, 제목, [workflow](../../process-automation/workflow/introduction-to-workflow.md) 상태(예: 승인됨, 초안 등) 및 작업 메뉴도 포함되어 있습니다. |
| 명부    | 블로그 항목의 커버 이미지 대신 작성자의 프로필 사진이 포함된 목록으로 카드 보기 유형과 동일한 정보를 표시합니다.                                                                                                             |
| 테이블   | 파일 렌더링이 없는 목록에 다른 보기 유형과 동일한 정보를 표시합니다.                                                                                                                                      |

![See the different View Types in the Site Administration Blogs Menu](./managing-blog-entries/images/03.png)

## 블로그 항목 삭제

1. 삭제하려는 항목을 확인하세요.

1. 선택한 항목을 휴지통으로 이동하려면 **휴지통**(![휴지통 아이콘](../../images/icon-trash.png))을 클릭하세요.

![Select and delete a blog entry](./managing-blog-entries/images/01.png)

## 블로그 항목 공유

사이트 관리의 블로그 앱에서 블로그 항목을 공유할 수도 있습니다. 블로그 항목을 공유하면 해당 블로그 항목과 공동 작업할 사용자를 초대합니다. 공유는 기본적으로 활성화되어 있습니다.
블로그 항목 공유는 [문서 및 미디어의 파일 공유](../documents-and-media/publishing-and-sharing/managing-document-access/sharing-documents-with-other-users.md)와 동일하게 작동합니다.

블로그 항목을 공유하려면,

1. 공유하려는 항목에 대해 **작업**(![작업 메뉴](../../images/icon-actions.png))을 클릭합니다. &rarr; **공유** .

1. 공동작업자의 이메일 주소를 입력하세요. 여러 개의 주소를 입력할 수 있습니다.

1. 수신 사용자가 항목을 공유할 수 있도록 하려면 **항목을 다른 사용자와 공유하도록 허용** 을 선택하세요.

1. 사용자에게 공유 권한을 부여하여 초대된 공동 작업자가 블로그 항목으로 수행할 수 있는 작업(업데이트(보기, 댓글 및 업데이트), 댓글(보기 및 댓글) 또는 보기(보기 전용))을 정의합니다.

1. **공유** 를 클릭하세요.
