# 회원을 사이트로 초대하기

회원 초대 위젯을 사용하여 사용자에게 사이트 가입 초대를 보낼 수 있습니다.

1. 페이지를 편집할 때 페이지 왼쪽 상단의 *추가* (![Add icon](../../../images/icon-add-app.png))를 클릭하여 위젯을 추가합니다.

1. 공동 작업 섹션에서 *회원 초대* 위젯을 찾아 페이지로 끌어다 놓습니다.

1. 게시된 페이지에서 *회원 초대* 를 클릭하여 새 양식을 불러옵니다.

    ![새 창에서 초대할 사용자를 선택합니다.](./inviting-members-to-your-site/images/02.png)

1. 초대를 받아야 하는 사용자 옆의 더하기 기호를 클릭합니다. 이미 초대를 받았지만 아직 응답하지 않은 사람은 이름 옆에 확인 표시가 나타납니다. 이미 초대를 수락한 사용자는 더 이상 목록에 표시되지 않습니다.

   목록에 없는 다른 사람의 경우 이메일로 초대하기 아래에 이메일 주소를 입력하고 *이메일 주소 추가*를 클릭할 수 있습니다.

1. 역할에 초대 아래의 드롭다운 메뉴를 사용하여 사용자의 역할을 선택합니다. *사이트 소유자*, *사이트 콘텐츠 검토자*, 또는 *사이트 관리자*중에서 선택합니다 .

    초대하려는 모든 사용자를 추가하고 역할을 선택했으면 *초대 보내기* 버튼을 클릭하여 초대합니다. 역할에 대한 자세한 내용은 [역할 및 권한](../../../users-and-permissions/roles-and-permissions/understanding-roles-and-permissions.md)을 참조하세요.

1. 사이트 초대는 사용자 알림 페이지의 요청 목록 탭 아래에 표시됩니다. 그런 다음 사용자는 초대를 확인하거나 무시하도록 선택할 수 있습니다.

    ![초대된 사용자의 알림 페이지에 초대 요청이 표시됩니다.](./inviting-members-to-your-site/images/03.png)

    사용자가 초대를 확인하면 정의한 역할이 할당된 사이트 멤버가 됩니다.
