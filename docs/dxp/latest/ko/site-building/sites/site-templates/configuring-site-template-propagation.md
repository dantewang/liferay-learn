# 사이트 템플릿 전파 구성

{bdg-secondary}`Liferay 7.4 U23+ 및 GA23+에 사용 가능`

사이트 템플릿을 사용하면 자동 전파를 위한 트리거를 구성할 수 있습니다. 기본적으로 Liferay는 변경 후 사용자가 처음 방문할 때 각 사이트에 대한 전파를 개별적으로 트리거합니다. 그러나 템플릿에 대해 자동 전파가 활성화될 때마다 동시에 연결된 사이트로의 전파를 트리거하도록 시스템을 구성할 수 있습니다.

```{important}
자동 전파는 템플릿에 대한 변경 범위에 따라 리소스를 많이 사용할 수 있습니다.
```

사이트 템플릿 전파를 구성하려면 다음 단계를 따르십시오.

1. *글로벌 메뉴* (![Global Menu](../../../images/icon-applications-menu.png))을 열고 *제어판* 탭으로 이동하여 *시스템 설정*을 클릭합니다.

1. *인프라* 설정으로 이동하고 사이드 메뉴에서 *사이트 템플릿* 을 클릭합니다.

1. *트리거 전파*을 확인하십시오.

   선택하지 않으면 사용자가 사이트를 처음 방문할 때 변경 사항이 사이트에 개별적으로 전파됩니다.

   선택한 경우 템플릿에서 자동 전파를 활성화하면 연결된 모든 사이트에 대한 전파도 동시에 트리거됩니다.

   ![트리거 전파를 확인하십시오.](./configuring-site-template-propagation/images/01.png)

1. *저장*을 클릭합니다.

사이트 템플릿에 대해 전파가 비활성화된 경우 전파를 활성화하면 변경 사항이 연결된 사이트에 동시에 전파되기 시작합니다. 전파는 템플릿에 대해 활성화된 상태로 유지되지만 후속 변경 사항은 사용자가 사이트를 처음 방문할 때 사이트에 개별적으로 전파됩니다.
