# 사이트 템플릿 전파 구성

{bdg-secondary}`Liferay 7.4 U23+/GA23+에서 사용 가능`

기본적으로 사이트 템플릿 변경 사항은 사용자가 템플릿을 업데이트한 후 사이트를 방문하면 연결된 사이트에 자동으로 전파됩니다. 그러나 연결된 모든 사이트에 동시에 전파되는 추가 수동 트리거를 구성할 수 있습니다. 수동 전파를 사용하려면 트리거 전파 설정을 사용합니다. 활성화된 경우 사이트 템플릿 편집기에서 [전파 준비됨](./propagating-template-changes.md#enabling-and-disabling-propagation) 버튼을 클릭하면 연결된 모든 사이트로 전파가 트리거됩니다. 이 설정을 사용하도록 설정해도 사용자가 연결된 사이트를 방문할 때 발생하는 템플릿 변경 사항의 자동 전파는 중지되지 않습니다.

```{important}
자동 전파는 템플릿의 변경 범위에 따라 리소스 집약적일 수 있습니다.
```

사이트 템플릿 전파를 구성합니다,

1. *글로벌 메뉴* (![Global Menu](../../../images/icon-applications-menu.png))를 열고 *제어판* 탭으로 이동한 다음 구성 아래 *시스템 설정* 을 클릭합니다.

1. 플랫폼에서 *인프라* 설정으로 이동하고 사이드 메뉴의 가상 인스턴스 범위에서 *사이트 템플릿* 을 클릭합니다.

1. 확인 *트리거 전파*.

   이 옵션을 선택하지 않으면 사용자가 사이트를 처음 방문할 때 변경 사항이 사이트에 개별적으로 전파됩니다.

   이 옵션을 선택하면 템플릿에서 자동 전파를 활성화하면 연결된 모든 사이트에 대한 전파도 동시에 트리거됩니다.

   ![트리거 전파를 확인합니다.](./configuring-site-template-propagation/images/01.png)

1. *저장*을 클릭합니다.

사이트 템플릿에 대해 전파가 비활성화되어 있는 경우 전파를 활성화하면 즉시 연결된 사이트에 변경 사항이 동시에 전파되기 시작합니다. 템플릿에 대한 전파가 활성화된 상태로 유지되는 동안 후속 변경 사항은 사용자가 사이트를 처음 방문할 때 개별적으로 사이트에 전파됩니다.
