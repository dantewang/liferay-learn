
# 사이트 템플릿 만들기

사이트 템플릿을 사용하면 사이트 생성을 위한 템플릿을 디자인하고 구성할 수 있습니다. 각 템플릿은 Sites와 동일한 도구를 사용하여 작성됩니다.

```{tip}
사이트 템플릿은 주로 여러 사이트에서 페이지를 유지하기 위한 것입니다. 사이트 전체에서 웹 콘텐츠 및 기타 자산을 만들고 유지 관리해야 하는 경우 [자산 라이브러리](../../../content-authoring-and-management/asset-libraries/asset-libraries-overview.md)를 사용하십시오. . 또한 사용 사례에 자동 전파가 필요하지 않은 경우 사이트 데이터를 [내보내기 및 가져오기](./exporting-importing-site-templates.md) 환경 간에 전송할 수 있습니다.
```

사이트 템플릿을 만들려면 다음 단계를 따르십시오.

1. **전역 메뉴**(![Global Menu](../../../images/icon-applications-menu.png))을 열고 **제어판** 으로 이동한 다음 **사이트 템플릿** 클릭합니다.

1. **추가** 버튼 ![Add Button](../../../images/icon-add.png)클릭합니다.

1. 템플릿에 대해 **이름** 및 **설명** 입력합니다.

1. 템플릿이 활성인지 비활성인지 확인합니다.

   활성 템플릿만 [사이트 생성](../creating-sites.md)동안 나열됩니다.

1. 사용자가 이 템플릿에서 상속된 페이지를 직접 수정할 수 있는지 여부를 결정합니다.

1. **전파 준비** 을 전환하여 자동 전파를 활성화 또는 비활성화합니다.

   자동 전파에 대한 자세한 내용은 [템플릿 변경 전파](./propagating-template-changes.md) 참조하십시오.

   ![사이트 템플릿의 이름을 지정하고 구성합니다.](./creating-site-templates/images/01.png)

1. **저장** 클릭합니다.

저장한 후 템플릿을 선택하여 편집을 시작할 수 있습니다.

템플릿 디자인 UI는 사이트 디자인 UI와 동일합니다. **사이트 템플릿 메뉴**(![Site Menu](../../../images/icon-product-menu.png))을 열면 사용 가능한 응용 프로그램에 액세스하고 스타일 북, 페이지 템플릿, 페이지 등을 추가할 수 있습니다.

```{important}
템플릿에서 상속된 페이지는 삭제할 수 없습니다. 또한 사이트 수준에서 하위 페이지를 직접 추가할 수 없습니다. 사이트 템플릿에서만 추가하거나 제거할 수 있습니다.
```
