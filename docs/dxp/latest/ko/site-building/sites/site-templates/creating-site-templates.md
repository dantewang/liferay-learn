# 사이트 템플릿 만들기

사이트 템플릿을 사용하면 사이트를 만들기 위한 템플릿을 디자인하고 구성할 수 있습니다. 각 템플릿은 사이트와 동일한 도구를 사용하여 구축됩니다.

```{tip}
사이트 템플릿은 주로 여러 사이트에 걸쳐 페이지를 유지 관리하기 위한 것입니다. 여러 사이트에서 웹 콘텐츠와 기타 자산을 만들고 유지해야 하는 경우 [자산 라이브러리](../../../content-authoring-and-management/asset-libraries/asset-libraries-overview.md)를 사용하세요. 또한 사용 사례에 자동 전파가 필요하지 않은 경우 사이트 데이터를 [내보내고 가져오기](./exporting-importing-site-templates.md) 환경 간에 전송할 수 있습니다.
```

사이트 템플릿을 만들려면 다음 단계를 따르세요:

1. **글로벌 메뉴**(![Global Menu](../../../images/icon-applications-menu.png))를 열고 **제어판** 으로 이동한 다음 **사이트 템플릿** 을 클릭합니다.

1. **추가** 버튼 ![Add Button](../../../images/icon-add.png)을 클릭합니다.

1. 템플릿에 **이름** 및 **설명** 을 입력합니다.

1. 템플릿이 활성 상태인지 비활성 상태인지 확인합니다.

   [사이트 생성 시에는 활성 템플릿만 나열됩니다](../adding-a-site.md).

1. 사용자가 이 템플릿에서 상속된 페이지를 직접 수정할 수 있는지 여부를 결정합니다.

1. **전파 준비 완료** 를 토글하여 자동 전파를 활성화 또는 비활성화합니다.

   자동 전파에 대해 자세히 알아보려면 [템플릿 변경 사항 전파](./propagating-template-changes.md) 를 참조하세요.

   ![사이트 템플릿의 이름을 지정하고 구성합니다.](./creating-site-templates/images/01.png)

1. **저장** 을 클릭합니다.

저장한 템플릿을 선택하여 편집을 시작할 수 있습니다.

템플릿을 디자인하는 UI는 사이트를 디자인하는 UI와 동일합니다. **제품 메뉴**(![Product Menu](../../../images/icon-product-menu.png))를 열어 사용 가능한 애플리케이션에 액세스하고 스타일북, 페이지 템플릿, 페이지 등을 추가하기 시작하면 됩니다.

```{important}
템플릿에서 상속된 페이지는 삭제할 수 없습니다. 또한 사이트 수준에서 하위 페이지를 직접 추가할 수 없습니다. 사이트 템플릿에서만 추가하거나 제거할 수 있습니다.
```
