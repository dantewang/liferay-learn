# 반응형 사이트 구축

전 세계 페이지 조회수의 절반 이상이 휴대폰이나 태블릿과 같은 모바일 기기에서 발생합니다. 모든 사이트는 데스크톱에서 보는 것처럼 모바일 장치에서도 보기 좋게 보여야 합니다. Liferay DXP는 어떤 장치가 귀하의 사이트를 방문하든 최상의 경험을 제공하는 데 도움이 됩니다.

즉시 사용 가능한 Liferay는 반응형 레이아웃을 사용합니다.

* Liferay의 UI 프레임워크를 사용하는 DXP 위젯과 사용자 정의 위젯은 화면 크기에 맞게 자동으로 크기가 조정됩니다.

  ![Adjust the widget's size.](./building-a-responsive-site/building-a-responsive-site/images/01.png)

* 탐색 및 제품 메뉴와 같은 UI 요소는 작은 화면에서도 계속 사용할 수 있도록 자동으로 조정됩니다.

  ![The main navigation adjusts its size.](./building-a-responsive-site/building-a-responsive-site/images/02.png)

* 화면 너비가 낮으면 Liferay는 모든 내용을 읽을 수 있도록 열을 결합합니다.

  ![Columns are combined when the width is low.](./building-a-responsive-site/building-a-responsive-site/images/03.png)

* 웹 개발자의 경우 테마 도구는 최적의 모바일 성능을 보장하는 데 도움이 됩니다.

기본 테마를 사용하는 경우 페이지는 이미 모바일 장치에 맞게 조정됩니다. 모든 것이 의도한 대로 나타나는지 확인하는 도구도 있습니다. 장치 시뮬레이터(![시뮬레이션](../../images/icon-simulation.png))는 다양한 장치에서 페이지가 어떻게 보이는지 보여줍니다.

```{tip}
Liferay 7.2에서는 모바일 장치 감지 시 사이트 동작을 제어하는 모바일 장치 규칙을 생성할 수 있습니다. 모바일 장치 규칙 생성에 대한 자세한 내용은 [모바일 장치 규칙 생성](./building-a-responsive-site/creating-mobile-device-rules.md) 을 참조하십시오.
```
