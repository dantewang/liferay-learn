# A/B 테스트 만들기

A/B 테스트를 만들기 전에 다음 사항을 기억하세요.

- 기본 경험 또는 세그먼트에 매핑된 개인화된 경험에 대한 테스트를 생성할 수 있습니다.
- 한 번에 한 페이지 또는 경험에 대한 A/B 테스트만 만들 수 있습니다.
- 실행 중인 A/B 테스트의 일부인 경험은 편집할 수 없습니다.
- 이미 활성 A/B 테스트가 실행 중인 경험에 대한 테스트를 생성할 수 없습니다.
- A/B 테스트의 일부인 콘텐츠 페이지 또는 경험을 삭제하면 테스트도 삭제됩니다.
- [테스트 실행](./running-and-monitoring-ab-tests.md) 없이 A/B 테스트 측면 패널을 닫으면 테스트가 초안으로 저장됩니다.
  
  ![테스트를 실행하지 않고 구성하면 테스트가 초안으로 저장됩니다](./creating-ab-tests/images/05.png)
  
  A/B 테스트를 만들려면 먼저 [테스트 만들기](#creating-the-test) 를 한 다음 하나 이상의 [테스트 변수](#creating-the-test-variant) 를 만듭니다.

```{important}
테스트를 생성하기 전에 환경이 A/B 테스트에 대한 [요구 사항을 충족](./verifying-ab-test-requirements.md)하는지 확인하세요.
```

## 테스트 만들기

1. 테스트할 콘텐츠 페이지로 이동합니다. 
1. 컨트롤 메뉴에서 **A/B 테스팅**(![A/B Test icon](../../../images/icon-ab-testing.png))을 클릭합니다.
1. 콘텐츠 페이지에 대한 다른 경험이 있는 경우 테스트할 경험을 선택합니다.
   
   ![A/B 테스트를 실행할 경험을 선택할 수 있습니다.](./creating-ab-tests/images/04.png)

1. **테스트** 생성 을 클릭합니다.

1. **테스트 이름** 을 입력하고 선택적으로 **설명** 을 입력합니다.
1. 테스트로 추적할 목표를 선택합니다.

   - **반송률** : 방문하고 어떤 활동도 나타내지 않고(예: **클릭** 또는 **스크롤**) 다른 페이지를 방문하지 않고 다른 페이지로 이동하는 비율입니다.
   - **Click** : 세션 당 페이지를 클릭한 사람의 비율.

1. **저장** 을 클릭하여 테스트를 **초안** 으로 저장합니다(아직 방문자에게 표시되지 않음). 

    ```{note}
    A/B 테스트 이름 옆에 있는 *작업* 버튼을 클릭하여 언제든지 새 A/B 테스트를 편집하거나 삭제할 수 있습니다. 삭제된 테스트는 복구할 수 없습니다(즉, 휴지통으로 보내지지 않음). 활성 실행 테스트에는 이러한 옵션을 사용할 수 없습니다.
    ```

## 테스트 변형 만들기

테스트 변형은 A/B 테스트로 최적화하려는 경험의 사용자 정의입니다. 각 A/B 테스트에는 변형이 하나 이상 있어야 합니다.

1. 콘텐츠 페이지의 A/B 테스트 사이드바 패널을 엽니다.
1. A/B 테스트에 대해 **클릭** 목표를 선택한 경우 테스트할 요소도 선택해야 합니다.
   
       1. **활성 테스트** 탭과 **클릭 목표** 섹션에서 **클릭 가능한 요소 선택** 를 클릭합니다.
       
                - **클릭** 목표에 대한 요소 ID를 사용하여 링크, 버튼 및 입력 필드를 타겟팅할 수 있습니다.
         - 테스트하려는 요소의 **요소 ID** 을 입력할 수도 있습니다.
         - 선택한 페이지 요소를 강조표시하려면 미리보기(![Preview](../../../images/icon-preview.png)) 아이콘을 클릭하십시오.
         - [테스트가 실행되는 동안](./running-and-monitoring-ab-tests.md)**클릭 가능한 요소** 변경 버튼을 사용할 수 없으며 **요소 ID** 를 변경할 수 없습니다.
           
           ![테스트하려는 콘텐츠 페이지 요소를 클릭하거나 클릭 목표 섹션에 해당 요소 ID를 입력합니다.](./creating-ab-tests/images/03.png)

    1. 콘텐츠 페이지에서 테스트할 요소를 클릭합니다.

    1. 클릭 **요소를 클릭 대상으로 설정** .
       
       ![요소를 클릭 대상으로 설정 버튼을 클릭하여 선택합니다.](./creating-ab-tests/images/01.png) 

         ```{note}
         클릭 타겟 요소는 전체 A/B 테스트에 적용되며 모든 대안에 있어야 합니다.
         ```

1. **변형** 섹션에서 **변형** 생성 을 클릭합니다.

1. Variant의 이름을 입력하고 **저장** 을 클릭합니다.
1. 변형 이름 옆에 있는 **편집**(![Edit icon](../../../images/icon-edit.png)) 버튼을 클릭합니다. 현재 콘텐츠 페이지는 변형의 기준선으로 편집 모드에서 열립니다.

1. 테스트하려는 변경 사항으로 변형을 편집합니다. 예를 들어 다른 CTA 텍스트를 테스트하려면 버튼을 새 텍스트로 업데이트합니다.

1. **대안** 저장 을 클릭합니다.

테스트 및 테스트 변형을 생성한 후 테스트를 [실행할 수 있습니다](./running-and-monitoring-ab-tests) .
