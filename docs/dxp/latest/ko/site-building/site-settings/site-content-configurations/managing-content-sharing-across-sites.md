# 사이트 간 콘텐츠 공유 관리

Liferay DXP 하위 사이트가 상위 사이트의 콘텐츠를 표시하도록 허용할 수 있습니다. 이 콘텐츠에는 다음이 포함됩니다.

- 웹 콘텐츠 구조
- 웹 콘텐츠 템플릿
- 문서 이름
- 어휘 및 범주
- Widget Templates
- 데이터 정의

인스턴스 또는 사이트 범위에서 콘텐츠 공유 설정을 구성할 수 있습니다. 인스턴스 범위 구성은 기본 사이트 범위 구성을 결정합니다. 예를 들어 인스턴스 범위에서 콘텐츠 공유를 비활성화하면 사이트 범위에서 기본적으로 설정이 비활성화됩니다.

## 인스턴스 범위에서 콘텐츠 공유 구성

1. 제품 메뉴를 열고 *제어판* &rarr; *구성* &rarr; *인스턴스 설정*로 이동합니다.
1. 콘텐츠 및 데이터 섹션에서 *공유*을 클릭합니다.
1. 가상 인스턴스 범위에서 *컨텐츠 공유*을 클릭하십시오.
1. *사이트 관리자가 자신이 관리하는 다른 사이트의 콘텐츠를 표시하도록 허용* 확인란을 선택하여 사이트 관리자가 자신이 관리하는 사이트 간에 콘텐츠를 공유할 수 있도록 합니다.
1. [사이트와 하위 사이트 간에 콘텐츠 공유 설정](#configure-the-content-sharing-setting-between-sites-and-subsites) 을 구성합니다.

## 사이트 범위에서 콘텐츠 공유 구성

```{note}
인스턴스 범위에서 콘텐츠 공유를 비활성화하면 사이트 범위에서 옵션을 사용할 수 없습니다.
```

1. 콘텐츠 공유 설정에 액세스합니다.

   - 라이프레이 DXP 7.4+에서

      1. 사이트 메뉴에서 *구성* &rarr; *사이트 설정*로 이동합니다.
      1. 콘텐츠 및 데이터 섹션에서 *공유*을 클릭합니다.

   - 이전 Liferay DXP 버전에서

      1. 사이트 메뉴에서 *구성* &rarr; *설정*로 이동합니다.
      1. 일반 영역에서 *문서 및 미디어* 섹션을 확장합니다.

1. [사이트와 하위 사이트 간에 콘텐츠 공유 설정](#configure-the-content-sharing-setting-between-sites-and-subsites) 을 구성합니다.

## 사이트와 하위 사이트 간의 콘텐츠 공유 설정 구성

*하위 사이트가 상위 사이트의 콘텐츠를 표시하도록 허용* 드롭다운 메뉴에서 다음 옵션 중 하나를 선택합니다.

- **기본적으로 활성화됨**: 하위 사이트는 기본적으로 상위 사이트의 콘텐츠를 표시할 수 있습니다. 사이트 범위에서 이 옵션을 변경할 수 있습니다.
- **기본적으로 비활성화됨**: 하위 사이트는 상위 사이트의 콘텐츠를 표시할 수 없습니다. 사이트 범위에서 이 옵션을 변경할 수 있습니다.
- **비활성화**: 하위 사이트는 상위 사이트의 콘텐츠를 표시할 수 없습니다. 사이트 범위에서는 이 옵션을 변경할 수 없습니다.
