# 사이트 문서 콘텐츠 보안

기본적으로 사이트의 문서 및 미디어 파일과 폴더는 웹 서버 디렉토리 색인에 표시되지 않으므로 브라우저를 통해 볼 수 없습니다. 웹 서버 디렉토리 색인을 통해 사이트의 문서 검색을 활성화하려면 사이트에 대해 이 설정을 활성화할 수 있습니다.

활성화되면 `http://localhost:8080/documents/site-name` (예: *Marketing*라는 사이트의 경우 `http://localhost:8080/documents/marketing` )에서 사이트의 문서 및 미디어 파일을 찾아볼 수 있습니다. ).

## 웹 서버 디렉토리 색인에서 문서 가시성 구성

1. 디렉터리 인덱싱 활성화 옵션에 액세스합니다.

    - 라이프레이 DXP 7.4+에서

      1. 사이트 메뉴에서 *구성* &rarr; *사이트 설정*로 이동합니다.
      1. 콘텐츠 및 데이터 섹션에서 *문서 및 미디어*을 클릭합니다.

       ![Liferay DXP 7.4+에서는 사이트 설정 섹션에서 디렉토리 인덱싱 활성화 옵션을 변경합니다.](./securing-site-documents-content/images/02.png)

   - 이전 Liferay DXP 버전에서

      1. 사이트 메뉴에서 *구성* &rarr; *설정*로 이동합니다.
      1. 일반 영역에서 *문서 및 미디어* 섹션을 확장합니다.

       ![이전 Liferay DXP 버전에서는 설정 섹션에서 디렉토리 인덱싱 활성화 옵션을 변경합니다.](./securing-site-documents-content/images/01.png)

1. 이 설정을 활성화 또는 비활성화하려면 *디렉터리 인덱싱 활성화* 설정을 전환합니다.

    ```{important}
    활성화되면 보기 권한이 있는 관리자 및 사용자만 웹 서버 디렉토리 인덱스에서 사이트의 문서 및 미디어 파일을 검색할 수 있습니다.
    ```

1. *저장*을 클릭합니다.
