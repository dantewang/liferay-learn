# 사이트 현지화

```{note}
Liferay 설치를 처음 설정하는 경우 [초기 인스턴스 현지화](../../installation-and-upgrades/setting-up-liferay/initial-instance-localization.md)를 먼저 검토하는 것이 좋습니다.
```

## 사이트 언어 구성

사이트의 기본 언어를 설정할 수 있습니다. 사이트의 활성화된 언어를 정의할 수도 있습니다.

1. [사이트 설정 UI](./site-settings-ui-reference.md#language) 에서 언어 탭을 클릭합니다.

   ![사이트 설정에서 언어 탭을 클릭합니다.](./site-localization/images/01.png)

1. *이 사이트에 대한 사용자 지정 기본 언어 및 추가 사용 가능한 언어 정의*를 클릭합니다.

   ![사용자 지정 기본 언어 정의 및 이 사이트에 사용 가능한 추가 언어를 클릭합니다.](./site-localization/images/02.png)

1. 기본 언어를 목록에 있는 언어로 변경할 수 있습니다.

1. 활성화 또는 비활성화할 추가 언어를 선택할 수도 있습니다.

   ![활성화할 추가 언어 선택](./site-localization/images/03.png)

1. 왼쪽 및 오른쪽 화살표 버튼을 사용하여 추가 언어를 활성화하거나 비활성화합니다.

1. *저장* 버튼을 클릭하여 변경 사항을 저장합니다.

## 지역화할 수 있는 사이트 필드 수정

일부 필드에는 추가 언어 플래그 버튼이 있습니다. 이 필드는 다른 언어로 번역될 수 있습니다.

![현지화 가능한 필드는 언어 플래그로 표시됩니다.](./site-localization/images/04.png)

1. 필드 옆의 언어 플래그를 클릭합니다.

   ![현지화 가능 필드 옆에 있는 언어 플래그를 클릭하여 새 값을 입력합니다.](./site-localization/images/05.png)

1. 선택한 언어로 새 값을 입력합니다. 기본값은 참조를 위해 필드 아래에 표시됩니다.

   ![선택한 언어의 새 값을 입력합니다.](./site-localization/images/06.png)

Liferay는 각 언어에 대한 번역을 자동으로 저장하고 해당 언어가 인스턴스에 대해 선택되면 번역된 값을 표시합니다. 필드에 대한 번역이 제공되지 않으면 해당 필드에 대한 기본 인스턴스 언어의 값이 표시됩니다.
