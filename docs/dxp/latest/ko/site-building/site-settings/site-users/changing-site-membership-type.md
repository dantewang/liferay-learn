# 사이트 회원 유형 변경

사이트의 회원 유형은 사용자가 사이트에 얼마나 자유롭게 가입할 수 있는지를 결정합니다. 사이트 구성원에는 세 가지 유형이 있습니다.

- **개방:** 사용자는 원할 때마다 가입하고 나갈 수 있습니다. 사이트는 내 사이트 앱의 모든 사용자에게 표시됩니다.

- **제한됨:** 사이트는 내 사이트 응용 프로그램에 표시되며 사용자는 가입하려면 회원 자격을 요청해야 합니다.

- **비공개:** 사이트 관리자는 명시적으로 사용자를 초대하여 사이트에 추가해야 합니다. 비공개 멤버십 사이트는 내 사이트 앱에 표시되지 않습니다.

## 사이트 구성원 유형 설정

사이트 회원 유형을 변경하려면 다음 단계를 따르세요.

1. 회원 유형 옵션에 액세스:

    - 라이프레이 DXP 7.4+에서

      1. 사이트 메뉴에서 **구성** &rarr; **사이트 설정** 로 이동합니다.
      1. 플랫폼 섹션에서 **사이트 구성** 을 클릭한 다음 **세부사항** 을 클릭하십시오.

       ![Liferay DXP 7.4+에서는 사이트 설정 섹션에서 사이트 구성원 유형을 변경합니다.](./changing-site-membership-type/images/02.png)

   - 이전 Liferay DXP 버전에서

      1. 사이트 메뉴에서 **구성** 로 이동하여 **일반** 섹션을 클릭합니다.
      1. 세부 정보 영역에서 회원 유형 옵션을 찾습니다.

       ![이전 Liferay DXP 버전에서는 설정 섹션에서 사이트 멤버십 유형을 변경합니다.](./changing-site-membership-type/images/01.png)

1. **회원 유형** 선택기를 열고 옵션을 선택합니다.

1. **저장** 을 클릭합니다.
