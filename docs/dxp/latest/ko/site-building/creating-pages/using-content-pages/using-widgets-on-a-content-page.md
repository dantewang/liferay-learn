# 콘텐츠 페이지에서 위젯 사용

<!--TASK: Reconsider-->

위젯 섹션은 [위젯 페이지의 메뉴 추가](../using-widget-pages/adding-widgets-to-a-page.md)와 동일하게 작동합니다. 사용 가능한 위젯의 전체 목록이 나타나고 해당 위젯을 페이지에 추가할 수 있습니다. 주요 차이점은 위젯의 기본 구성 옵션만 콘텐츠 페이지에서 사용할 수 있다는 것입니다.

```{note}
위젯은 위젯의 [render-weight](https://resources.learn.liferay.com/reference/latest/en/dxp/definitions/liferay-portlet-app_7_4_0.dtd.html#render-weight) 에 따라 렌더링됩니다. 페이지, 콘텐츠 페이지에는 해당되지 않습니다. 대신 페이지에 나타나는 순서대로(예: 왼쪽에서 오른쪽, 위에서 아래) 렌더링됩니다.
```

## 콘텐츠 페이지에 위젯 추가

1. [콘텐츠 페이지에 요소를 추가](./adding-elements-to-content-pages.md)하는 동안 [위젯 패널](./content-page-editor-ui-reference.md)을 열고 사용 가능한 위젯을 찾아보세요. 위젯을 선택하거나 이름으로 위젯을 검색하세요.

1. 위젯을 배치하려는 레이아웃의 열과 행으로 위젯을 드래그하세요.

위젯은 Wiki 디스플레이일 수도 있고 자산 게시자와 같은 동적 게시 도구일 수도 있습니다. 위젯으로 표시하는 콘텐츠는 긴 형식의 텍스트, 이미지 갤러리 또는 그 사이의 모든 콘텐츠일 수 있습니다.

## 콘텐츠 페이지에 위젯 구성

1. 위젯 위로 마우스를 가져갑니다.

1. **작업**(![작업 메뉴 아이콘](../../../images/icon-actions.png))을 클릭하고 **구성** 을 선택합니다. 여기에서는 위젯에 따라 여러 가지 옵션이 있습니다.

   ![You can configure widgets on content pages through their options menu.](./using-widgets-on-a-content-page/images/01.png)

1. 변경 사항을 적용하려면 **저장** 을 클릭하세요.

## 콘텐츠 페이지에 대한 위젯 권한 구성

Liferay Portal 7.3 GA2부터 콘텐츠 페이지에서 위젯에 대한 권한을 구성할 수 있습니다. 콘텐츠 페이지에서 위젯 권한을 구성하려면 다음 단계를 따르세요.

1. 위젯 위로 마우스를 가져갑니다.

1. **작업**(![작업 메뉴 아이콘](../../../images/icon-actions.png))을 클릭하고 **권한** 을 선택합니다.

1. 나타나는 새 창에서 역할에 대한 권한을 선택/선택 취소합니다.

1. 변경 사항을 적용하려면 **저장** 을 클릭하세요.

```{note}
[페이지 템플릿](../adding-pages/creating-a-page-template.md)을 기반으로 페이지를 생성하면 권한도 복사됩니다. 마스터 페이지에 대한 권한은 페이지가 아닌 [마스터 페이지](../defining-headers-and-footers/managing-master-page-templates.md)에서 설정됩니다.
```

## 콘텐츠 페이지에서 위젯 내보내기/가져오기

1. 위젯 위로 마우스를 가져갑니다.

1. **작업**(![작업 메뉴 아이콘](../../../images/icon-actions.png))을 클릭하고 **내보내기 / 가져오기** 를 선택합니다.

1. 내보내기 또는 가져오기 탭을 선택하여 해당 프로세스 구성을 시작합니다.

## 콘텐츠 페이지에서 위젯 구성 템플릿 구성

1. 위젯 위로 마우스를 가져갑니다.

1. **작업**(![작업 메뉴 아이콘](../../../images/icon-actions.png))을 클릭하고 **구성 템플릿** 을 선택합니다.

1. 위젯의 현재 설정을 저장하려면 **현재 구성을 템플릿으로 저장** 을 클릭하세요.

1. 템플릿 이름을 입력하고 **저장** 을 클릭하세요.

## 위젯에 대한 콘텐츠 페이지 제한 사항

콘텐츠 페이지에서 위젯에 대한 기본 구성 옵션 및 권한에 액세스할 수 있지만 여전히 위젯 페이지에서만 사용할 수 있는 몇 가지 옵션이 있습니다.

- **중첩된 애플리케이션** : 위젯 내의 위젯은 위젯 페이지에서만 지원됩니다.

- **룩앤필 구성** : 위젯 페이지에서 위젯의 작업(![작업 아이콘](../../../images/icon-actions.png))을 통해 룩앤필 구성 메뉴에 액세스할 수 있습니다. CSS를 세밀하게 제어할 수 있습니다. 콘텐츠의 모양과 느낌은 테마에서 또는 조각을 사용하여 정의되므로 콘텐츠 페이지의 위젯에는 사용할 수 없습니다.
