# 페이지 댓글 사용

콘텐츠 페이지를 구축할 때 귀하와 귀하의 팀은 Liferay의 페이지 댓글 기능을 사용하여 협업할 수 있습니다. 댓글은 Liferay DXP 7.3+에서 기본적으로 활성화됩니다.

![Use page comments to collaborate with your teammates.](./using-page-comments/images/01.png)

## 페이지 댓글 추가

1. 페이지 요소(예: 조각 또는 위젯)가 포함된 콘텐츠 페이지로 이동하여 **편집** 버튼(![편집 버튼](../../../images/icon-edit-pencil.png))을 클릭합니다. 페이지 편집을 시작하려면

1. 사이드바 메뉴에서 **댓글** 패널(![댓글 아이콘](../../../images/icon-comments-w.png))을 열고 원하는 조각이나 위젯을 선택합니다.

   또는 원하는 페이지 요소를 선택하고 해당 **댓글** 아이콘을 클릭할 수 있습니다. 댓글 패널이 열립니다.

   ![Click the Comments icon for the page element.](./using-page-comments/images/02.png)

   ```{note}
   [조각 하위 요소](../page-fragments-and-widgets/using-fragments/configuring-fragments/fragment-sub-elements-reference.md), 컨테이너 또는 그리드에는 주석을 추가할 수 없습니다.
   ```

1. 측면 패널에 댓글을 입력하고 **댓글** 을 클릭하세요.

댓글이 추가된 후 **작업** 버튼(![액션 버튼](../../../images/icon-actions.png))을 통해 댓글을 편집하거나 삭제할 수 있습니다. 사용자는 이에 답글을 추가할 수도 있습니다.

![Edit, delete, or reply to comments.](./using-page-comments/images/03.png)

```{note}
댓글을 삭제하면 해당 댓글에 대한 모든 답글이 삭제됩니다.
```

## 페이지 댓글 검토

페이지 조각이나 위젯에 댓글이 추가되면 댓글 측면 패널에서 이를 검토할 수 있습니다.

![Review page comments in the comments side panel.](./using-page-comments/images/04.png)

공동 작업 시 **해결** 버튼(![해결 버튼](../../../images/icon-resolve.png))을 클릭하여 댓글을 해결됨으로 표시할 수 있습니다. 기본적으로 해결된 댓글과 답변은 숨겨집니다. 그러나 **해결된 댓글 표시** 를 선택하면 해결된 댓글을 볼 수 있습니다.

![Resolve comments.](./using-page-comments/images/05.png)

해결된 댓글을 보는 동안 녹색 **해결** 버튼을 클릭하면 댓글을 다시 열 수 있습니다.

![View and reopen resolved comments.](using-page-comments/images/06.png)
