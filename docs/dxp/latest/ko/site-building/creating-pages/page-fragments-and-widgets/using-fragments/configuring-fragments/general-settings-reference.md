# 일반 설정 참조

HTML, 단락, 구분 기호 및 소셜 조각을 제외한 모든 조각에는 일반 설정이 있습니다. 이러한 옵션 중 일부는 표준(예: 가시성 및 프레임)인 반면 다른 옵션은 각 조각에 고유합니다. 다음 참조에는 일반 설정이 있는 조각만 포함됩니다.

프래그먼트의 일반 설정에 액세스하려면 다음 단계를 따르십시오.

1. 조각을 지원하는 페이지 또는 템플릿 편집을 시작합니다.

1. 사이드바 메뉴에서 **브라우저** 패널(![Browser](../../../../../images/icon-hierarchy.png))을 열고 원하는 조각을 선택합니다.

   또는 페이지 또는 템플릿에서 원하는 조각을 클릭합니다.

1. 페이지 요소 측면 패널에서 **일반** 탭을 클릭합니다.

![페이지 요소 측면 패널에서 일반 설정에 액세스할 조각을 선택합니다.](./general-settings-reference/images/01.png)

## 표준 일반 설정

### 프래그먼트 숨김

| 분야       | 묘사                                                                                                                               |
|:-------- |:-------------------------------------------------------------------------------------------------------------------------------- |
| 프래그먼트 숨김 | 다른 뷰포트에 대한 조각을 숨기려면 확인란을 선택합니다. 이 설정이 페이지 뷰포트와 상호 작용하는 방법에 대한 자세한 내용은 [조각 가시성 구성](../configuring-fragment-visibility.md) 참조하십시오. |

### 프레임

| 분야          | 묘사                                                                                                       |
|:----------- |:-------------------------------------------------------------------------------------------------------- |
| 너비/높이       | 조각 프레임의 기본 높이와 너비를 결정합니다.                                                                                |
| 최소 너비/최대 너비 | 다른 뷰포트에 표시될 때 프레임의 최소 및 최대 너비를 결정합니다.                                                                    |
| 최소 높이/최대 높이 | 다른 뷰포트에 표시될 때 프레임의 최소 및 최대 높이를 결정합니다.                                                                    |
| 워크플로        | 조각 콘텐츠가 프레임을 오버플로하도록 허용할지 여부를 결정합니다. 기본적으로 조각 콘텐츠는 해당 프레임으로 제한됩니다. 오버플로가 허용되면 표시 방법을 결정할 수 있습니다(예: 스크롤). |

## 컨테이너 옵션

| 필드 | 설명 |
| :--- | :--- |
| 링크 | 컨테이너가 URL, 페이지 또는 매핑된 URL에 연결되는지 확인합니다. 이 설정은 다음 중 사용 가능한 필드를 결정합니다. |
| URL(**URL의 경우**) | 컨테이너에 대해 현지화할 수 있는 URL을 입력합니다. 다른 언어에 대한 URL을 설정하려면 먼저 편집 막대를 통해 페이지 또는 템플릿의 언어를 전환하십시오. 그런 다음 원하는 URL을 입력합니다. |
| 페이지(**페이지용**) | 컨테이너에 대한 Liferay 페이지를 선택합니다. |
| 항목(**매핑된 URL의 경우**) | 문서 및 미디어, 웹 콘텐츠, 블로그 또는 범주에서 항목을 선택합니다. 선택한 항목은 매핑을 위한 필드를 제공합니다. |
| 필드(**매핑된 URL의 경우**) | 컨테이너에 매핑하려는 URL 필드를 선택합니다. 유효한 필드를 선택하면 **URL** 필드가 자동으로 채워집니다. |
| URL(**매핑된 URL의 경우**) | 이 필드는 선택한 **필드** URL로 자동으로 채워집니다. |
| 새 탭에서 열기 | 링크가 새 탭에서 열리는지 확인합니다. |
| 콘텐츠 표시 | 컨테이너의 콘텐츠가 블록, 플렉스 행 또는 플렉스 열에 표시되는지 여부를 결정합니다. 플렉스 옵션에는 추가 구성이 있습니다. |
| 플렉스 랩(**플렉스 로우 또는 플렉스 컬럼용**) {bdg-secondary}`7.4 U19+/GA19+` | Flex 행 및 열에 대한 래핑 구성: 래핑 안 함, 래핑 또는 래핑 리버스. |
| 항목 정렬(**플렉스 열의 경우**) {bdg-secondary}`7.4 U19+/GA19+` | 열의 요소가 정렬되는 방식(시작, 가운데, 끝, 늘이기 또는 기준선)을 결정합니다. |
| 콘텐츠 양쪽 맞춤(**Flex Row용**) {bdg-secondary}`7.4 U19+/GA19+` | 행의 요소가 정렬되는 방식(시작, 가운데, 끝, 사이 또는 주변)을 결정합니다. |
| 컨테이너 폭 | 컨테이너가 유동 또는 고정 너비를 사용하는지 여부를 결정합니다. |

| 플렉스 랩(**플렉스 로우 또는 플렉스 컬럼용**) | Flex 행 및 열에 대한 래핑 구성: 래핑 안 함, 래핑 또는 래핑 리버스. |
| 항목 정렬(**플렉스 열의 경우**) | 열의 요소가 정렬되는 방식(시작, 가운데, 끝, 늘이기 또는 기준선)을 결정합니다. |
| 콘텐츠 양쪽 맞춤(**Flex Row용**) | 행의 요소가 정렬되는 방식(시작, 가운데, 끝, 사이 또는 주변)을 결정합니다. |
| 컨테이너 폭 | 컨테이너가 유동 또는 고정 너비를 사용하는지 여부를 결정합니다. |

| 분야      | 묘사                                                                                                                                                                                                                    |
|:------- |:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 문자 수 표시 | 입력한 문자 수를 표시하고 최대 문자 수를 초과하면 경고를 표시합니다. 개체의 필드 구성에서 최대 문자 수를 구성할 수 있습니다. 자세한 내용은 [필드 유형 UI 참조](../../../../../building-applications/objects/creating-and-managing-objects/fields/adding-fields-to-objects.md) 참조하십시오. |

|:----- |:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

### 기타 양식 조각 구성

| 분야            | 묘사                                                           |
|:------------- |:------------------------------------------------------------ |
| 분야            | 조각을 개체 필드에 매핑합니다.                                            |
| 필수로 표시        | 양식 제출에 필요한 조각 필드를 표시합니다. 필수 개체 필드는 자동으로 필수로 표시됩니다.           |
| 레이블 표시        | 조각의 레이블을 표시하거나 숨깁니다.                                         |
| 라벨            | 프래그먼트에 대해 현지화 가능한 레이블을 설정합니다. 기본적으로 각 조각은 개체 필드의 레이블을 사용합니다. |
| 도움말 텍스트 표시    | 조각의 도움말 텍스트를 표시하거나 숨깁니다.                                     |
| 도움말 텍스트       | 조각에 대해 현지화할 수 있는 도움말 텍스트를 설정합니다.                             |
| 자리표시자         | 숫자 입력 및 텍스트 입력 조각의 경우 예제 값을 입력합니다.                           |
| 버튼 텍스트        | 파일 업로드 조각의 경우 파일 첨부를 위한 현지화 가능한 버튼 텍스트를 설정합니다.               |
| 지원되는 파일 정보 표시 | 파일 업로드 조각의 경우 지원되는 확장명 및 파일 크기 제한 목록을 표시하거나 숨깁니다.            |
