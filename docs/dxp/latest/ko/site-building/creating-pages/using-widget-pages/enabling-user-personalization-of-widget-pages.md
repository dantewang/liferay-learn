# 위젯 페이지의 사용자 개인화 활성화

관리자는 위젯 페이지의 일부 또는 전체 섹션(페이지 레이아웃의 행 및 열)을 사이트 구성원에 대해 사용자 정의할 수 있도록 지정할 수 있습니다.

해당 사용자 정의 가능 영역에서 수행된 페이지 사용자 정의는 사용자 정의를 수행한 사용자에게만 나타납니다.

## 페이지 사용자 정의 활성화

1. **사이트 메뉴**(![사이트 메뉴](../../../images/icon-product-menu.png))를 열고 **사이트 빌더** &rarr; **페이지** 로 이동합니다.

1. 원하는 위젯 페이지 옆에 있는 **작업**(![Actions](../../../images/icon-actions.png))을 열고 **구성** 을 선택합니다.

1. 페이지 상단의 **일반** 탭을 선택하고 **사용자 정의 설정** 패널로 이동합니다.

1. **사용자 정의 가능** 확인란을 선택하세요.

   ![To enable page customizations, click on the Configure Page button next to the page, expand the Customization Settings area, and check Customizable option.](./personalizing-pages/images/01.png)

1. 사용자 정의할 각 섹션에 대해 **사용자 정의 불가능** 옵션을 **사용자 정의 가능** 으로 전환합니다.

## 페이지 사용자 정의

페이지에 대한 사용자 정의가 활성화되면 사이트 회원은 오른쪽 상단에 있는 추가 메뉴에서 페이지의 사용자 정의 가능한 섹션에 앱을 추가할 수 있습니다. 사이트 회원은 사용자 정의 가능한 영역에 대해 두 가지 유형의 사용자 정의를 수행할 수 있습니다.

- 앱 추가 또는 제거
- 애플리케이션 구성

```{note}
해당 앱은 추가된 사이트에 연결되어 있으므로 사이트 구성원은 사용자 정의 가능한 영역 내에서 인스턴스화할 수 없는 앱의 구성을 변경할 수 없습니다.
```

사이트 회원은 제어 메뉴에 있는 페이지 옵션 메뉴(![Options](../../../images/icon-options.png))에서 이러한 상황별 옵션에 액세스할 수 있습니다.

**사용자 정의 없이 페이지 보기** : 기본 페이지를 표시하고 추가 메뉴를 숨깁니다.

**내 사용자 정의 재설정** : 사용자 정의된 페이지를 기본 페이지로 복원하고 모든 사용자 정의를 삭제합니다.

**내 사용자 정의 페이지 보기** : 사용자의 사용자 정의가 포함된 페이지를 표시하고 사용자가 페이지에 위젯을 추가할 수 있도록 추가 메뉴를 사용할 수 있도록 합니다.

![Customizable areas are highlighted green when organizing apps on the page.](./personalizing-pages/images/02.png)

사용자 정의 가능한 페이지의 관리자는 사이트 회원과 동일한 두 가지 보기(기본 페이지 보기 및 사용자 정의 페이지 보기)를 갖습니다. 기본 페이지의 사용자 정의할 수 없는 섹션에 대한 변경 사항은 다른 페이지와 마찬가지로 모든 사용자에게 영향을 미치는 반면, 사용자 정의된 섹션에 대한 변경 사항은 자신에게만 영향을 미치며 사용자의 사용자 정의 내용을 덮어쓰지 않습니다.

## 역할에 페이지 사용자 정의 권한 부여

기본적으로 사이트 회원은 페이지를 사용자 정의할 수 있지만 사이트 회원이 아닌 사람과 게스트는 할 수 없습니다. 관리자는 기존 역할에 대한 권한을 업데이트하거나 [새 역할을 생성](../../../users-and-permissions/roles-and-permissions/creating-and-managing-roles.md)하고 할당할 수 있습니다. 다른 사용자가 페이지를 사용자 정의할 수 있도록 합니다. 페이지를 사용자 정의할 수 있는 역할 권한을 부여하려면 다음 단계를 따르십시오.

1. **글로벌 메뉴**(![응용 프로그램 메뉴 아이콘](../../../images/icon-applications-menu.png))를 열고 **제어판** &rarr; **사용자** &rarr; **역할** 로 이동합니다.

1. 원하는 역할 옆에 있는 **작업**(![Actions](../../../images/icon-actions.png))을 열고 **편집** 을 선택합니다.

1. 역할에 대한 **권한 정의** 탭을 클릭하고 요약 아래의 **사이트 및 자산 라이브러리 관리** &rarr; **사이트 빌더** &rarr; **페이지** 로 이동합니다.

1. 리소스 권한 제목 아래의 페이지 섹션까지 아래로 스크롤하고 **사용자 정의** 권한을 확인하세요.

   사용자는 사용자 정의 권한을 사용하여 앱의 모양과 느낌을 사용자 정의하고 앱 설정을 가져오거나 내보낼 수 있습니다.

1. 아래로 스크롤하고 **저장** 을 클릭하여 변경 사항을 적용합니다.
