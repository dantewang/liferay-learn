
# 게시 도구 비교

Liferay는 콘텐츠 생성 및 사이트 구축을 위한 두 가지 게시 도구인 게시 및 스테이징을 제공합니다. 간행물은 뛰어난 공동 작업 경험과 다른 Liferay 기능과의 뛰어난 통합을 제공하기 때문에 거의 모든 사용 사례에서 스테이징보다 권장됩니다. 한 가지 예외는 별도의 서버에서 작업 환경과 생산 환경을 호스팅해야 하는 경우입니다. 이를 위해서는 원격 라이브 스테이징을 사용해야 합니다.

[$LIFERAY_LEARN_YOUTUBE_URL$]=https://www.youtube.com/embed/xPVWyMrTa3k

## 출판

출판물은 사이트 및 콘텐츠를 편집하기 위한 변경 추적 도구입니다. 이를 통해 기여자는 **환경의 게시된 사이트에 영향을 주지 않고 편집할 수** 작업 환경을 만들 수 있습니다. 각 게시는 회사 범위이며 여러 사이트 및 자산 라이브러리에서 변경 사항을 그룹화할 수 있습니다. 프로젝트 및 팀에 필요한 만큼 발행물을 만들 수 있습니다.

기본적으로 발행물에 대한 액세스는 해당 작성자로 제한됩니다. 그러나 제작자는 다른 회원을 초대하고 각 출판물에 특정한 역할을 할당할 수 있습니다. 게시는 독립적이므로 기여자는 서로에게 영향을 주지 않고 변경 사항을 만들고 게시할 수 있습니다. 간행물의 기여자는 자신의 타임라인에서 작업하고 준비가 되면 변경 내용을 게시할 수 있습니다.

게시 프로세스 중에 Liferay는 프로덕션과의 충돌을 확인하고 사용자에게 이를 검토하고 해결하라는 메시지를 표시합니다. 일부 충돌은 자동으로 해결될 수 있지만 다른 충돌은 수동 해결이 필요합니다. 일단 게시되면 충돌이 없으면 변경 사항이 다른 게시로 전파됩니다. 간행물은 게시된 변경 사항에 대한 자세한 기록을 유지하므로 필요한 경우 변경 사항을 신속하게 검토하고 되돌릴 수 있습니다.

이러한 기능을 함께 사용하면 뛰어난 편집 및 공동 작업 환경을 제공합니다. 인스턴스에 대한 간행물을 설정하려면 [간행물 활성화](./publications/enabling-publications.md) 참조하십시오.

```{important}
Liferay 7.4 U44/GA44부터 Publications는 [워크플로우](../../process-automation/workflow.md) 과 완전히 통합되므로 Publications에서 추적하는 엔터티에 대한 워크플로를 활성화할 수 있습니다.
```

## 스테이징

스테이징을 사용하면 기여자는 변경 사항을 **라이브** 환경에 게시하기 전에 공유 **스테이징** 환경에서 작업합니다. 사용자는 사이트 및 자산 라이브러리에 대한 스테이징을 개별적으로 활성화합니다. 활성화되면 Liferay는 원래 사이트 또는 라이브러리의 별도 사본을 생성하고 동일한 서버([로컬 라이브 스테이징](./staging/configuring-local-live-staging.md)) 또는 별도의 서버([원격 라이브 스테이징](./staging/configuring-remote-live-staging.md))에서 호스팅합니다.

모든 변경 사항은 사이트 또는 자산 라이브러리 범위입니다. 사이트 또는 자산 라이브러리 내에서 준비하려는 정확한 페이지 및 응용 프로그램 데이터를 결정할 수 있습니다. 명시적으로 준비된 데이터에 대한 변경 사항만 추적됩니다. 실시간 환경이 들어오는 사용자 트래픽을 처리하는 동안 준비가 되면 개별적으로 또는 모두 함께 변경 사항을 게시합니다.

스테이징보다 게시가 권장되지만 배포 모델에 다중 계층 아키텍처를 사용하거나 작업 및 프로덕션 환경에 대해 서로 다른 보안 요구 사항이 있는 경우 원격 라이브 스테이징이 필요합니다. Publications에서 현재 원격 옵션을 제공하지 않기 때문입니다. 다른 모든 사용 사례에서는 간행물을 사용하십시오.

```{important}
스테이징을 사용해야 하는 경우 개발 초기에 활성화해야 합니다. 활성화되면 스테이징은 즉시 환경 간에 데이터 복사를 시작하므로 많은 데이터가 이미 존재하는 경우 오류가 발생할 수 있습니다. 스테이징은 데이터를 작은 블록으로 자주 게시할 때 가장 잘 작동합니다. 변경 사항을 점진적으로 게시하면 게시 오류를 방지하는 데 도움이 됩니다.
```
