# 스테이징 권한 관리

스테이징을 사용하면 스테이징 및 페이지 버전 관리를 직접 처리하는 사용자 권한을 관리할 수 있습니다.

* [일반 스테이징 권한](#general-staging-permissions)
* [관련 사이트 권한](#related-site-permissions)
* [개별 변형 권한](#individual-variation-permissions)

## 일반 스테이징 권한

1. *제어판* &rarr; *사용자* &rarr; *역할*로 이동합니다.

1. 수정하려는 역할에 대해 *작업* 버튼( ![Actions button](../../../images/icon-actions.png) )을 클릭하고 *편집을 선택*.

   또는 *추가* 버튼(![Add button](../../../images/icon-add.png))을 클릭하고 *새 역할* 섹션을 작성하여 새 역할을 만들 수 있습니다.

1. *권한 정의* 탭을 클릭합니다.

1. 사이드 메뉴에서 *사이트 및 자산 라이브러리 관리* &rarr; *게시* &rarr; *스테이징*을 클릭합니다.

1. 사용자 역할에 할당할 일반 준비 권한을 선택하고 해당 범위를 정의합니다.

   ![원하는 스테이징 권한을 할당하고 해당 범위를 정의합니다.](./managing-staging-permissions/images/04.png)

## 관련 사이트 리소스 권한

*권한 정의* 탭에서 스테이징과 관련된 기타 관련 사이트 *리소스 권한* 을 할당할 수도 있습니다. 사이드 메뉴에서 *제어판* &rarr; *사이트* &rarr; *사이트*을 클릭합니다.

다음 권한은 스테이징과 관련이 있습니다.

**페이지 변형 추가**: 이 권한은 스테이징 바의 페이지 변형 관리 화면에서 *페이지 변형 추가* 버튼을 숨기거나 표시합니다.

**사이트 페이지 변형**추가 : 이 권한은 스테이징 바의 사이트 페이지 변형 관리 화면에서 *사이트 페이지 변형 추가* 버튼을 숨기거나 표시합니다.

**애플리케이션 정보 내보내기/가져오기**: Publish Staging 권한이 부여되지 않은 경우 이 권한은 애플리케이션 수준 내보내기/가져오기 메뉴를 숨기거나 표시합니다. 내보내기/가져오기 앱에 대한 구성 권한도 필요합니다.

**페이지 내보내기/가져오기**: Publish Staging 권한이 부여되지 않은 경우 이 권한은 사이트 관리 메뉴에서 내보내기/가져오기 앱을 숨기거나 표시합니다.

**스테이징**관리 : 이 권한은 *사이트 관리* &rarr; *게시* &rarr; *스테이징* &rarr; *작업* 메뉴에서 스테이징 구성 메뉴를 숨기거나 표시합니다.

**Publish Application Info**: 이 권한은 애플리케이션 수준의 스테이징 메뉴를 숨기거나 표시합니다.

**Publish Staging**: 이 권한은 Staging Bar의 *Publish to Live* 버튼을 숨기거나 표시하고 4 Site Administration 메뉴의 Staging 앱에서 *Add Staging Process* 버튼을 숨기거나 표시합니다. 이 권한은 선택 취소 여부에 관계없이 애플리케이션 정보 내보내기/가져오기, 페이지 내보내기/가져오기 및 애플리케이션 정보 게시 권한 기능을 자동으로 적용합니다.

**스테이징 보기**: *스테이징 게시*, *페이지 관리*, *스테이징 관리*또는 *업데이트 권한* 가 부여되지 않은 경우 이 권한은 사이트 관리 메뉴의 스테이징 앱을 숨기거나 표시합니다.

## 개별 변형 권한

개별 변형에 대한 권한을 설정하여 특정 사용자가 모든 변형이 아닌 일부 변형을 관리할 수 있도록 할 수도 있습니다.

1. 준비 표시줄에서 *작업* 버튼( ![Actions button](../../../images/icon-actions.png) )을 클릭하고 *사이트 페이지 변형* 또는 *페이지 변형*을 선택합니다.

   ![스테이징 표시줄에서 작업 버튼을 클릭하고 구성할 변형 유형을 선택합니다.](./managing-staging-permissions/images/01.png)

1. 구성하려는 변형에 대해 *작업* 버튼( ![Actions button](../../../images/icon-actions.png) )을 클릭하고 *권한*를 선택합니다.

   ![권한을 선택합니다.](./managing-staging-permissions/images/02.png)

1. 확인란을 사용하여 선택한 변형에 대한 사용자 권한을 구성한 다음 *저장*을 클릭합니다.

   ![확인란을 사용하여 선택한 변형에 대한 사용자 권한을 구성합니다.](./managing-staging-permissions/images/03.png)
