# 게시 도구

Liferay는 콘텐츠 생성 및 사이트 구축을 위한 두 가지 게시 도구인 게시 및 스테이징을 제공합니다. 각 도구를 사용하여 최종 사용자에게 영향을 주지 않고 작업 환경에서 사이트와 콘텐츠를 편집할 수 있습니다. 그러나 그들은 다른 Liferay 기능과 다양한 통합이 있는 다양한 프레임워크를 사용하여 이를 달성합니다. 별도의 서버에서 작업 환경과 생산 환경을 호스팅해야 하는 경우가 아니면 출판물을 사용하십시오. 이 경우 원격 라이브 스테이징을 사용해야 합니다. 자세한 내용은 [출판 도구 비교](./publishing-tools/comparing-publishing-tools.md) 을 참조하십시오.

```{important}
현재 Liferay의 게시 도구는 [커머스](https://learn.liferay.com/commerce/latest/ko/index.html) 또는 [사용자 정의 객체](../building-applications/objects.md)를 지원하지 않습니다. .
```
