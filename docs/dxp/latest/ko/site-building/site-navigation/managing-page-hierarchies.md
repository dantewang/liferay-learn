# 페이지 계층 관리

Liferay를 사용하면 페이지를 계층 구조로 구성하여 사이트에 원하는 구조를 얻을 수 있습니다. 이는 기존 페이지에 하위 [를](../creating-pages/adding-pages/adding-a-page-to-a-site.md#adding-a-child-page) 추가하거나 기존 페이지를 상위/하위 관계로 재구성하여 수행할 수 있습니다.

사이트의 페이지 계층 구조를 보려면 _사이트 메뉴_(![사이트 메뉴](../../images/icon-product-menu.png))를 열고 _사이트 빌더_를 확장한 다음 _페이지_를 클릭하세요. 이는 해당 사이트의 페이지 구성에 대한 주요 참고 자료입니다. 탐색 메뉴는 일부 페이지를 표시하도록 사용자 정의되어 있지만 이 메뉴는 항상 사이트 페이지에 대한 기본 참조입니다.

![Open the Pages application to view and manage your pages hierarchy.](./managing-page-hierarchies/images/01.png)

첫 번째로 나열된 페이지는 사용자가 사이트를 방문할 때 보게 되는 첫 번째 페이지입니다.

페이지를 끌어서 놓아 계층 구조로 재구성할 수 있습니다. 페이지는 최대 4개 수준까지 중첩될 수 있습니다.

![Drag and drop pages to create the desired hierarchy.](./managing-page-hierarchies/images/02.png)

_사이트 빌더_ &rarr; _페이지_에서 새 페이지를 만듭니다. 페이지를 추가하는 방법을 알아보려면 [사이트에 페이지 추가](../creating-pages/adding-pages/adding-a-page-to-a-site.md) 참조하세요.

## 페이지 옵션 구성

사용자는 _옵션_(![옵션](../../images/icon-options.png))을 클릭하여 페이지 옵션에 액세스할 수 있습니다.

| 기능         | 묘사                                 |
| :--------- | :--------------------------------- |
| **보기**     | 사이트에서 선택한 페이지를 봅니다.                |
| **구성**     | 페이지 구성 화면을 봅니다.                    |
| **페이지 복사** | 선택한 페이지를 복제하는 현재 사이트에 새 페이지를 만듭니다. |
| **사용허가**   | 페이지의 권한을 구성합니다.                    |
| **분리된 위젯** | 페이지에서 제거된 위젯과 관련된 데이터를 지웁니다.       |
| **삭제**     | 페이지와 해당 데이터를 모두 삭제합니다.             |

## 페이지 트리 사용

사이트의 페이지 계층 구조를 관리하는 또 다른 방법은 _페이지 트리_를 사용하는 것입니다. 여기에서 현재 사이트의 페이지를 보고 구성할 수 있습니다.

페이지 트리에 액세스하려면

1. _제품 메뉴_(![제품 메뉴](../../images/icon-product-menu.png))를 클릭합니다.

1. _페이지 트리_를 클릭하세요.

   ![The Page Tree function is at the top of the product menu.](./managing-page-hierarchies/images/03.png)

1. 왼쪽 메뉴에는 사이트의 페이지 계층 구조가 표시됩니다.

   ![View each Site's Page Tree.](./managing-page-hierarchies/images/04.png)

1. 이 드롭다운 컨트롤에서 탐색 메뉴를 선택할 수 있습니다.

1. 여기에서 할 수 있습니다

   - 검색 창(1)을 사용하여 트리에서 페이지를 찾습니다.
   - 페이지 미리보기를 보려면 페이지를 클릭하세요.
   - 추가(![추가](../../images/icon-plus.png)) 및 구성(![구성](../../images/icon-settings.png)을 사용하여 페이지를 추가하거나 페이지를 구성합니다. )) 버튼과 각 페이지 이름 옆에 있는 액션 메뉴(![액션 메뉴](../../images/icon-actions.png))(2).
   - 트리(3)에서 페이지를 끌어서 놓아 탐색 메뉴에서 페이지 순서를 수정합니다.

   ![You can add, preview, configure, or reorder pages from the Page Tree menu.](./managing-page-hierarchies/images/05.png)
