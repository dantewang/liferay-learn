# 페이지 계층 관리

Liferay를 사용하면 페이지를 계층 구조로 구성하여 원하는 사이트 구조를 얻을 수 있습니다. 이는 기존 페이지에 하위 페이지를 [추가하거나 기존 페이지를 상위](../creating-pages/adding-pages/adding-a-page-to-a-site.md#adding-a-child-page) 하위 관계로 재구성하여 수행할 수 있습니다.

사이트의 페이지 계층 구조를 보려면 **사이트 메뉴**(![Site Menu](../../images/icon-product-menu.png))을 열고 **사이트 빌더** 를 확장한 다음 **페이지** 을 클릭하십시오.

![Pages 계층 구조를 보고 관리하려면 Pages 애플리케이션을 엽니다.](./managing-page-hierarchies/images/01.png)

첫 번째로 나열된 페이지는 사용자가 귀하의 사이트를 방문할 때 보게 되는 첫 번째 페이지입니다.

페이지를 드래그 앤 드롭하여 계층에서 페이지를 재구성할 수 있습니다. 페이지는 최대 4단계까지 중첩될 수 있습니다.

![페이지를 드래그 앤 드롭하여 원하는 계층 구조를 만듭니다.](./managing-page-hierarchies/images/02.png)

## 페이지 옵션 구성

사용자는 **옵션**(![Option](../../images/icon-options.png)) 아이콘을 클릭하여 페이지 옵션에 액세스할 수 있습니다.

| 기능         | 묘사                                 |
|:---------- |:---------------------------------- |
| **보기** | 사이트에서 선택한 페이지를 봅니다.                |
| **형성하십시요** | 페이지 구성 화면을 봅니다.                    |
| **사본 페이지** | 선택한 페이지를 복제하는 현재 사이트에 새 페이지를 만듭니다. |
| **권한** | 페이지의 권한을 구성합니다.                    |
| **분리된 위젯** | 페이지에서 제거된 위젯 관련 데이터를 지웁니다.         |
| **삭제** | 페이지와 모든 데이터를 삭제합니다.                |
