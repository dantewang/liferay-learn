## 탐색경로 위젯 사용

이 위젯은 링크를 통해 관련 페이지나 사이트로 이동하는 방법을 제공합니다.

1. 조각 및 위젯에서 **Breadcrumb** 위젯을 페이지에 추가합니다. 콘텐츠 관리 카테고리에서 찾을 수 있습니다.

옵션 버튼(![옵션 버튼](../../images/icon-options.png))을 통해 위젯을 구성할 수 있습니다.

![The configuration window has four different tabs.](./using-the-breadcrumb-widget/images/01.png)

**디스플레이 템플릿:** 탐색 옵션에 대해 원하는 디스플레이 정렬을 선택합니다.

**현재 사이트 표시:** 이 페이지가 있는 현재 사이트를 표시하려면 전환하세요.

**게스트 사이트 표시:** 현재 페이지가 있는 게스트 사이트를 표시하려면 전환하세요.

**페이지 표시:** 현재 페이지를 표시하려면 전환하세요.

**상위 사이트 표시:** 현재 사이트의 상위 사이트를 표시하려면 전환하세요.

**애플리케이션 탐색경로 표시:** 애플리케이션 탐색경로를 표시하려면 토글합니다.
