# LivePerson

[LivePerson](https://www.liveperson.com/) 은 AI 기반 챗봇 플랫폼입니다. 이 문서에서는 Liferay 인스턴스와의 통합을 활성화하기 위해 [계정 ID를 찾는 방법을 설명](../../enabling-automated-live-chat-systems.md).

## LivePerson 계정 ID 찾기

1. 귀하의 [LivePerson 계정](https://authentication.liveperson.net/login.html)에 로그인하십시오.

1. 왼쪽 하단 모서리에 있는 *사용자 아이콘* 을 클릭합니다.

    ![홈 대시보드와 해당 정보를 볼 수 있습니다.](./liveperson/images/01.png)

1. 아래 스크린샷에서 강조 표시된 부분이 Liferay 채팅 제공자 계정 ID로 사용됩니다. 이 부분을 선택하고 복사하여 채팅 제공자 계정 ID로 사용하여 Liferay 인스턴스와 [자동 라이브 채팅 통합을 활성화](../../enabling-automated-live-chat-systems.md)하십시오.

     ![ID의 이 부분을 복사하여 Liferay 채팅 공급자 계정 ID로 사용하십시오.](./liveperson/images/02.png)
