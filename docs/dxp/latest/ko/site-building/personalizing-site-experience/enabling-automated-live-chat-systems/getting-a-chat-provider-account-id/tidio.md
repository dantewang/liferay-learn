# Tidio

[Tidio](https://www.tidio.com/) 은 고객 참여 플랫폼입니다. 이 기사는 Liferay 인스턴스와의 통합을 활성화하기 위해 [계정 ID를 찾는 방법을 설명](../../enabling-automated-live-chat-systems.md).

## Tidio 계정 ID 찾기

1. [Tidio 계정](https://www.tidio.com/panel/login) 에 로그인합니다.

1. *기어 아이콘* 을 클릭하여 설정 메뉴로 들어갑니다.

    ![설정 메뉴에서 기어 아이콘을 볼 수 있습니다.](./tidio/images/01.png)

1. *일반* 섹션으로 이동하여 *개발자* 메뉴 옵션을 클릭합니다.

    ![이미지는 일반 섹션과 개발자 메뉴 옵션을 보여줍니다.](./tidio/images/02.png)

1. *공개 키* 을 복사하고 이를 채팅 제공자 계정 ID로 사용하여 [Liferay 인스턴스와 자동화된 라이브 채팅 통합](../../enabling-automated-live-chat-systems.md) 을 활성화합니다.

    ![채팅 공급자 계정 ID로 사용할 공개 키를 복사합니다.](./tidio/images/03.png)
