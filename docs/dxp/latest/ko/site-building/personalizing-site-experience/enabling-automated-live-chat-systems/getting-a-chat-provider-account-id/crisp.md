# Crisp

[Crisp](https://crisp.chat/) 은 다중 채널 메시징 플랫폼입니다. 이 문서에서는 Liferay 인스턴스와의 통합을 활성화하기 위해 [계정 ID를 찾는 방법을 설명](../../enabling-automated-live-chat-systems.md).

## Crisp 계정 ID 찾기

1. [Crisp 계정](https://app.crisp.chat/initiate/login/) 에 로그인합니다.

1. 측면 탐색 모음에서 *Cog* 을 클릭하여 Crisp 계정 설정에 액세스합니다.

    ![측면 탐색 모음에서 Cog를 클릭하여 Crisp 계정 설정에 액세스합니다.](./crisp/images/01.png)

1. *웹사이트 설정*을 클릭하고 Liferay 인스턴스에 연결할 웹사이트를 선택한 다음 *설정*을 클릭합니다.

    ![웹사이트 설정을 클릭하여 Liferay 인스턴스에 연결할 웹사이트를 선택합니다.](./crisp/images/02.png)

1. *설정 지침*을 클릭하십시오.

   웹사이트 ID 필드를 찾습니다. 스크린샷에 강조 표시된 제공된 값은 채팅 공급자로 사용할 수 있습니다. *계정 ID* ~ [는 Liferay 인스턴스와 자동 라이브 채팅 통합](../../enabling-automated-live-chat-systems.md) 을 활성화합니다.

    ![설정 지침을 클릭하여 계정 ID를 찾으십시오.](./crisp/images/03.png)
