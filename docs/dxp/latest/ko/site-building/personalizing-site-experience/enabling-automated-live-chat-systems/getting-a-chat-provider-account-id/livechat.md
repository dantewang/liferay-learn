# LiveChat

[LiveChat](https://www.livechat.com/) 은 여러 플랫폼에서 팀과 사용자를 연결하는 메시징 플랫폼입니다. 이 기사는 Liferay 인스턴스와의 통합을 활성화하기 위해 [계정 ID를 찾는 방법을 설명](../../enabling-automated-live-chat-systems.md).

## LiveChat 계정 ID 찾기

1. 귀하의 [LiveChat 계정](https://my.livechatinc.com/) 에 로그인하십시오.

1. 왼쪽 하단 모서리에 있는 *설정 톱니바퀴* 을 클릭합니다.

    ![LiveChat 대시보드 페이지를 볼 수 있습니다.](./livechat/images/01.png)

1. *채널* &rarr; *채팅 페이지*를 클릭합니다.

    ![LiveChat의 채팅 페이지를 볼 수 있습니다.](./livechat/images/02.png)

1. 아래 스크린샷에서 강조 표시된 부분이 Liferay 채팅 제공자 계정 ID로 사용됩니다. 이 부분을 선택하고 복사하여 채팅 제공자 계정 ID로 사용하여 Liferay 인스턴스와 [자동 라이브 채팅 통합을 활성화](../../enabling-automated-live-chat-systems.md)하십시오.

     ![강조 표시된 부분을 복사하여 채팅 제공자 계정 ID로 사용합니다.](./livechat/images/03.png)
