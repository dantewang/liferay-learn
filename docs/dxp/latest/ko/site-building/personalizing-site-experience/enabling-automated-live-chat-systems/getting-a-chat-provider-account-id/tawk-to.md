# 톡톡

[Tawk.to](https://www.tawk.to/) 은 라이브 채팅 및 고객 지원 플랫폼입니다. 이 문서는 Liferay 인스턴스와의 통합을 활성화하기 위해 [계정 ID를 찾는 방법을 설명](../../enabling-automated-live-chat-systems.md).

## Tawk.to 계정 ID 찾기

1. [Tawk.to 계정](https://dashboard.tawk.to/login) 에 로그인합니다.

1. *관리* 버튼을 클릭합니다.

    ![Tawk-to Administration 버튼을 볼 수 있습니다.](./tawk-to/images/01.png)

1. *채널* 로 이동하여 *채팅 위젯*을 클릭합니다.

    ![채팅 위젯 옵션이 표시됩니다.](./tawk-to/images/02.png)

1. *Direct Chat Link* 헤더 아래에서 URL의 강조 표시된 부분을 복사하여 채팅 제공자 계정 ID로 사용하여 [Liferay 인스턴스와 자동 라이브 채팅 통합을 활성화](../../enabling-automated-live-chat-systems.md).

    ![직접 채팅 링크의 이 부분을 Liferay 채팅 계정 ID로 사용하십시오.](./tawk-to/images/03.png)
