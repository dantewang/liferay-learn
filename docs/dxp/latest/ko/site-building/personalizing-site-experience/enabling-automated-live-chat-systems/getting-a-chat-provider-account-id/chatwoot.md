# Chatwoot

[Chatwoot](https://www.chatwoot.com/) 은 오픈 소스 고객 커뮤니케이션 플랫폼입니다. 이 문서는 Liferay 인스턴스와의 통합을 활성화하기 위해 [계정 ID를 찾는 방법을 설명](../../enabling-automated-live-chat-systems.md).

## Chatwoot 계정 ID 찾기

1. 귀하의 [Chatwoot 계정](https://app.chatwoot.com/app/login) 에 로그인하십시오.

1. 측면 탐색 모음에서 *받은 편지함* 을 클릭합니다.

    ![측면 탐색 모음에서 받은편지함을 클릭합니다.](./chatwoot/images/01.png)

1. 받은 편지함 구성을 보려면 *설정* 을 클릭합니다.

    ![받은 편지함 구성을 보려면 설정을 클릭합니다.](./chatwoot/images/02.png)

1. *구성* 메뉴 탭을 클릭합니다.

    `websiteToken`의 필드 값을 찾습니다. 이 값은 스크린샷에 강조 표시되어 있으며 Liferay 인스턴스와 [자동 라이브 채팅 통합을 활성화](../../enabling-automated-live-chat-systems.md)하기 위한 채팅 제공자 계정 ID로 사용할 수 있습니다.

    ![구성 메뉴 탭을 클릭하여 계정 ID를 찾으십시오.](./chatwoot/images/03.png)
