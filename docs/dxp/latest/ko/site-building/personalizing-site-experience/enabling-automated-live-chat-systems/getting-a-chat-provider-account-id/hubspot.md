# Hubspot

[Hubspot](https://www.hubspot.com/) 실시간 채팅 기능을 포함하는 CRM 플랫폼입니다. [Hubspot과의 통합](../../enabling-automated-live-chat-systems.md) 활성화하려면 라이브 채팅과 Hubspot 사용자 계정 ID가 필요합니다.

```{important}
Hubspot과 통합하려면 활성 실시간 채팅이 있어야 합니다. 실시간 채팅을 생성하고 구성하려면 [Hubspot의 기술 자료](https://knowledge.hubspot.com/chatflows/create-a-live-chat) 의 단계를 따르세요.
```

## Hubspot 사용자 계정 ID 찾기

1. [Hubspot 계정](https://app.hubspot.com/login) 에 로그인하세요.

1. 오른쪽 상단에 있는 프로필로 이동하세요.

   사용하려는 채널에 해당하는 계좌번호를 복사하세요. 이 번호는 Liferay Portal에서 Click to Chat을 활성화할 때 사용되는 채팅 제공자 계정 ID와 일치합니다.

   ![Go to your profile in the top right corner and copy the account number to the channel you want to use.](./hubspot/images/01.png)
