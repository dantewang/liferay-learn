# Smartsupp

[Smartsupp](https://www.smartsupp.com/) 은 다채널 고객 커뮤니케이션 플랫폼입니다. 이 기사는 Liferay 인스턴스와의 통합을 활성화하기 위해 [계정 ID를 찾는 방법을 설명](../../enabling-automated-live-chat-systems.md).

## Smartsupp 계정 ID 찾기

1. [Smartsupp 계정](https://app.smartsupp.com/app/sign/in)에 로그인합니다.

1. *톱니바퀴 아이콘* &rarr; *채팅창*을 클릭합니다.

    ![Smartsupp의 왼쪽 막대에는 옵션이 표시됩니다. 기어 아이콘과 채팅 상자를 선택합니다.](./smartsupp/images/01.png)

1. 아래 스크린샷에서 강조 표시된 부분이 Liferay 채팅 제공자 계정 ID로 사용됩니다. 이 부분을 선택하고 복사하여 채팅 제공자 계정 ID로 사용하여 Liferay 인스턴스와 [자동 라이브 채팅 통합을 활성화](../../enabling-automated-live-chat-systems.md)하십시오.

     ![예제 Smartsupp 키가 제공됩니다. 이것을 채팅 제공자 ID로 사용합니다.](./smartsupp/images/02.png)
