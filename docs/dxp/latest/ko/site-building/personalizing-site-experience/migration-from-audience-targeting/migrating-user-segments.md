# 사용자 세그먼트 마이그레이션

사용자 세그먼트는 공통 속성 집합을 가진 사용자 하위 집합을 나타냅니다. Liferay [의 세분화](../segmentation/creating-and-managing-user-segments.md) 은 이전 Liferay DXP 버전의 대상 대상 지정과 유사하게 작동합니다. 대상 대상 지정에서는 사용자가 사용자 세그먼트에 속하기 위해 일치해야 하는 규칙 ** 를 생성합니다. 세분화에서 복잡한 필터를 정의하기 위해 다양한 방식으로 결합할 수 있는 *조건* 을 생성합니다. 사용자 세그먼트 생성 방법에 대한 자세한 내용은 [사용자 세그먼트 생성 및 관리](../segmentation/creating-and-managing-user-segments.md)을 참조하십시오.

## 잠재고객 타겟팅 규칙의 업그레이드 경로

Audience Targeting 사용자 세그먼트와 Liferay DXP 7.2+ 세그먼트 간의 유사성으로 인해 대부분의 구성은 업그레이드 중에 자동으로 Liferay Segmentation으로 전송됩니다. 다음 표는 Liferay 세그먼트 조건으로 마이그레이션할 때 대상 지정 규칙의 업그레이드 경로를 보여줍니다.

| 잠재고객 타겟팅 규칙 | 세그먼트 조건    | 업그레이드 경로                             |
|:----------- |:---------- |:------------------------------------ |
| 브라우저        | 브라우저       | 자동화. 대안으로 `에` 작업이 포함된 사용자 에이전트 필드 사용 |
| 주문 속성       | 주문 속성      | 자동화                                  |
| 언어          | 언어         | 자동화                                  |
| 최근 로그인 날짜   | 최근 로그인 날짜  | 자동화                                  |
| 조직 구성원      | 조직         | 자동화                                  |
| 운영체제        | User Agent | 자동화                                  |
| 이전에 방문한 사이트 | 사용 불가      | 자동화                                  |
| 일반 역할       | 역할         | 자동화                                  |
| 사이트 회원      | 사이트        | 자동화                                  |
| 사용자 그룹 구성원  | 사용자 그룹     | 자동화                                  |
| 나이          | 사용 불가      | 제안: 사용자 정의 필드                        |
| 페이스북(각종)    | 사용 불가      | 제안: 사용자 정의 필드                        |
| 성           | 사용 불가      | 제안: 사용자 정의 필드                        |
| 점수          | 사용 불가      | 추천: 쿠키                               |
| 방문한 페이지/콘텐츠 | 사용 불가      | 추천: 쿠키                               |
