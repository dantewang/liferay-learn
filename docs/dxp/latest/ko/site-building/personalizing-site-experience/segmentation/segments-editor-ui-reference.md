# 세그먼트 편집기 UI 참조

세그먼트를 사용하면 사용자를 공통 속성 집합을 가진 다양한 그룹으로 구성할 수 있습니다. 세그먼트 편집기 UI에 액세스하려면 다음 단계를 따르세요.

1. **사이트 관리** 로 이동하여 세그먼트를 생성하려는 사이트를 선택하세요.

1. **사이트 메뉴**(![사이트 메뉴](../../../images/icon-product-menu.png))를 열고 **사람** &rarr; **세그먼트** 로 이동합니다.

1. 새 사용자 세그먼트를 추가하려면 **추가**(![추가](../../../images/icon-add.png))를 클릭하세요.

편집기 상단에서 이름을 설정하고, 세그먼트를 **저장** 하거나, **취소** 를 선택하여 변경 사항을 취소할 수 있습니다. 조건 섹션에서는 정의된 세그먼트의 현재 구성원을 볼 수 있습니다.

페이지 오른쪽에는 다음 옵션이 포함된 **속성** 메뉴가 있습니다.

- 사용자
- 조직
- 세션
- 세그먼트

![You use the segment editor to create new segments.](./segments-editor-ui-reference/images/01.png)

연산과 접속사는 속성과 결합되어 기준을 정의합니다.

## 연산과 접속사

사용자 세그먼트 인터페이스를 통해 여러 유형의 정보를 수집할 수 있습니다. 일부 데이터는 텍스트 상자에 입력되는 반면 다른 데이터는 선택기를 사용하여 날짜 선택기와 같은 특정 기준이나 도구를 선택합니다. 일부 필드는 특정 상황에 따라 사용자 또는 상담원 데이터와 기준 간의 관계를 선택할 수 있는 연산자를 사용합니다.

* **같음**

* **같지 않음**

* **보다 큰**

* **크거나 같음**

* **미만**

* **작거나 같음**

* **포함한다**

* **포함하지 않음**

기준의 성격에 따라 연산자 선택에 다양한 조합이 포함될 수 있습니다. 예를 들어, 아래 설명된 **날짜** 선택에는 **포함** 및 **포함하지 않음** 을 제외하고 이전에 나열된 모든 옵션이 포함되는 반면, **이메일 주소** 선택에는 **같음** , **같지 않음** , **포함** 및 **포함됨이 포함됩니다.** 를 포함하지 않습니다.

각 기준과 범주 사이에 **And/Or** 접속사를 정의할 수 있습니다. **And** 의 경우 기준을 충족하려면 모든 기준이 충족되어야 합니다. **Or** 를 사용하면 정의된 기준 중 하나라도 참이면 참입니다. 연산자를 혼합하여 복잡한 사례를 만들 수도 있습니다.

## 사용자 속성

사용자 속성은 캡처하려는 사용자 속성입니다. 이는 계정에 정의된 사용자 메타데이터로 구성되지만 특정 그룹 멤버십(예: 역할 및 사용자 그룹)과 사용자 프로필이 마지막으로 수정된 날짜와 같은 정보도 포함됩니다. 다음과 같은 사용자 속성을 사용할 수 있습니다.

| 재산     | 묘사                                                    |
| :----- | :---------------------------------------------------- |
| 수정된 날짜 | 사용자 정보가 마지막으로 변경된 날짜를 선택할 수 있는 날짜 선택기와 관계 선택기를 제공합니다. |
| 이메일 주소 | 사용자 프로필에 제공된 이메일을 입력할 수 있는 텍스트 상자를 제공합니다.             |
| 이름     | 사용자 프로필에 제공된 이름을 입력하세요.                               |
| 직위     | 사용자 프로필에 제공된 직위를 입력하세요.                               |
| 성      | 사용자 프로필에 제공된 성을 입력하세요.                                |
| 일반 역할  | 사용자가 속한 역할을 선택합니다.                                    |
| 사용자 이름 | 사용자의 화면 이름을 입력합니다.                                    |
| 사이트    | 사용자가 회원으로 속한 사이트를 선택합니다.                              |
| 팀      | 사용자가 속한 팀을 선택합니다.                                     |
| 사용자    | 목록에서 특정 사용자를 선택합니다.                                   |
| 사용자 그룹 | 사용자가 속한 사용자 그룹을 선택합니다.                                |
| 사용자 이름 | 사용자의 전체 이름입니다.                                        |

!!! note
    세그먼트를 통해 게스트 사용자에게 콘텐츠를 표시하려면 **로그인** 세션 속성을 선택하고 해당 값을 **False** 로 설정합니다. 이 작업은 게스트 역할과 동일한 목적을 수행하는 익명 사용자를 효과적으로 나타냅니다.

Liferay DXP 7.2 SP2+부터 사용자는 다음 속성을 사용할 수 있습니다.

| 재산 | 묘사                                         |
| :- | :----------------------------------------- |
| 생일 | 사용자의 생년월일을 선택할 수 있는 날짜 선택기와 관계 선택기를 제공합니다. |
| 태그 | 사용자에게 적용되는 태그를 입력합니다.                      |

## 조직 속성

조직 속성은 세그먼트에 포함할 선택 가능한 조직 목록입니다. 여기에는 **이름** 및 **수정 날짜** 와 같은 사용자 선택과 유사한 기준이 포함되어 있습니다. 다음 조직 속성을 사용할 수 있습니다.

| 재산     | 묘사                          |
| :----- | :-------------------------- |
| 수정된 날짜 | 조직 정보가 마지막으로 수정된 날짜를 입력하세요. |
| 계층 경로  | 상위 조직의 이름을 입력하세요.           |
| 이름     | 조직의 이름을 입력하세요.              |
| 조직     | 특정 조직을 선택합니다.               |
| 상위 조직  | 특정 상위 조직을 선택합니다.            |
| 유형     | 조직 유형이 정의된 경우 조직 유형을 선택합니다. |

Liferay DXP 7.2 SP2+부터 조직에서도 다음 속성을 사용할 수 있습니다.

| 재산 | 묘사                    |
| :- | :-------------------- |
| 국가 | 조직과 연결된 특정 국가를 선택하세요. |
| 지역 | 조직과 연결된 특정 지역을 선택하세요. |
| 태그 | 조직에 적용되는 태그를 입력하세요.   |

## 세션 속성

세션 속성에는 사용자 활동, 브라우저 및 시스템 정보를 기반으로 한 기준이 포함되어 있습니다. 이를 사용하여 사용자의 기기나 OS를 타겟팅하거나 사용자가 특정 캠페인 기반 랜딩 페이지를 통해 웹사이트에 접속한 경우와 같은 활동 기반 기준에 사용할 수 있습니다. 다음 세션 속성을 사용할 수 있습니다.

| 재산           | 묘사                         |
| :----------- | :------------------------- |
| 브라우저         | 브라우저에서 속성을 입력합니다.          |
| Cookies      | 브라우저 쿠키의 이름을 입력하세요.        |
| 기기 브랜드       | 사용 중인 기기의 브랜드 이름을 입력하세요.   |
| 기기 모델        | 사용 중인 기기의 모델명을 입력하세요.      |
| 기기 화면 해상도 높이 | 화면 해상도 높이 값을 입력합니다.        |
| 기기 화면 해상도 너비 | 화면 해상도 너비 값을 입력합니다.        |
| 언어           | 현재 언어를 선택하세요.              |
| 최근 로그인 날짜    | 사용자가 마지막으로 로그인한 날짜를 선택합니다. |
| 현지 날짜        | 사용자가 사이트
한 현재 날짜를 선택하세요.     |
| 참조 URL       | 사용자가 마지막으로 방문한 URL을 입력하세요. |
| 로그인 됨        | 사용자가 로그인했는지 여부를 선택합니다.     |
| URL          | 현재 URL을 입력하세요.             |
| User Agent   | 사용자 에이전트 속성을 입력합니다.        |

Liferay DXP 7.2 SP2+부터 다음 속성도 세션에 사용할 수 있습니다.

| 재산      | 묘사                     |
| :------ | :--------------------- |
| 호스트 이름  | 가상 호스트 이름을 입력하세요.      |
| 요청 파라미터 | 요청 매개변수의 이름과 값을 입력합니다. |

Liferay DXP 7.4 U42+/GA42+부터 세션에 대해 다음 속성도 사용할 수 있습니다.

| 재산         | 묘사                          |
| :--------- | :-------------------------- |
| IP 지오코더 국가 | 사용자의 IP 주소와 일치하는 국가를 선택하세요. |

## 세그먼트 속성

> 사용 가능: Liferay DXP/Portal 7.3+.

세그먼트 속성을 사용하면 기존 세그먼트를 새로운 복합 세그먼트로 결합할 수 있습니다. 복합 세그먼트는 기반이 되는 세그먼트의 속성을 상속하고 구성원을 자동으로 업데이트합니다. 자세한 내용은 [복합 세그먼트](./creating-and-managing-user-segments.md#compound-segments) 참조하세요.
