# 세그먼트 편집기 UI 참조

여기에서는 세그먼트 편집기 UI에 대한 개요를 설명합니다. 세그먼트 편집기에 액세스하려면 다음 단계를 따르십시오.

1. *사이트 관리* 로 이동하여 세그먼트를 생성할 사이트를 선택합니다.

1. 사이트 메뉴에서 *People* &rarr; *Segments* 를 클릭합니다.

1. *사용자 세그먼트 추가* 버튼(![Add](../../../images/icon-add.png))을 클릭합니다.

편집기 상단에서 이름을 설정하고, 정의된 대로 세그먼트의 현재 구성원을 보고, *세그먼트 저장* 또는 *취소* 을 선택하여 변경 사항을 취소할 수 있습니다.

![Segment Editor의 상단 부분에는 세그먼트 이름과 해당 구성원이 있습니다.](./segments-editor-ui-reference/images/01.png)

페이지 오른쪽에는 다음 옵션이 있는 *속성* 메뉴가 있습니다.

- 사용자
- 조직
- 세션
- 세그먼트

![세그먼트 편집기를 사용하여 새 세그먼트를 생성합니다.](./segments-editor-ui-reference/images/02.png)

연산과 접속사는 속성과 결합되어 기준을 정의합니다.

## 연산과 접속사

사용자 세그먼트 인터페이스에서 여러 유형의 정보를 수집할 수 있습니다. 일부 데이터는 텍스트 상자에 입력되는 반면 다른 데이터는 선택기를 사용하여 특정 기준이나 날짜 선택기와 같은 도구를 선택합니다. 일부 필드는 특정 컨텍스트에 따라 사용자 또는 상담원 데이터와 기준 간의 관계를 선택할 수 있는 연산자를 사용합니다.

* *같음*

* *같지 않다*

* *~보다 큰*

* *크거나 같음*

* *미만*

* *작거나 같음*

* *포함한다*

* *포함하지 않는다*

기준의 특성에 따라 연산자 선택에 다른 조합이 포함될 수 있습니다. 예를 들어, 아래에 설명된 *날짜* 선택 항목에는 *에는* 이 포함되고 *에는*가 포함되지 않는 것을 제외하고 이전에 나열된 모든 옵션이 포함되는 반면, *이메일 주소* 선택에는 *은*, *은*이 아님, *는* 이 포함되고 *는 포함되지 않습니다. 포함하지*.

각 기준과 각 범주 사이에 "and" 또는 "or" 접속사를 정의할 수 있습니다. "and"의 경우 기준이 충족되려면 모든 기준이 참이어야 합니다. "또는"을 사용하면 정의된 기준 중 하나라도 참이면 참이 됩니다. 연산자를 혼합하여 복잡한 사례를 만들 수도 있습니다.

## 사용자 속성

사용자 속성은 캡처하려는 사용자 속성입니다. 이는 계정에 정의된 사용자 메타데이터로 구성되지만 특정 그룹 구성원(예: 역할 및 사용자 그룹)과 사용자 프로필이 마지막으로 수정된 날짜와 같은 정보도 포함합니다. 다음 사용자 속성을 사용할 수 있습니다.

| 재산            | 묘사                                                |
|:------------- |:------------------------------------------------- |
| 수정된 날짜        | 사용자 정보가 마지막으로 변경된 날짜를 선택할 수 있는 날짜 선택기 및 관계 선택기 제공 |
| Email.address | 사용자 프로필에 제공된 이메일을 입력할 수 있는 텍스트 상자 제공              |
| 이름            | 사용자 프로필에 제공된 이름을 입력합니다.                           |
| 그룹            | 사용자가 구성원인 사이트를 선택하십시오.                            |
| 직위            | 사용자 프로필에 제공된 직함을 입력합니다.                           |
| 성             | 사용자 프로필에 제공된 성을 입력합니다.                            |
| 역할            | 사용자가 속한 역할을 선택합니다.                                |
| 사용자 이름        | 사용자의 화면 이름을 입력합니다.                                |
| 팀             | 사용자가 속한 팀을 선택합니다.                                 |
| 사용자 그룹        | 사용자가 속한 사용자 그룹을 선택합니다.                            |
| 사용자           | 목록에서 특정 사용자를 선택합니다.                               |
| 이름            | 사용자의 전체 이름입니다.                                    |

Liferay DXP 7.2 SP2+부터 다음 속성도 사용자가 사용할 수 있습니다.

| 재산 | 묘사                                          |
|:-- |:------------------------------------------- |
| 생일 | 사용자의 생년월일을 선택할 수 있는 날짜 선택기 및 관계 선택기를 제공합니다. |
| 태그 | 사용자에게 적용되는 태그를 입력합니다.                       |

## 조직 속성

조직 속성은 세그먼트에 포함할 조직의 선택 가능한 목록입니다. 여기에는 *이름* 및 *수정한 날짜*과 같이 사용자 선택과 유사한 기준이 포함됩니다. 다음 조직 속성을 사용할 수 있습니다.

| 재산     | 묘사                          |
|:------ |:--------------------------- |
| 수정된 날짜 | 조직 정보가 마지막으로 수정된 날짜를 입력합니다. |
| 이름     | 조직의 이름을 입력합니다.              |
| 계층 경로  | 상위 조직의 이름을 입력합니다.           |
| 조직     | 특정 조직을 선택합니다.               |
| 상위 조직  | 특정 상위 조직을 선택합니다.            |
| 유형     | 조직 유형이 정의된 경우 조직 유형을 선택합니다. |

Liferay DXP 7.2 SP2+부터 다음 속성을 조직에서도 사용할 수 있습니다.

| 재산 | 묘사                    |
|:-- |:--------------------- |
| 국가 | 조직이 연결된 특정 국가를 선택합니다. |
| 지역 | 조직이 연결된 특정 지역을 선택합니다. |
| 태그 | 조직에 적용되는 태그를 입력합니다.   |

## 세션 속성

세션 속성에는 사용자 활동, 브라우저 및 시스템 정보를 기반으로 하는 기준이 포함되어 있습니다. 이를 사용하여 사용자의 장치 또는 OS를 타겟팅하거나 사용자가 특정 캠페인 기반 랜딩 페이지를 통해 웹사이트에 들어온 경우와 같은 활동 기반 기준에 사용할 수 있습니다. 다음 세션 속성을 사용할 수 있습니다.

| 재산           | 묘사                          |
|:------------ |:--------------------------- |
| 브라우저         | 브라우저에서 속성을 입력합니다.           |
| 쿠키           | 브라우저 쿠키의 이름을 입력하십시오.        |
| 기기 브랜드       | 사용 중인 장치의 브랜드 이름을 입력하십시오.   |
| 기기 모델        | 사용 중인 기기의 모델명을 입력하세요.       |
| 기기 화면 해상도 높이 | 화면 해상도 높이 값을 입력합니다.         |
| 기기 화면 해상도 너비 | 화면 해상도 너비 값을 입력합니다.         |
| 언어           | 현재 언어를 선택합니다.               |
| 최근 로그인 날짜    | 사용자의 마지막 로그인 날짜를 선택합니다.     |
| 현지 날짜        | 사용자가 위치한 현재 날짜를 선택합니다.      |
| 참조 URL       | 사용자가 마지막으로 방문한 URL을 입력하십시오. |
| 로그인 됨        | 사용자가 로그인했는지 여부를 선택합니다.      |
| URL          | 현재 URL을 입력합니다.              |
| User Agent   | 사용자 에이전트 속성을 입력합니다.         |

Liferay DXP 7.2 SP2+부터 다음 속성도 세션에 사용할 수 있습니다.

| 재산      | 묘사                     |
|:------- |:---------------------- |
| 호스트 이름  | 가상 호스트 이름을 입력합니다.      |
| 요청 파라미터 | 요청 매개변수의 이름과 값을 입력합니다. |

Liferay DXP 7.4 U42+/GA42+부터 다음 속성도 세션에 사용할 수 있습니다.

| 재산         | 묘사                          |
|:---------- |:--------------------------- |
| IP 지오코더 국가 | 사용자의 IP 주소와 일치시킬 국가를 선택합니다. |

## Segments 속성

> 사용 가능: Liferay DXP/Portal 7.3+.

세그먼트 속성은 기존 세그먼트를 새로운 복합 세그먼트로 결합하는 방법을 제공합니다. 복합 세그먼트는 기반이 되는 세그먼트에서 속성을 상속하고 멤버를 자동으로 업데이트합니다. 자세한 내용은 [복합 세그먼트](./creating-and-managing-user-segments.md#compound-segments)을 참조하십시오.
