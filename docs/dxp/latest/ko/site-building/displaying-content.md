# 콘텐츠 표시

다양한 방법으로 콘텐츠를 표시할 수 있습니다. 선택하는 옵션은 표시하려는 자산 유형, 한 번에 표시하려는 자산 수, 콘텐츠 표시 방법에 따라 달라집니다. Liferay를 사용하여 콘텐츠를 표시하는 방법에 대해 자세히 알아보려면 [콘텐츠 표시](./displaying-content/displaying-content-intro.md) 읽어보세요.
