# 실시간 알림

Liferay Cloud는 시스템 관리자에게 프로젝트의 예기치 않은 동작을 알릴 수 있습니다. 예기치 않은 동작의 예로는 자동 확장 이벤트, 예상보다 높은 메모리 소비, 할당된 CPU 할당량 도달, 데이터베이스 연결 문제 등이 있습니다.

## 경고 기본 설정

사용자는 Liferay Cloud Console을 통해 모든 환경에 대해 [경고 기본 설정](https://console.liferay.cloud/account/alerts-preferences) 을 설정할 수 있습니다.

1. 콘솔 탐색 모음의 오른쪽 상단에 있는 *사용자 프로필* 아이콘을 클릭합니다.

1. *경고 기본 설정*을 클릭합니다.

1. 선호하는 알림 방법에 대한 확인란을 클릭하여 수신할 알림 유형을 선택합니다. 경고를 비활성화하려면 관련 확인란을 선택 취소합니다.

1. 완료되면 *경고 기본 설정 저장* 을 클릭합니다.

![실시간 알림](./real-time-alerts/images/01.png)

사용자는 경고 페이지에서 경고 기본 설정에 액세스할 수도 있습니다.

## 알림 페이지

[경고 페이지](https://console.liferay.cloud/alerts)에서 콘솔로 전달된 경고를 보고 관리합니다. 페이지 오른쪽 상단의 벨 아이콘을 클릭하여 이 페이지에 액세스할 수 있습니다.

경고 테이블에는 다음 경고 정보가 나열됩니다.

* **상태**: 이 열은 경고의 현재 상태를 나타냅니다. 각 경고는 특정 이벤트에 의해 트리거됩니다. 이벤트가 계속 발생하는 경우 경보 상태는 *진행 중*입니다. 이벤트가 완료되면 경보 상태는 *해결됨*입니다.

* **Started at**: 이 열은 경보가 시작된 시기를 나타냅니다.

* **기간**: 이 열은 경고가 지속된 기간을 나타냅니다.

표 위에 있는 버튼을 사용하여 경고를 필터링하고 관리합니다.

* **읽지 않은 경고/모든 경고**: 읽지 않은 경고만 보거나 모든 경고를 봅니다.

* **모든 환경/기타 환경**: 환경별로 경보를 필터링합니다.

* **모두 읽음으로 표시**: 모든 경고를 읽음으로 표시합니다.

* **경고 기본 설정**: 경고 기본 설정 페이지로 이동합니다.
