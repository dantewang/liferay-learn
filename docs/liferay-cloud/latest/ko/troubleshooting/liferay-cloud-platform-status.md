# Liferay 클라우드 플랫폼 상태

Liferay Cloud Platform은 여러 시스템으로 구성됩니다. 사용자는 [Liferay Cloud Platform](https://status.liferay.cloud/) 상태 페이지를 통해 Liferay Cloud Platform 시스템, 사건 기록 및 계획된 유지 관리 기간의 상태를 볼 수 있습니다.

## Liferay Cloud Console을 통해 상태 페이지에 액세스

사용자는 Liferay Cloud 콘솔을 통해 Liferay Cloud Platform 상태 페이지에 액세스할 수 있습니다.

1. 콘솔 페이지의 오른쪽 상단에 있는 *도움말* 아이콘을 클릭합니다.

1. *가동 상태*을 클릭합니다.

![그림 1: 가동 시간 상태를 클릭하여 Liferay 클라우드 플랫폼 상태 페이지를 봅니다.](./liferay-cloud-platform-status/images/01.png)

## 클라우드 플랫폼 시스템 현황 보기

Liferay Cloud Platform 상태 페이지에는 다음 시스템의 현재 상태가 나열됩니다.

* 인프라 서비스
* 데이터베이스 서비스
* 스토리지 서비스
* 모니터링 및 로깅 서비스
* 네트워킹 서비스
* 라이프레이 클라우드 콘솔
* 라이프레이 클라우드 API

![그림 2: Liferay Cloud Platform 시스템의 현재 상태를 봅니다.](./liferay-cloud-platform-status/images/02.png)

## 인시던트 기록 및 예약된 유지 관리 기간 보기

사용자는 Liferay Cloud Platform 상태 페이지에서 과거 사건 및 예정된 유지 관리 기간을 볼 수 있습니다.

Liferay Cloud Platform 상태 페이지를 아래로 스크롤하여 최근 사건을 확인하십시오.

확장된 기록을 보려면 페이지 하단의 *사건 기록* 을 클릭하십시오.

![그림 3: 과거 인시던트 및 계획된 유지 관리 기간을 봅니다.](./liferay-cloud-platform-status/images/03.png)

## 업데이트 구독

사용자는 *업데이트를 구독하여 플랫폼 상태 변경에 대한 실시간* 을 받을 수 있습니다.

*업데이트 구독*을 클릭하고 가장 편리한 알림 방법을 선택하십시오.

![그림 4: Liferay Cloud Platform의 상태에 관한 업데이트를 받으려면 구독하십시오.](./liferay-cloud-platform-status/images/04.png)
