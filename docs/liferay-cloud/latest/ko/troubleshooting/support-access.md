# 지원 액세스

지원 액세스는 Liferay 엔지니어가 Liferay 클라우드 프로젝트 환경에 직접 액세스할 수 있도록 하여 문제 해결을 촉진하는 선택적 설정입니다. Liferay Cloud 관리자는 개별 프로젝트 환경에 대한 지원 액세스를 활성화하거나 비활성화할 수 있습니다.

지원 액세스가 활성화되면 Liferay 지원 엔지니어는 다음 정보에 대한 읽기 액세스 권한을 갖습니다.

* 프로젝트 콘솔
* 서비스 로그
* 제어판 설정
* Liferay Cloud 서비스가 호스팅되는 지역
* 팀원 및 관련 역할
* 회원들의 활동

```{note}
지원 액세스는 Liferay 지원 엔지니어가 프로젝트에서 자산을 배포하거나 쓰기 작업을 수행하도록 허용하지 않습니다**.
```

## 지원 액세스 설정 변경

지원 액세스는 각 환경에서 기본적으로 활성화되어 있지만 관리자는 언제든지 비활성화할 수 있습니다.

지원 액세스 설정을 변경하려면 다음 단계를 따르십시오.

1. **설정** &rarr; **지원 액세스** 로 이동합니다.
1. 토글 스사이트를 원하는 설정으로 설정합니다(**활성화** 또는 **비활성화**).

![그림 1: 관리자는 지원 액세스를 활성화하거나 비활성화할 수 있습니다.](./support-access/images/01.png)

## 지원 액세스에 대한 변경 사항 모니터링

지원 액세스에 대한 모든 변경 사항은 변경한 사람과 시기를 나열하는 새 활동을 등록합니다. 이러한 활동은 **활동** &rarr; **일반** 에 나타납니다.

![그림 2: 일반 활동 패널에는 지원 액세스에 대한 모든 변경 사항이 표시됩니다.](./support-access/images/02.png)

지원 액세스 설정이 변경되면 Liferay Cloud도 모든 팀원에게 이메일을 보냅니다.

![그림 3: 지원 액세스 설정이 변경되면 모든 사람에게 이메일로 알립니다.](./support-access/images/03.png)
