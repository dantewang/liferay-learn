# 셸 액세스

Liferay Cloud의 명령줄 도구는 속도, 제어, 추적 가능성, 스크립팅 및 자동화 기능을 제공하여 개발자의 작업 흐름에 기여합니다. 셸 액세스를 통해 애플리케이션 내부에서 진행 중인 작업을 보다 쉽게 확인할 수 있습니다. 예를 들어 셸을 사용하여 로그에서 쉽게 발견되지 않는 부작용을 찾을 수 있습니다. 한 번만 실행되는 데이터 채우기 또는 보고서 생성을 위한 함수를 호출할 수도 있습니다.

```{note}
백업 및 데이터베이스 서비스는 셸 액세스를 제공하지 않습니다. 
```

## 웹 콘솔을 통해 셸에 액세스

Liferay Cloud 웹 콘솔을 통해 셸에 액세스하려면 다음 단계를 따르십시오.

1. 환경 페이지로 이동합니다.
1. 왼쪽 메뉴에서 **서비스** 을 클릭합니다.
1. 액세스하려는 서비스를 선택한 다음 **Shell** 탭을 클릭합니다.

![그림 1: Liferay Cloud의 웹 콘솔을 통해 셸에 액세스합니다.](./shell-access/images/01.png)

## 터미널을 통해 셸에 액세스

터미널을 통해 서비스의 셸에 액세스하려면 다음 단계를 따르세요.

1. [아직 설치되지 않은 경우 Liferay Cloud 명령줄 도구](../reference/command-line-tool.md), 를 설치합니다.
1. `lcp shell`명령을 실행한 다음 액세스할 서비스를 선택하십시오.
1. 원하는 명령을 실행하십시오.

![그림 2: 명령줄을 통해 셸에 액세스합니다.](./shell-access/images/02.png)

또는 액세스 서비스 및 프로젝트를 이미 알고 있는 경우 대신 다음 명령을 실행할 수 있습니다.

```shell
lcp shell -p projectID -s serviceID
```

## 쉘 제한 사항

셸은 문제를 해결하거나 서비스에서 일회성 작업을 수행하는 훌륭한 도구이지만 영구적인 변경을 위한 것은 아닙니다. 서비스를 배포하거나 다시 시작하면 영구 볼륨에 없는 모든 파일이 새 빌드로 대체됩니다. 셸의 경우 이는 서비스에서 실행하는 모든 명령이 일시적이며 볼륨 내에서 변경하지 않는 한 서비스를 재배포하거나 다시 시작할 때 재설정됨을 의미합니다.
